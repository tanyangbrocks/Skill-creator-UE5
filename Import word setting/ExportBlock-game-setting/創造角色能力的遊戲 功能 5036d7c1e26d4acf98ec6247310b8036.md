# 創造角色能力的遊戲 功能

創造角色能力的遊戲 功能
玩家在地圖可執行動作
（各自可設定不同風格：孩童/少年/成人/熱血/冷靜）

發動技能、使用武器、使用工具
（傷害類技能、武器可以手動鎖定，或設定在特定區域，如：禁戰區，自動鎖定，避免誤按）

移動類：跑、衝、跳、爬、滾、滑、游、飛

社交類：以打字或語音用玩家聲線說話、打招呼、鼓掌、比讚、擺pose、笑、哭、點頭、搖頭、嘆息、敬禮、鞠躬、挑釁

表情/眼神：微笑、傷心、興奮、淚眼汪汪、鼓勵、尷尬、害羞、害怕、緊張、疑惑、驚訝、渴望、生氣、瞪、鄙視、竊笑、冷笑、戀愛、淫蕩。（想用上述沒有的也可以以直接傳貼圖、emoji，對話框會浮出來）

互動類：握手、（跳）擊掌、擊拳、摸頭、捏臉、戳臉、拍肩、拍背、親吻、擁抱、牽手走/跑

生活類：躺下、坐、蹲、跪、吃、喝、排泄、發呆/沉思、打哈欠、打噴嚏、咳嗽、抓取

複雜類：寫字、繪畫、閱讀、滑手機、打電腦、操作儀器、玩遊戲、吹口哨

職業技能：演奏、舞蹈、開車

情慾類：脫衣/褲、自慰、打炮、性侵「僅限對NPC」、高潮、呻吟、舔、吸

玩家間可彼此發佈任務、組隊，分享技能攻略

可開設職業類別企業、股市、賭局、公會、學院，甚至自創副本、國家、地形、世界。

玩家可自行設計服飾、裝備、靈獸、載具（裝備道具亦有不同被動效果）

玩家戒指個人異次元空間：
世界風格：自然山水、宇宙及星辰、虛擬世界、巨塔。

景觀：含空間大小、數量、氣候、聲光特效等。

物件：
房屋：可購買、蓋複數棟，可選平房、高樓、地下室、洞窟、城堡、浮空島等種類。
森林/農田/養殖場/魚池：養育、產資源。
店面：玩家申請開業後取得，可選擇外顯於現實世界。
功能性符文陣：建構結界、聲光效果，或是添加特殊法則如時間加速、虛數空間等。
雕像、神廟：宗教用途。
訓練場：提供眷屬、神寵進行訓練（掛機收益）。
輔助職業工作室：其它職業用途的專屬工作室，如工匠鋪、煉金閣、實驗室、科技工廠等。
坐騎車位：可停放坐騎、載具。

（神寵、眷屬、英靈、鬼物等召喚物可以在空間中亂晃）

儲物戒指內物品分類
裝備/工具：裝備、工具。
素材：可直接拿來強化升級，如精鐵、附魔石；異靈水晶、反異靈固化劑。
資源：植物、礦石、海產、生物材料、異火、異雷、詭氣、奇冰、設計藍圖、技能樣本。
貨幣：靈石、魔晶、錢幣、任務/活動蒐集貨幣。
成品：已製造加工完成之料理、藥物、飼料、契約書（未生效/生效）、放置型建築、載具、機械等。
道具：卡牌、任務道具、宗教道具、靈職道具、圖騰/符文/刻印等。

儲能瓶：用以儲存暫時不用的能量種類。

戰鬥模式：
高塔：神或魔王為世人所建造的試煉之地，不同場景快速刷怪、刷資源為主。

迷宮：特殊寶物為選拔合適主人所創造的空間，除戰鬥外還配合機關、解謎、小遊戲，抵達終點可獲得寶物。

地下城：領主級以上的「獸」創造的空間，含小怪房、獎勵房、事件房、Boss房等。

次元門：前往不同世界歷史的通道，為重大歷史殘留的痕跡和情緒。為故事任務副本，解謎要素高，具備不同通關條件和隱藏任務。

野生怪戰鬥（聚落型、領主型、野生強者、傷害和平怪、寶物資源出世、守護獸、行動災厄型、移動戰鬥族群型）

NPC戰鬥（任務事件、戰鬥劇情、特殊地點、傷害到NPC、受NPC通緝）

PVP（非禁戰區無禁戰令者遇到直接打、友誼切磋、各種競技場、團隊混戰、組織或國家戰爭、排行榜戰、排位賽、吃雞生存、職業考核戰、任務劇情）

戰鬥職業證照考核（含角色條件及玩家能力）
戰鬥：傭兵、冒險者、劍士（近戰走位）、槍兵、法師、刺客、輔助、狙擊手（弓、槍炮）、坦克、治癒師、馴獸師、斥候

輔助職業證照考核：廚師、商人、工匠、礦工、農夫、漁夫、飼育員、圖騰師、鍊金術師、科學家、工程師、設計師、學士、樂師、舞者、車手、棋士、偵探

可解鎖職業類技能能力效果和該領域進階功能（這些職業都是要用考的）

裝備
工匠可畫出符合自身等級之藍圖，並湊出材料，最終打造。打造完成瞬間品質及攻擊方式固定。（除非使用科技改良）
工匠技能可使用強化（等級）、精煉（機率提高+1）、熔煉（替換詞條）、升星（提高星級）。
附魔可使裝備增加詞條。
鍊能武裝可為裝備附加套裝效果。
煉靈可讓武器轉為法寶，使其自動對敵，但要消耗精神力。
將法寶煉化為本命法寶可讓法寶威能大增。

裝備品質：黑鐵、白銀、黃金、翡翠、鑽石、神器。
裝備星級：0～6星。

裝備種類：
武器：僅能同時穿戴一個（不含法寶），如冷兵器、熱武器、科技武器、法寶等。

裝甲：分為頭、外身、貼身、腿、鞋、臂、手套，至多七件。如重甲、法袍、輕裝、休閒裝

飾品：分為項鍊、手環、吊墜、錶、戒、腰帶、眼鏡；同部位不可重複，最多四件。

儀器：最多三件，如通訊器、增幅器、偵測器、護身法器。

工具：非戰鬥用途，最多同時攜帶三件，如相機、耳機、筆、繩索、電鑽、手電筒、鑰匙、吸塵器、掃把、體育用具等。

植物
植物等級：普通、稀世、靈植、奇珍、天精。

農夫、園藝等技能可解鎖種植相關技能或增益。

食用型（回復、增buff、料理、毒藥）
特殊型（聚靈、聚魔、儲存、獵捕、吉祥）

放置型建築
砲臺（箭塔、激光炮、重炮、迫擊炮、自走炮等）
堡壘（防禦堡壘、兵工廠堡壘、移動城）
裝置（回血臺、buff臺、偵測器、地雷、炸彈）

神寵
不具靈智的「獸」被收服或具備靈智的「族」成為契約中的眷屬，皆歸類為神寵。可進行召喚。

獸的實力之等級劃分與「獸」相同：初級、中級、高級、領主級、君王級、災害級、災厄級、天災級。

載具
載具等級：D~S
載具類型：武裝、坐騎、競速
陸（機車、跑車、單車、坐騎、坦克）
海（船、海坐騎、潛艇）
空（飛機、直升機、飛船、飛天坐騎）

競速類型：
直接和路人、NPC發起競速
特殊場域競速比賽（含陸、海、空）
排位賽、車手考核、賭博

附體
可暫時對象繼承部份能力值、技能，產生靈體形象虛影。
鬼物：凡間或冥界怨念深重的死靈。等級分為整、亂、奇、域、淵。
英靈：部分神話人物、歷史人物、故事中人物、契約玩家。等級分為軼聞、響名、上古、傳說、神話。

變身
可變化角色型態、可切換技能組方案。
依照變化型態決定對應方案可增加的技能點。

融合契約
兩人各部份操控，或由一人操控（可設定）

鍊金
圖騰、符文、刻印創造
基因改造：洗點、種族外觀或能力改變
鍊能武裝：為裝備附加套裝效果
卡牌製造：儲存技能、限時靈職、限時增益，如1小時任務收益卡、1小時經驗翻倍卡
藥物製造：如魔藥（需魔力）、藥粉、丹藥（需靈力）、注射型藥劑等。

科技
打造科技裝備、裝備改裝
打造放置型建築（砲臺、堡壘、裝置、儀器）
打造載具、家具

料理
地點：玩家空間的廚房、野生灶台、建築物的廚房、開設的餐廳
料理方式：蒐集食譜、素材及用具，食材需加工。以反應力、連點、精準度的小遊戲料理。視廚師經驗解鎖快速或批量料理，以及高級食譜。
食譜等級：初級、中級、高級、頂級、絕世
料理美味度：以數值計算，可提高心情值或賦予幸福、愉悅等狀態。
料理功效（回復、buff、提升等，不一定和美味度成正關係）

挖礦
角色之所有攻擊行動可破壞地形，但地形會隨時間逐漸修復。（損壞越嚴重、修復速度越快）
普通地面深度達到100米可能有礦，但機率和數量較低；特殊礦場會產出不同種類礦物。
使用特定武器、技能可能提升掉寶率及數量、品質。

樂師
相關職業技能可讓角色演奏樂器或歌唱、用內建簡易功能編曲並發佈等。可直接在路邊演奏、接受他人（NPC）委託演奏，去KTV、酒吧等場所時使用，甚至開演唱會。

進行方式為落下型譜面之節奏遊戲。按難度、速度、評分及角色加成決定獲得之經驗。

舞者
相關職業技能可讓角色進行舞蹈。可直接在路邊舞蹈、接受他人（或NPC）委託表演，去舞廳、酒吧等場所時使用，甚至開演唱會。

進行方式譜面變化多端的節奏遊戲。按難度、速度、評分及角色加成決定獲得之經驗。

工作
傭兵執照：可擔任護衛隊、保全、打手、軍團長、警察。

冒險者執照：可建立或加入討伐隊或騎士團（還需考量職階執照）、公會自由委託、

不需執照：臨時工、清潔人員、

商人
基礎功能可依法開設店面（通常NPC會賣得比較貴，且和玩家交易能獲得友情點）。
進階可創立公司，進行上市、加盟、發佈拍賣會；開設賭場、飯店、情色酒店、觀光景點等娛樂場所。
但必須向該國繳交營業稅、娛樂稅等，比例照該國而定。（只有商人要繳稅，其它玩家皆為旅人）

若開立黑市，可進行法條之外的交易，如交易玩家素材體、綁架並販賣NPC、僱用殺手、毒品流通（對NPC較有用）、運輸非法武器和傳播邪教等。若向銀行、地下錢莊貸款，可能受到律法懲處或未知災害。

常見貨幣種類：
下品靈石 中品靈石 上品靈石 極品靈石
（每100個進階）
白魔晶 綠魔晶 藍魔晶 紫魔晶 紅魔晶
（每100個進階）
銅錢 白銀 黃金（每100個進階）
泰克通用幣（攜帶金融卡，存於虛擬帳戶）

各類貨幣匯率不固定，玩家可隨時查看並交易。

犯罪
進行殺人、搶劫、縱火、綁架、走私違禁品等行為時，除非被玩家、NPC發現異常並通報，否則不會被通緝。

擊殺：所有生物原屍體會保留，可自行利用輔助位技能、廚師技能或肢解刀分解屍體素材。
（綁架或屠殺NPC時，NPC會哭、會哀嚎、會求饒、會試圖交涉，更會憤而反擊。）

AI功能
內建一個能解惑遊戲問題並協助跳轉的AI（聲控or打字），名為萬能精靈。它可以自動分析並提出、套用技能分配建議。

具有主線故事，也可能會因玩家行為用AI生成不同故事、任務、獎勵。AI也會隨機在地圖生成寶物、神裝、特殊事件等機緣。

NPC是由AI自動生成的AI，天賦能力點多落在10~30之間，姓名、外貌、聲線、性格、負面狀態和靈魂、體質、心靈多為隨機；每個NPC都是AI亂數生成，皆為獨一無二的。其數量遠超玩家，任何階級地位、種族、等級、能力或年齡層都有，各自有自己的家庭、朋友、生活、工作、性格及情感。

可以和NPC互動，NPC可能會稱讚角色外貌、能力等（大部分NPC不知道玩家存在，視玩家為天縱之材）。NPC也可能出現流氓、變態（不會太強）、精神異常者，也可能心高氣傲，具備野心和陰謀。

每間民宅玩家都可以闖入，有玩家進入後生成NPC、家具等。可進行殺戮、搶劫、性侵等行動，但必然會被通緝。
