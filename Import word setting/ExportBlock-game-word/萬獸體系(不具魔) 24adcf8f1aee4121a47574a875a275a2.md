# 萬獸體系(不具魔)

肉體強大、個體多半不具魔的物種，通常喜好獵食、戰鬥。

萬獸體系全族種皆可進化，但進化方式與難度不一。

根據史書記載，萬獸體系共有兩千八百二十三族，以下為常見種族：

—–黃階—–

鬼族：

[鬼族](%E9%AC%BC%E6%97%8F%20e65c4c3c92634e2ab014330344a4e71d.md)

哥布林族：

[哥布林族](%E5%93%A5%E5%B8%83%E6%9E%97%E6%97%8F%20689a3c93e6884b12a88a43c17851b801.md)

犬族：肉體力量強，靈力和妖力天賦高，多生活於叢林、草原、河川、峽谷等地。與人族較為親近，經常成為人族契約寵獸。

貓族：肉體力量強，靈力和妖力天賦高，多生活於叢林、草原、河川、峽谷等地。與人族較為親近，但部份貓族喜歡捕捉並奴役人類。

萬獸妖族：

[妖族：萬獸妖族&天行妖族&水行妖族&妖精族&肢節妖族](%E5%A6%96%E6%97%8F%EF%BC%9A%E8%90%AC%E7%8D%B8%E5%A6%96%E6%97%8F&%E5%A4%A9%E8%A1%8C%E5%A6%96%E6%97%8F&%E6%B0%B4%E8%A1%8C%E5%A6%96%E6%97%8F&%E5%A6%96%E7%B2%BE%E6%97%8F&%E8%82%A2%E7%AF%80%E5%A6%96%E6%97%8F%2023833e9257dd4852ba37c7170400aa2f.md)

半人馬族：近似類人體系，有高度發展之文化，善使用真氣與弓箭，常利用占星術預測未來，多分佈於草原。

狂牛族：肉體強，擅長使用真氣，能將力量積蓄於牛角並高速衝撞敵人，破壞力極強。

雌性狂牛族能生產蘊含不同力量的牛奶，受各族歡迎，此為狂牛族主要經濟來源。狂牛族多分佈於草原。

神象族：肉體極度強悍，體型大。長鼻子十分靈活，使其能製作並使用工具，但不擅使用能量。多分佈於草原。

戰羊族：真氣天賦極高，為萬獸體系中的智者，擅於謀略與經商，多分佈於草原。但由於肉質軟嫩鮮美，且整體實力較弱，容易被異族針對捕食，故通常將據點設於隱密處，或尋求友善異族庇護。

蜥蜴族：能以雙足站立，擅於使用真氣及各種兵器，也會進行建築、農耕、狩獵，族群內自給自足。多分佈於河川、沼澤。

彩鹿族：擅於以妖術施展幻術，有強烈的情感和藝術天份，與精怪體系親和度高，模樣嬌小可愛。可由於體質弱且肉質軟嫩鮮美，毛皮和鹿角又是珍貴材料，故遭萬獸體系大量獵殺、圈養，以致如今很難看到野生彩鹿族蹤跡。

現今多數野生彩鹿族居住於精怪體系領地，並受其保護。

—–玄階—–
恐龍族：肉體極強，真氣、妖力天賦也高，多分佈於草原或叢林。有眾多分支，喜食與居住環境對應的萬獸、水行、天行種族。

奇美拉族：肉體極強，真氣、妖力天賦也高，分佈於世界各處。個體差異極大，擁有神異的個體固有技，喜歡掠食各種萬獸體系來強化自身。

天蛇族：善於使用肉體、靈力、真氣、妖力，還擁有不俗的智慧，文明發展度高。淫欲、食慾極強，經常侵略其它種族，甚至包含人族、鬼族。

—–地階—–

貔貅族：擁有強大的命運和財富，族群雖戰力相對不高，但與其為敵者將遭受強烈厄運反噬，因此地位崇高。精通靈力與水屬性。

月蟾族：可自由縮放體型，靈力與妖力天賦高，壽命極長，喜歡隱居在深山中。主要以肢節體系為食。
