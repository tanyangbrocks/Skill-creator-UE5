# 靈玉

存放在魂中，乃魂之先天特性，可也有少數方法能後天改變。

靈玉璀璨度

璀璨度的影響

★單一屬璀璨度越高，其該屬性靈力天賦、靈力總量、該屬神通皆越強。

★璀璨每多1，靈力修煉速度、靈力、神通威能都加一倍。即金15所修煉金屬性靈力之速度，為金1的十四倍。

璀璨度數值

★一般而言，普通修士之五行靈玉的總璀璨度介於五和七，超過十的極少。

★一般只要單屬過三算天資優異，過五就算天才，過七是絕世天驕，十以上是妖孽。
總璀璨二十以上的千年難得一見。歷史最高紀錄是五十璀璨度。
