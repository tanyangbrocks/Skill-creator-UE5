# 源能

機械種族核心所使用的主能量，也適合運用於高科技產品，普通生物需經過特殊改造才能擁有源能。

源能最初被發現是存於地底「源礦」中，後續才發現機械族掌握大量源能。

源能力量性質與原理

源能是一種安定、精細、可被高效利用的新型能量。源能經過「超大型奈米數位積體源路」後，能產生極大量相異訊號，其效能和潛力遠高於電子機械，且源能的取得不會產生污染，傳輸過程完全零損耗。

此外，源能做為一種「能量體系」，亦能透過特殊方式讓生物體吸收、利用，不排斥任何能量，還可變幻出各種元素型態。

但由於源能只能從「源礦」和「數位次元」中取用，雖無污染，但取得難度遠高於電能，故科技體系傾向於利用源能為引導來強化精密電子科技。

源能力量分級

電動源能（X、Y、Z）

X電動源能之量級至少達1奎因，亦為最常見的源能量級。

核動源能（X、Y、Z）

X核動源能之量級至少足以媲美1克物質進行質能轉換之源能。

宇宙源能（X、Y、Z）

X宇宙源能之量級至少足以在純物質世界中形成恆星。

超時空源能（X、Y、Z）

X超時空源能之量級至少能單憑能量擾動時空，造成法則錯亂。

★每層源能依量級又劃分為X、Y、Z，其層級差距隨層級上升而上升。
