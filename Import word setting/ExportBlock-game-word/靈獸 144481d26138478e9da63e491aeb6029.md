# 靈獸

善用靈力，以「血脈」區分之等級（同樣分天地玄黃）。

根據史書記載，世間魔獸共有兩萬四千三百二十二種，以下為常見種族：

📌橙色文字表示對人類有攻擊性，需加以討伐

黃階血脈靈獸
靈豬、彈羊、靈茸鹿、歹猿、寒鴉、血蝠、跳跳馬、百靈狗魚、天翼靈鸚、牛角魔狼、大耳草球、寒毒箭蛙、草河獅、灼焰鯊、岩甲球獸、蛇虎獸、血面鬼魈
