# 仙界

為「神-成熟」之界面型世界，各界面修煉者達「道極」後，可嘗試突破界面屏障飛升至仙界。

仙界之界面層次、法則等級與神界同等，但其中大多生靈未達神之領域。此界之法則多由修煉者直接掌控，其中又以「道祖」為主，但「道祖」仍會被「至高神壇」約束。

📌仙界僅有修練體系，且生靈間實力差異極大。

仙界格局

又分為無數「仙域」，每仙域有獨立之生態、環境，可視為一方小世界。

由於仙界空間實在過於廣闊，故難以計算其內部共有多少族群、多少勢力。
