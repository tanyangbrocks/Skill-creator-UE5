# 天官鬼老的秘密

天官鬼老乃上古惡鬼，對人類而言是宛如天災般的噩夢。似乎從有歷史記載的起源就有他的存在，因此沒人知道他到底活多久（他自己也忘了）。

性情轉變的原因

其實他意外愛上了正太版的拉普拉斯。曾想要捕食，卻因其逆天絕美童顏而下不了手。

後來拉普拉斯變為女性時，他發現自己也不排斥。就這樣，天官鬼老為追求拉普拉斯，應對方要求禁食人肉，並持續安定至今。

內心封印

然而，天官鬼老的本性仍是暴虐殘忍，因此他給自己下了「天官十二禁」來封印之。將自身暴虐之心與狂暴力量封印住。

胃裡的不滅族—小耀

天官鬼老曾捕獲一名不滅族男童—小耀，並將其吃下肚後持續消化。

雖然無法品嚐味道，軀體卻能不斷吸收小耀的血肉，偶爾也會將其吐出，重新啃食小耀再吞下肚。而年幼且精神脆弱的小耀也陷入了難以想象的、永恆的痛苦折磨。

-

小耀原本是名個性膽怯怕生，一點疼痛就會恐懼大哭的纖細稚嫩男童，卻身懷奇異肉、還擁有先天道體，是不滅族中萬年難得的天才。天官鬼老進攻不滅族時，年僅五歲的小耀被重點追殺，最終被天官鬼老殘忍地啃食。

（再看之後他要不要被救）
