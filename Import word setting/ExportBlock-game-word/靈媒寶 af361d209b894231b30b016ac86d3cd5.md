# 靈媒寶

**符籙**

消耗品，將術法神通以「靈紋」形式刻印封存於符籙，可大量囤積靈力、神通。

符籙分級

凡灰、靈裊、封地、天照、道勾、無字

結構

封存靈力的「符紙」、蘊含力量的「靈紋」、引導靈力的「靈墨」，甚至畫符的「靈筆」也會影響符籙效果。

符籙師分級

凡灰、靈裊、封地、天照、道勾、無字

★個別對應其最高能完成的符籙等級。

★但每級又細分三星來衡量其熟練度、製符速度和符籙品質。

常見符籙

護罩、聚靈、五行術法、控物、封印……

特殊符籙
符器：將符籙畫在物品、甚至人體上。

符寶：將法寶威能以符籙的形式留存。

**陣法**

佈置陣法，或將其刻印在物體上，甚至有「符陣」（以符籙發動陣法）。

分級

始陣、靈環、地界、玄天、道鐫、無徑

結構

「元」為引、「徑」為導、「徑」通「迴」、「迴」生「陣」。

★元的種類和分佈影響陣法品質、徑的分配影響陣法效率、迴的數量影響陣法威能。
★可僱用守陣師、陣奴來確保法陣安全。

陣法師分級

始陣、靈環、地界、玄天、道鐫、無徑。

★個別對應其最高能完成的陣法等級。

★但每級又細分三星來衡量其熟練度、佈陣速度和陣法品質。

★法道的「陣法」與魔法的「法陣」功用類似，卻比法陣還艱澀難懂。因此許多法道修煉者會轉用「法陣」。

常見陣法

洞府：聚靈陣、結界、禁制、陷阱殺陣、幻陣等。

戰鬥佈陣：劍陣、聚靈陣、幻陣、邊走位邊速成的殺陣等。

其它常見陣法：傳送陣、封印陣、大隱匿陣、消音陣、獻祭陣、召喚陣等。
