# 䵻龗（最強至高能量）

異源複合能量：

---

䵻龗：音ㄍㄨˇ ㄌㄧㄥˊ，違背「靈魔互斥定律」後，使靈力、魔力相結合之產物。

由於通常只能在「龍族」身上看見此力量，故又稱其為「䵻龗」。

該能量不僅異常強大，還能完全替代靈力或魔力。（靈力強大的修煉者只要成功融合一點魔法，就能成為大魔法使）
