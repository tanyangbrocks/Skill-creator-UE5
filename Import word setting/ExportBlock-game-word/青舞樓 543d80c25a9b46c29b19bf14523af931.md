# 青舞樓

修煉體系民間最興盛的娛樂產業之一。

由於修煉體系人人可練武，也有諸多修煉者，女性未必皆是較為嬌弱的一方，男性也未必要比女性更有尊嚴。

青舞樓常見有以下三館：

花玉館

成員以12~25歲的美麗少女為主，分為嫩花、嬌露、霜月、天裳四階級。

葉秀館

成員以12~25歲的清秀少年為主，分為嫩草、軟枕、玉白、仙鶴四階級。

幼嬌館

成員以5~12歲的可愛幼童為主，分為嫩生、奶糕、冬甜、美雀四階級。

-

📌三館比例相差不大，服務項目皆含歌舞、琴棋書畫、角色扮演、陪酒、聊天、調情、性交等，只要不傷害或羞辱成員皆可。（若造成損傷，則需付取物價購買）

成員階級越高，工作性質越傾向賣藝而非賣身，除非客戶出更多錢。

📌每位成員皆有標示「取物價」，客人付出足量金額就可將其買走，對於越有才華、情趣、容貌越美的成員，所需取物價也越高。

---

按理說，青舞樓成員多為自願入樓，但也有少數犯罪組織會與各大青舞樓進行交易。由於蒼究人口販運猖獗，此事難以追查。

由於百姓生活困苦且經常練武，性慾普遍高漲，故情色產業十分盛行。

部份青舞樓可從特殊管道取得修煉者的美顏丹藥，故成員的顏值與體態和常人完全不是同級別，能引起眾人狂熱。但也可能因過度嬌嫩絕美，引來鬼族獵捕。
