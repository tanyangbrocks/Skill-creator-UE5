# 鬼族

妖力天賦高，外型千奇百怪，通常極為畸形、噁心，類似恐怖片中的詭異怪物，類似虛，軀體可能有類似人臉、人手的多重構造，高階可能具魔。多生活於叢林、河川、峽谷，以及地淵，部份高階鬼族能潛入人類社會。

鬼族為人族之伴生種族，雖不吃人也可存活，但吃人為其天性。他們吃人可使自身不斷進化，外表也越接近人類。

大多數鬼族剛誕生時沒有靈智，但吃足夠多的人族就可開啟。

會說話的鬼族都也使用人族語言，但三種都有可能。

種族派系

紫鬼：數量最多，能使用強大妖力，食人慾望格外強烈，總可在短時間內獵食大量人族。實力居中，但擁有諸多詭異機制，也擁有最強大的尋找獵物能力。

赤鬼：實力、體力最強，外型與人族最相似，對人族性慾旺盛，無論男女老少都能引發其不同程度的性慾，同時也是鬼族持續進化的關鍵。整體而言最令人感到恐懼。

青鬼：通常智商較高，更易開啟靈智，喜玩弄、驚嚇、折磨人族後再緩慢享用。通常在早期就能獲得「換皮」或「寄生」。低階時雖實力最弱，但能以特殊手段捕食弱小人族個體，反而最容易進化至高階，是普通人的噩夢。

異鬼：不屬於上述派系之個體，通常為基因突變，亦沒有固定常見型態。

正如同其它生靈有「性別」之分，鬼族後代為何種派系乃完全隨機。

鬼修

約有百萬分之一的鬼擁有足夠強的靈玉，能修煉仙道。鬼族修煉者能透過捕食人類來提升靈力修為，如天官鬼老。但他們無法藉由辟穀來降低食人的慾望。

由於鬼修的實力、成長性、隱匿和逃跑能力都遠高於正常鬼族，鬼族領地也缺乏修煉資源，故大多會暗中潛入人族社會。

其大量捕食人類就能同時提升種族能力和修為，很容易成為匹敵魔鬼、王鬼的強者。

鬼族進化階級

鬼族只要攝取足量人類血肉就可增強力量並進化，進化階級為……

獸鬼：最原始之鬼族，實力約超凡級～征戰級。

半獸鬼：食用少量人體（約5~15普通人），即可進化至半獸鬼，實力約征戰級～主將級。

微智鬼：開啟部份靈智，實力約主將級～群星級。

智鬼：完全開啟靈智，實力約群星級～耀日級。智鬼以上的鬼又被稱為高階鬼族，智鬼以下則是低階鬼族。

——巔峰級以上——

魔鬼：具強大魔力、妖力、精神力，且通常智力極高。

王鬼：身體能力、智力極強，且非常難被殺死。

鬼族的共通技能

初階能力（出生即擁有）：

超速再生：在不破壞臟器的情況下，能迅速再生器官、組織，擁有遠超蟑螂的頑強生命力。隨分裂生殖能力增強，甚至連臟器都能修復。

分裂生殖：在不破壞臟器的情況下，切下1/3～1/7之間大小的軀體能夠成長為新的個體。隨分裂生殖能力增強，軀體大小範圍越廣，甚至破壞臟器也能生殖。但高階鬼族由於生殖能力下降，會導致分裂生殖能力逐漸消失。

敏銳五感：相對於人類，視覺及觸覺最低是人類平均的30倍、聽覺100倍、嗅覺3000倍、味覺5000倍。

隨著敏銳五感增強，這些數值還會在提升！

其中，鬼對人的體味尤其敏銳。正常來說，在普通人察覺到鬼之前，鬼一定會提前察覺到該人的存在。

此外，它們能夠精準分辨所有人類個體的體味、音色、觸感、模樣乃至口感滋味等細節，且永不遺忘。

肉體變化：軀體能在一定程度自由扭曲變形、縮放，以便捕獵或躲藏。

精神波：鬼族狩獵時，能發出精神波動，將自身的慾望、邪念與惡意露骨地傳達給目標，令目標更驚恐。

恐懼鬼氣：散發出讓類人物種極度恐懼的鬼氣，普通人會當場嚇到失禁並無法動彈，並喪失戰鬥、逃跑的意志，部份能讓人無法發出聲音。

續命唾液：讓被啃食的獵物不容易死亡，還能猛烈刺激神經，大幅加劇獵物神經敏感度、意識清晰度和精神恐懼程度，並讓獵物時間感加速3000倍以上。

其中，獵物不會因失血過多而死亡，續命唾液會維持細胞運作所需養份，並大幅加強神經訊號，也源源不絕地傳遞極致恐懼情緒。獵物無論承受再大的痛苦都不會失去意識或崩潰，反而思緒會因恐懼而更加清醒。

ps：鬼族續命唾液經常被人類用做拷問工具，但由於太過殘忍，被許多區域禁止使用。「被鬼吃掉」對人類而言是已知最痛苦的死法，且隨著續命唾液的強化，獵物感受到的痛苦會無止盡增強。但即便如此，每日仍有大量人族被鬼族獵食。

換皮：將獵物的人皮完整剝下後穿上，並偽裝獵物的外貌、舉止與聲音。但由於不具備獵物記憶，和他人交流時容易被識破，只能用於降低獵物戒心，或純粹折磨獵物。

中階能力（獸鬼～微智鬼可能擁有）：冷熱抗性、毒抗性、鋼皮……

分裂自爆：自行自爆為無數碎塊，過段時間後可再聚合。

寄生：可寄生在人類身上，啃食其內臟並操控其軀殼。微智鬼以下鬼族若成功寄生人類，將提前獲得靈智及對方記憶。

高階能力（智鬼以上始擁有）：地淵次元門、魔法抗性、靈力抗性、精神抗性……

模擬人類：完全變為人類的模樣、聲音、氣息，無論人族、鬼族都能騙過。

噬靈：吃人時可選擇將該人的靈魂一併吞食，可將其轉化為鮮美養份，或讓該人魂魄囚禁於體內，承受永恆的痛苦、恐懼與折磨。

📌鬼個體獲得中、高階能力的時機不同，且每個體的每種能力強度不一，也會隨實力增強而提升。
