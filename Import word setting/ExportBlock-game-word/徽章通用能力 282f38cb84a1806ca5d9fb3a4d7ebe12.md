# 徽章通用能力

個人資料

將能量注入徽章，可投影出一個記載個人資料的發光面版。個人資料面版有一定個人化功能，可自行設計。

展示階級

持有者可注入能量讓徽章大放自身顏色的光芒，並釋放出該階級的氣息。在一定限制內，注入能量越多，綻放的光芒和釋放的氣息越強。

ex：耀日級徽章持有者能讓徽章綻放耀眼的紅色光芒，並釋放耀日級氣息。

聯繫工具

每個徽章都有專屬編號ID，可藉由撥號、加聯絡人的方式和其它超界行者進行語音通話。類似手機。

此外，超界聯盟也會發重要公告給徽章持有者。當有人求救時，超界聯盟也可能會將求救訊號發給附近的超界行者成員。

求救

徽章持有者可發送自己的位置座標給超界聯盟總部，用以確認自己位置，或請求支援。

然而，徽章等級越低，求救訊號的強度也越低，容易被各種結界、環境屏蔽訊號。

★位置座標必須由徽章持有者主動發送。為顧及成員隱私，超界聯盟無法自行追蹤徽章位置。
