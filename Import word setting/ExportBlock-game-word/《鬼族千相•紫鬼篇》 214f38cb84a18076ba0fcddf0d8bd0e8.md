# 《鬼族千相•紫鬼篇》

📌低階鬼族個體除了本身型態所固有之能力外，還會擁有自身獨特S鬼域。

[鬼域](%E9%AC%BC%E5%9F%9F%20a5f2afa1b6f8492b84d69e9816b7ec77.md)

📌此處所述為尚未經分裂生殖之原始型態，分裂生殖後能力會自然削弱。

以下紀錄常見型態，這些型態能力較為強勢，故可大量繁殖：

雜體型（獸鬼）

完全由人類的眼、耳、鼻、口、手臂、腿、毛髮、骨骼所組成，通體紫色，組成方式沒有一定架構和規律，具有妖力，但多數只會發射妖力彈和凝聚妖力屏障。除了鬼族共通能力和專屬鬼域外，沒有其餘能力，戰力相對低於其他型態。分佈範圍極廣，低海拔陸域皆有其蹤跡。

最常見的紫鬼，但由於經常群體行動，反而是大威脅。即便是落單的紫鬼，也要先試探其S鬼域之能力。

指型（獸鬼）

軀體為長滿眼球的巨大紫色人類手指，可能是五指任何一指，也可能是兩指以上黏在一起。發霉的指甲上長滿漆黑長髮，以45°翻開，指甲與指甲肉間長著佈滿利牙的血盆大口，會從中伸出四到五根舌頭捕捉獵物，或製造巨大的吸力來吸食獵物。欲移動時，體表眼球上會長出慘白的細小嬰兒手指來移動。會使用妖力彈、妖力屏障。多分佈於人類居住地附近平原。

指型的身軀會超速再生，必須摧毀指甲蓋上方持續跳動的紫色大腦，但其在被摧毀的瞬間會產生劇烈爆炸。許多初次遭遇指型的超界者以近身打擊、斬開大腦，最終與其同歸於盡。

混手型（獸鬼）

軀體為不停蠕動的網球大小紫色肉塊，長滿大量形似人類的慘白嬰兒手臂，手臂能夠快速伸縮扭曲，並於其上長出眼、鼻、耳、口、舌尖刺與利齒等器官來探查、捕食、囚禁獵物。能以手臂快速爬行，手臂被破壞會迅速再生，只有肉塊被破壞才會死亡，但四周手臂會本能地保護中央肉塊。多分佈於荒野、森林，平時會縮起手臂，將自身埋入泥土、樹叢或建築縫隙中。體內有妖力但不會運用。

混手型相對不難對付，可一旦入侵人類居住地，將會在極短時間以手臂捕食附近所有人民，導致無人有機會求援或通報。而其本體也會在屠村、屠城後快速進化成更恐怖的存在。

巨眼型（獸鬼）

軀體為一至三顆長紫色翅膀的大眼球相連，大眼球上遍佈數百粒小眼球，體表每處都能裂開成血盆大口，並瞬間長出堅硬銳利尖牙，故任何敵人的實體攻擊都能在一定程度阻擋並反制。視力極好，飛行速度如子彈般飛快，部份可使用幻術、催眠、妖力屏障、速度提升等基礎妖術，比較明顯的弱點僅有高強度遠程能量轟擊，但巨眼型會優先鎖定正在凝聚能量的個體攻擊。多居住於洞穴，但能鎖定遠方人類並前去獵食。

即便是最弱的巨眼型也往往能瞬間秒殺普通超界者及冒險者小隊，極度棘手，需委託專業獵鬼士處理。

大腳型（半獸鬼）

軀體是一隻如高樓般的紫色人類巨腿，體表遍佈大量眼、鼻、耳。中央有巨口，能從中釋放大範圍精神波，讓周遭人類在意識清醒下，四肢不受控制地走入牠的巨口。若有人擺脫或無效化其控制，其體表的鼻毛、睫毛會迅速伸長並攻擊、捕捉獵物後送入口中。能夠緩慢化做粉塵並抹去氣息，隨後在附近人最多的區域凝聚，沒有其它移動手段。若遭遇致命威脅，會加快化作粉塵的速度，向遠方逃遁。

本身戰力不高，但需有抵抗精神控制手段才能對付，普通超界者在野外遭遇極度危險。經常在不知不覺中吃光多座村莊，故有些村莊會留有立即通報超界聯盟的珍貴器具。

異網型（半獸鬼）

軀體是由紫色細長人類舌頭編織成的大網，常黏著於森林樹木間，網上分泌大量續命唾液和強力黏液，組成網的舌頭會快速噴射出更多舌頭來捕捉獵物，一旦獵物被網黏上，網會迅速噴吐大量舌頭將獵物裹成繭，並於繭中射出銳利舌頭切割、貫穿、攪拌獵物再迅速消化，而網本身會藉由舌頭黏貼物體來移動，在森林中能靈活移動。會使用妖力彈及妖力屏障，也會用妖力大幅提升速度。多分佈於森林、山地。

所有舌頭都會超速再生，必需用燃燒、冰凍等能同時對整體造成大面積傷害的方式來處理，然而舌頭網隨時在快速變形且本身體積較大，故招式範圍要足夠廣，或是需要多人同時集火討伐。

複足型（半獸鬼）

頭部是肉瓣狀張開，且每片肉瓣與中央長滿無數利齒的血盆大口，這樣的頭部有三至五個相鄰。軀體是紫色條狀肉蟲，肉蟲有堅硬外骨骼，側邊長滿慘白的嬰兒腿，背上遍佈漆黑的人類頭髮。

複足型察覺敵人將至時會先用肉瓣鑽入地底，隱匿氣息，隨後從目標腳下驟然衝出，以肉瓣巨口突襲捕獵。會利用嬰兒腿高速爬行、衝刺、跳躍，背上的堅韌頭髮會化為銳利觸手穿刺、切割、捆綁目標。全身有超速再生，必須摧毀深埋在體內的大腦才可擊殺。

能同時發射大量妖力彈，凝聚妖力屏障，釋放使人被催眠、肌肉萎縮、產生幻覺、反應遲緩的妖力能量場。

雖沒有特殊機制，但移動靈活、攻勢猛烈、進攻手段極多，地底突襲也難閃躲。若複足型具備靈智，戰力就相當於微智鬼。

禁魔丸（獸鬼～半獸鬼）

軀體為一顆長有嘴巴的黑毛球，表面長出許多扭動的紫色孩童手指。只吃人類血肉，若長期沒進食人類或消耗過大，毛髮會變成灰色。

其本身無戰鬥力，亦沒有自主捕食能力，但能消耗精力展開一定範圍的禁魔領域（領域中雙方皆不可使用魔法），若給牠餵食更多獵物，領域將能擴張。

主要由強大鬼族或高階鬼族餵養、攜帶牠，用來突襲以魔法作戰的隊伍。
