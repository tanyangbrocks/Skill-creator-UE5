# 源導武器

源核武器（蒼究之陽）

利用源能包裹、儲存核融合之能量，形成核融合彈，為科技體系最恐怖的大殺器。

其製作技術、原料受到泰克諾爾高層嚴密控管。

PS：純粹源能的引爆威力也很強，但會耗費太多源能，不實用。

源壓熱劍

高密度源能之光劍，可精確斬斷物質、能量，極其昂貴。

源壓激槍

以高密度源能做為子彈的激光槍，可精確穿透物質、能量，極其昂貴。

源光盾

高密度源能之光盾，可阻擋物質、能量。

源科技

利用源能打造的高科技產品，但仍以「電」做為主要能量供給，在泰克諾爾十分常見，其它部份地區也會應用。
