# 鬼族的繁殖

1.雙鬼交：鬼族沒有性別（但人化後可能因喜好而賦予自己性別），任意兩隻鬼族即可交配生產。但赤鬼通常只想與類人交配，紫鬼與青鬼的各自慾望也遠勝交配，故通常需高階鬼族刻意安排。

單次生產能誕下三十~五十個孩子，通常每半年可生產一次。

但若鬼族族群缺乏食物，就可能吃掉部份新生兒。

2.分裂生殖：鬼族可進行分裂生殖，但每次分裂產出的鬼族都會越變越弱。分裂後，原先的自我會完全消滅，新生個體將各自分得原生體部份自我。

3.鬼人交：鬼族可和類人體系交配（不論男女），尤其赤鬼對此有強烈慾望。赤鬼的體液都有續命唾液之功效。

交配後，生產者一律為人類方，無論年齡、性別等生理狀況都會強制懷孕，但若是年幼的孩童或老者可能因無法供給養份而死亡。

最終誕下的孩子有60%是鬼族（且較為強大）、30%是類人方種族、10%是鬼人族。

若腹中胎兒為鬼族或鬼人族，會在發育成熟前啃食母體的內臟，撕裂母體肚皮而誕生。鬼族或鬼人幼胎在誕生的同時會分泌大量「續命唾液」，這將加劇母體的痛苦和恐懼，但卻能使其不會死亡，並協助其傷口癒合。

若腹中胎兒為類人方種族，則只能進行剖腹產。

4.高階鬼族的後代

智鬼以上鬼族的繁殖能力下降至與人類相似，但後代一出生就是智鬼以上的高階鬼族。
