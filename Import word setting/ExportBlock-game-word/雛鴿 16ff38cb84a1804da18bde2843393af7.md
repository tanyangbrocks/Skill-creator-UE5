# 雛鴿

以綁架、販賣孩童為主要目標的犯罪組織。

雛鴿首腦

---

雛鴿目標&分佈

擁有十幾個大小據點，東域、南域之據點較多，西域、北域較少。

部分據點會偽裝成孤兒院、教會、公益團體、學校等。

目標以0~12歲孩童為主，亦含13~18歲少年。多綁架貧民窟中孤兒，隻身在偏僻處閒晃、走丟的孩子，或因戰亂流離失所的孩童，以避免引起注意。

被綁架者多數會先在據點遭到性侵、凌虐、囚禁，有些甚至被殺害或充當食物，之後才轉賣去各處做為奴隸、性奴、活祭品、娼妓、勞動力，或直接出售給鬼族。

也有部份孩童會成為A片素材，或以毒品控制、洗腦後，直接成為雛鴿新成員。現今雛鴿多數基層成員都是曾被綁架的孩子，但其心智已被徹底改造為罪犯，並成為重度戀童癖者。

由於雛鴿和鬼族及部份高官、貴族有合作關係，其據點相對不容易被搗毀。
