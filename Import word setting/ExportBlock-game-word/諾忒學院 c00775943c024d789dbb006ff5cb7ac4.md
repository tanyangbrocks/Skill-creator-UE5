# 諾忒學院

諾忒學院是諾忒城最好的修煉者學院，不僅能提供加速修煉的寶地、強大的術法神通，還能結識各地的修煉天才。

-

可進入諾忒學院的年齡是12～15歲，最低門檻是靈力達到準煉氣期之60%。

-

進入諾忒學院後，新生指導員歐老師帶你們前去認識學院內的各大特色與資源，如修煉聖地化神血池、祕境天濛山、仙境桃花源、萬億絕滅迷霧大陣，以及儲備底蘊深厚的藏經閣、靈寶樓等。

諾忒學院特殊規矩：

📌學院每年歲末時分會有一次考核，要是考核不達標，將會被強制退學！（用以將有限資源提供給最優秀人才，退學率約50%）

📌每年新生需要接受學長們的群毆（下馬威），來挫銳氣。

📌每年年中會有擂臺戰。
