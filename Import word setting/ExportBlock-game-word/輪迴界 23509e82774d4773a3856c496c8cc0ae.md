# 輪迴界

又分為天堂、通魂市、轉生殿三處。

天堂

由於特殊法則，生靈之「魂」的壽命將被完全凍結。

厭倦天堂生活後，也可選擇離開天堂，但離開天堂後無法再次歸來。

通魂市

白魂區

由於各界不時會有「魂」消亡，冥界也會生產全新的「魂」補充。

外魂區

內部為已經入世過卻又死亡之魂魄，部份為暫時停留、等待轉生之魂；部份則為當前為未足轉生規範，無法轉生之「隱魂」。

外魂區構造

-

當外魂區的「隱魂」在每次輪迴中經歷之生靈都遺忘其存在，將會真正地魂飛魄散，永久消亡。

轉生殿

有轉生資格者進入此地，與各大界域由「奈何橋」連接，負責處理一切轉生事務（包含特殊之跨界轉生）。

轉生流程：

1.欲轉生者排隊領取號碼牌，叫號後前往專屬會面隔間，由專職死神服務。

2.依據現有「價值」總量決定可轉生之界面、物種、身份、天賦等。

（選擇後，價值不會因此消耗，所以通常每次轉生之際遇都比前次好）。

3.最後，喝下「孟婆湯」後洗去記憶、將所有靈魂外顯資訊存入深層區域，並踏入「來生門」進行轉生。

轉生規範：

1.每次進行轉生時，所需求之價值門檻逐步提高，未達價值門檻者不會被「摩加」允許轉生。

2.「魂」最多可轉生九次，此後該魂無論如何都不可經由「摩加」的審判來進行轉生。

轉生六道

轉生界面主要分為……

三善道：天道、修羅道、人道；

三惡道：餓鬼道、畜生道、地獄道
