# 經脈&竅穴（武道為主）

人體之氣血通路分為「十二正經」、「奇經八脈」以及其各自的「絡脈」，而經脈中的細小節點則為「竅穴」。

「竅穴」為原先為半封閉狀態，將其完全沖開後可大幅提升氣血循環，並排除體內雜質、強身健體。「竅穴」僅能用真氣或混合真氣之能量來沖開。

十二正經：

分別為手三陰經（肺、心包、心）、手三陽經（大腸、三焦、小腸）、足三陽經（胃、膽、膀胱）、足三陰經（脾、肝、腎），其上共有三百六十五處竅穴。

奇經八脈：

奇經八脈包括督脈、任脈、沖脈、帶脈、陰維脈、陽維脈、陰蹻脈、陽蹻脈，其上竅穴較難沖開，但沖開後有洗經易髓之神效。

任督二脈：

屬於奇經八脈的特殊兩脈，「任脈」共有二十四竅穴，分布於面、頸、胸、腹的前正中線上；「督脈」共有二十八竅穴，位於背後中脊，兩者下交於會陰之間，上交於唇。

兩脈上所有竅穴全數貫通後，可極大加強體魄，甚至讓靈力運轉更快，打通後將成為武道、仙道雙修者的大優勢。

竅穴相關技術—點穴：

武道人士能運用真氣點擊敵人穴位來使人無法動彈（止穴）、狂笑（笑穴）、虛弱（力穴）、無法出聲（啞穴），甚至當場死亡（死穴）。

但真氣渾厚者能自行沖開堵塞穴位，擁有靈力、魔力、妖力或強大精神力者更可直接以能量化解點穴效果。
