# 論道

感悟「道」

修士可透過感悟萬物來理解其中蘊含的「道」，而修煉界的「道」即為「法則」。

雖然修士必須抵達「道極」境界才可掌控法則之力，但道極之前的修士也能透過感悟大道來強化對自身力量的理解，從而提升對應實力。

例如：使用金屬性靈力者可感悟金之大道，劍修可透過感悟劍修來強化劍勢，法道修士感悟神魂大道強化靈識……

論道

感悟「道」的方式有很多，如觀看或進行戰鬥、修煉中產生思緒、奇遇中觀想壁畫或石碑等，但最常見的感悟「道」之方式為「論道」。

論道即是複數修士們（至少兩名）聚在一起，彼此討論自身對某大道的心得，從而獲取體悟與思緒。此外，廣義的論道也包含討論修煉心得、特殊功法、心境調整，對仙途的看法等。

當修士透過論道獲得思緒後，通常需再花費一定時間消化思緒才能完全領悟。
