# 寵獸御獸師

常見種類與特殊事項

1.利用召喚術和魔力，搭配「契約羊皮紙」與魔寵簽訂契約。

[召喚術](%E5%8F%AC%E5%96%9A%E8%A1%93%204ce1fdf23b874f3c93ae33a3fb574371.md)

2.利用精神力和靈力，搭配「御章」，與靈寵簽訂契約。

[法道（識海）](%E6%B3%95%E9%81%93%EF%BC%88%E8%AD%98%E6%B5%B7%EF%BC%89%202f1d95bdc702424699350aa38db85158.md)

3.與種族生靈簽訂之眷屬契約，與妖獸或其它體系生靈簽訂契約，亦可歸類為寵獸御獸師。

★相比高靈智的種族生靈，靈智相對低下的魔獸、靈獸、妖獸更容易簽訂契約成為寵獸。

★有靈智的種族生靈也可自願簽約，簽約後的種族生靈稱為「眷屬」。

---

寵獸御獸師協會

儘管是以不同體系建立契約，但契約後所需之特殊攻略，優勢和困擾大致相同。故寵獸御獸師們抱團取暖，成立寵獸御獸師協會，以便同行快樂御獸。
