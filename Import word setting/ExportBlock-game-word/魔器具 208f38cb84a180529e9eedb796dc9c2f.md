# 魔器具

一切應用到魔力的物品皆可視為魔器具。

魔器具種類

魔杖

[魔杖](%E9%AD%94%E6%9D%96%2021bf38cb84a180239862faaa9d5a4d35.md)

[魔陣之杖](%E9%AD%94%E9%99%A3%E4%B9%8B%E6%9D%96%202ddf38cb84a180c19c06dcf0e910f6f6.md)

魔裝

蘊含魔力的裝備，如魔劍、魔盾、魔甲、斗篷、法袍、巫師帽、飛天掃帚等，為最常見魔道具，多利用對魔力物品的處理和「法陣」技術來製造。

魔道具

[魔道具](%E9%AD%94%E9%81%93%E5%85%B7%202def38cb84a180a2a380ece7dddb6e39.md)

魔儀

為特殊用途創造的魔力儀器，如魔力測定儀、結界碑、瑪納探測器、魔力大螢幕，多利用「魔紋編程」技術來製造。

---

魔器具常見製造技術

魔器具除了對魔力物品本身的鍛造、提取、切割、融合等，也經常涉及煉金術、法陣、魔紋編程等技術。
