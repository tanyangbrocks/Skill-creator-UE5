# 蒼究世界簡介

蒼究世界，乃諸天萬界中備受恩寵與祝福之半神界域。
生靈可透過修煉六道昇華生命層次、透過支配六法掌控多元能量、透過奮力拚搏締造繁華文明。
這精彩豐富的資源體系造就了種族的多元與強大。

-

然而種族似乎多元、強大過度了，混亂的資源體系使物種間數次爆發大戰導致生靈塗炭，乃至世界毀滅。
為避免世界混亂或種族失衡，蒼天決定採用「伴生種族」策略，令各種族兩兩一同誕生，並設下獨特的相互制衡機制。
此後，世間伴生種族生來便相互制衡，如龍與鳳、精與靈、光與影、天使與惡魔，以及——
人族與鬼族。

-
當然兩族間制衡情形嚴重，人族各體系成長皆需用上鬼族軀體材料；而鬼族喜以人族為食，並以此為主要提升途徑。兩族自誕生之初便是死敵！
人鬼兩族爆發無數次大戰後，情勢才逐漸趨於穩當，發展出各自繁榮的領地。

-

人族領地主要分佈於天輝大陸，其中又由三大勢力為主宰，分別為：
西域的「永恆聯邦泰克拿勒」！
南域的「神聖皇朝戴內色堤」！
東域的「若夢埃頌雷特聖福林大天帝國（簡稱六龍聖天帝國）」！
而三大勢力聯合其它人族小勢力又結為「天輝星盟」來共同抵禦妖族。當然，表面是結盟，實則內部暗潮湧動，並不團結。

-

今日，有道不同凡響的特殊靈魂自九天之上凝聚，蒼天隱約感應到它的未來，竟派出七大天使長為其歌頌，使其綻放之耀芒愈發燦爛。
這道璀璨奪目的特殊靈魂正緩慢飛向六龍聖天帝國境內......
而它，就是你！
