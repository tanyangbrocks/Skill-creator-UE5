# 種族間特殊關係

血脈優化

部份種族會派達到金丹期的成員改變軀體構造，並與特定對像聯姻。

聯姻後不僅兩族關係較好，後代也可能因此變得更強，達成種族血脈的優化。

同域共生

通常各族都有屬於自己的領地，不可肆意侵犯。但偶爾也會出現不同種族共處一地，互利互惠的場景。

常見伴生種族制衡
人&鬼：強制相互獵殺關係，制衡力強。

翼神龍&瑞神龍：本能爭奪龍的正統性，制衡力強。

鳳、天球：本能爭奪空域霸主，制衡力中。

巨人&泰坦：本能爭奪大地霸主，制衡力中。

犬&貓：理念相反關係，但如今對立不明顯，制衡力弱。

光&影：相互威脅存在關係，制衡力強。

樹妖&花妖：搶奪新生成員、氣息不相容關係，制衡力強。
