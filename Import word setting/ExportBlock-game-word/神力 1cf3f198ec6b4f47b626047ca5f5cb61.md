# 神力

[神界基本設定](%E7%A5%9E%E7%95%8C%E5%9F%BA%E6%9C%AC%E8%A8%AD%E5%AE%9A%209c0ee12bbc7c491289a18d200cc78dec.md)

神力概述：

📌由價值所構成。生靈死亡後若達到神之領域，將依照其價值產生相應神力。

📌擁有神位之「天神」以神力行使掌管森羅萬象之權限。

📌神力的源頭必定來自神界，下界無法靠自己的力量創造出神力。

神力操作：

神力在下界幾乎能辦到不涉及「法則」的任何事，也能模仿能量階級低於神力的任何力量。

以神力發動的攻擊在下界是毀天滅地的存在。因此一旦下界出現神力，會第一時間被神界感知並監視。

-

神力屬於「世界六意」體系，沒有和其它能量混合的正規體系。

但強大的神力使用者也可能開發出神力混合魔力、靈力等能量之形式，其使用法完全因人而異，神力也被譽為最難以操控的能量。

神力使用者：

除了神本身以外，神在下界之使徒也能擁有神力。

神的「轉生體」、「降臨體」或是神待過的環境、神使用過的器物都可能擁有神力。

偶爾也會有凡間生靈意外獲得神力，但其源頭必定來自神界。

神力戰鬥：

神力使用者對決時，勝負通常取決於雙方的使用技巧、使用經驗。許多活了千年萬年的古老神也不敢說自己能完美駕馭神力。
