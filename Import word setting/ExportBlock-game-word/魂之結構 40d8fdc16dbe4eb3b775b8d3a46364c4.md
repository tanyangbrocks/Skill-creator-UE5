# 魂之結構

魂幽體：（最外層）

儲存並維持「魂」結構、形體之構造，主要由「魂力」凝聚而成。

魂精環晶絲：（外層）

由無數晶絲編織而成，記錄生靈當前之生命能量、記憶、性格情緒、思想、超能力等，比較容易受外界誘導而產生變化，也易因死亡、轉生而清除。

靈轉星基質：（中層）

記錄「魂」之當前生靈物種身份、技能、靈玉、生命層次等烙印在靈魂較深處之資訊。

部份可能經由死亡、轉生而被消除或改變，但通常無法被直接干涉。

源界神憶核：（內層）
記錄「魂」之累積價值、轉生資訊等最細節資訊。除非將「魂」直接破壞，否則絕對無法改寫、干涉或抹除內層源界神憶核之資訊。

---

魂之構造關係

1️⃣「魂幽體」中飄散許多紀錄思想、情緒、生命能量等容易變化的「魂精環晶絲」。

2️⃣「魂幽體」最中央有大量「魂精環晶絲」纏繞成球狀，內部儲存許多「靈轉星基質」。

3️⃣「靈轉星基質」內部有顆無法被干涉的「源界神憶核」漂浮其中。
