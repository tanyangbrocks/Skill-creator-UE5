# 拉普拉斯（哈沃●蘭斯洛）

本名哈沃●蘭斯洛，乃戰時貧苦農家百姓。後來自封為「拉普拉斯」，乃「知識、智慧」之意。

擁有相異性別之雙重人格，男性內向膽怯、善良體貼、厭惡戰鬥；女性性情暴躁、陰險狡詐、熱愛近身博殺，而兩性格都對知識與書籍有強烈渴望。

其外貌也會隨人格變換：嬌羞白毛正太（善遠程魔法）/火辣金毛御姊（善近戰武裝）。

因愛好和平、欲終止戰亂，故和人祖一起組織「淒白輓歌」。最終成功平定亂世，還為人族魔法建立嚴謹體系，如拉普拉斯標準直擊魔法等。雖然早已具備成神資格，卻至今仍留在人世，處理魔法塔、魔法研究等事務，偶爾也會外出欣賞和平的世界。

雖然當初只是魔法師，但他貫徹活到老、學到老的理念。如今已達到九環九漩、聖魔靈；還透過各種方式融合了十二種體系能量，並全部達到巔峰；甚至開啟二十四種超能力、去人道次元族習得「概念」之力、獲得四種「超格技能」、成為星龍鳳神的使徒並獲得大量神力，實力已可力抗「中位神」，乃當今世上最強者。
