# 召喚術

分為眷屬召喚、具現化、異界召喚、反召喚。

眷屬召喚

召喚自身眷屬，通常搭配「契約羊皮紙」等魔具

召喚對象

種族生物、寵獸、已契約之異界聖靈等。

★寵獸之馴化、飼養、契約皆屬眷屬召喚之領域。

具現化
以魔力憑空生成具有特殊能力之特定物體，並操縱之。

由於此法極為耗費魔力，效率遠不及隨身攜帶武器並為其刻劃魔力迴路，故不受推崇。

-
異界召喚

召喚來自異界的生靈。

異界面：冥界、靈界生物等。
召喚對象：天使、惡魔、精靈、英靈、神等。

異界召喚通常需要強大精神力指定座標，甚至使用獻祭品讓異界生靈回應召喚，但也很有可能因召喚後力量虛弱，而被召喚生靈反支配，或意外被過度強大的異界生靈趁機侵入。

（隨即祭品召喚非常容易引發強大異界生靈侵入，這也造成野生異界生靈入侵蒼究的事件頻繁發生）

跨界召喚：

跨越更多界面來進行召喚，如召喚地球人。

不屬於蒼究體系之人雖然有機率在穿越途中吸取大量能量而變得強大，但更高機率是在途中崩毀，或召喚後變得不完整、殘廢、基因突變成為異形怪物，也有可能同樣極為普通。

-
反召喚

指定座標，將自身召喚至該處，效果等同傳送。

例如：

瞬移亞空間（代替儲物）

傳送門次元穿梭
