# 邪靈體系（遠古物種，全具魔）

普遍被認為邪惡、不祥。善於使用妖力，且實力強大之體系。

血族：

[血族](%E8%A1%80%E6%97%8F%201e3f38cb84a1803e91b2fdfab55f0333.md)

夜叉族：肉體力量極強，善用妖力、兵器。性格嗜血暴虐，喜獵食異族。

巫妖族：擁有強大魔力、妖力，可一定程度「掌控」魔法，有能夠「召喚冥界生靈」的種族固有技。

邪根族：

[邪根族](%E9%82%AA%E6%A0%B9%E6%97%8F%202a4f38cb84a18060ba5efab93a365d07.md)

—–地階—–

暗夜妖族：怨念和負面情緒的結晶體，肉體強大，擁有最強妖力，為暗屬性半靈體。

海魔族：深海中最強種族，肉體強大，妖力、魔力、靈力天賦極強，有眾多分支。

暴食族：形似史萊姆的邪惡污穢集合體，藉由吞噬生物讓自己變強，也讓自己逐漸有了形體、情感和智慧。妖力強大而詭譎。
