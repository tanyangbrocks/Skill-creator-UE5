# 鬼族從屬國

在東域、西域、南域、北域皆存在部份鬼族從屬國，為完全被鬼族勢力統治之人族小國。

鬼族為維持人族肉質，相較於直接圈養人類，更傾向於創造一個能讓人類生存的環境。

通常有地淵或鬼國派遣的強大鬼族坐鎮，使星盟大勢力暫且難以前去解放人民，或地處偏僻又有陣法阻礙，人族高層難以尋找。

事實上，部份鬼族從屬國是人族星盟高層與鬼族地淵高層的私下交易項目，受契約影響，人族會刻意忽略之。

以及「邦厄格」、「繆蠻」等從屬國，西域有許多泰克拿勒未顧及處被鬼族統治，為鬼族從屬國大宗。

東域有……

波光柱人國

以及「白付守」、「饒方室」等從屬國，也有許多鬼族從屬國。

此外還有南域的「薩皮軟」、北域的「阿辣食」等鬼族從屬國，其數量相對較少。
