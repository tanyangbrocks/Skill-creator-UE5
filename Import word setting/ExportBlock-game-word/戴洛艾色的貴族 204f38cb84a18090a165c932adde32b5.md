# 戴洛艾色的貴族

貴族多為由法王冊封的騎士團長、大魔法使等家族，其權力遠高於一般平民，但通常比不過各大聖教會頂層。貴族爵位分為「公爵」、「侯爵」、「伯爵」、「子爵」、「男爵」。

貴族血脈
貴族普遍顏值較高，且後代眾多。源自於他們長期以權力大量捕捉民間高顏值者，並發生性關係。部份直系血脈甚至擁有十多個伴侶。

貴族極度重視強大血脈的延續，通常貴族地位越高，魔法天賦越強。
後代天賦佳者可被選為直系血脈，天賦弱者則為旁系血脈。部份家族會將魔法天賦極低的成員送去鬼族機構，以換取大量資源。

公爵家族

僅能由法王冊封。

只有從上古時代與初代法王親近之家族，或其勢力及掌握之資源龐大到影響國家局勢之家族。部份反而會被法王視為威脅。

全戴洛艾色僅有四大公爵家族。

侯爵家族

僅能由法王冊封。

多為由法王冊封的騎士團長、大魔法使等家族，乃對法王最忠心、最親近的族群。

伯爵家族

可由法王或公、侯爵家族冊封。

居於中間地位的伯爵家容易與上、下級產生利益往來，故戴洛艾色貴族腐敗的情形多來自伯爵家族。

子爵家族

可由伯爵以上家族冊封。

男爵家族

可由伯爵以上家族冊封。

多為實力相對強大的超凡者，或與伯爵家族有利益往來的商人家族，數量眾多。

屬於最底層的貴族，又被稱為準貴族。可即便是男爵，其地位、權力也遠高於平民。
