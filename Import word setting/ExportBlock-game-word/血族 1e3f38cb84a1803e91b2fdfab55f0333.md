# 血族

外觀近似人族，但會依血統濃度而有不同程度的特徵。

擁有強大魔力、妖力，肉身強度、再生力、壽命，智慧與學習能力也極強，擅長暗屬性，魔力親和力強。

種族固有技/特性：

1.幻血

經訓練後可幻化成蝙蝠、狼、蜘蛛等型態。（真血以上擁有）

2.吸血

可捕食其它生靈並吸食血液，視捕食生物提升「血脈力量」。

血族的食物以類人體系為主，也最為營養；其次為精怪體系，再者為萬獸、水行、天行體系。

同喰：吸食血族同類血肉（血族禁忌）

通常會使身體無法承受力量而爆體身亡，若當場失去理智成為「類食屍鬼」。若堅持下來即可提升「血統純度」，但仍會產生間歇性的精神錯亂。

多數血族不知道同喰可提升血統純度，這是因上級血統為穩固階級，並避免下級血統嘗試反抗，故隱瞞此提升手法。

（若能以額外體系力量提升精神強度、肉體強度，再去獵食遠弱於自身的吸血鬼，則可能挺過同喰副作用，穩步提純血脈）

3.賜血

藉由牙齒、指甲將自身血液注入其它生靈，可將對方血脈同化成血族眷屬，並使其無條件聽從自己。（但同化成功與否取決於自身血統，以及對方能力抗性等多重因素。）

4.長眠

躺在棺材中會停止成長，不會有飢餓等生理現象，壽命也停止消耗，並失去意識。（純血以上擁有）

5.唯心連結

能憑意念追蹤、感應、弱化、咒殺自己的目標生靈或事物。（聖血以上擁有）

血脈力量階級

吸血士卒

通常為被賜血的眷屬，會忍不住捕食自己的原種族同伴，長期不捕食則會身體虛弱，若照射到陽光會直接消滅死亡。

暗夜公爵

始可不畏懼陽光，能直接操控附近血族，但不能操縱血脈純度或血脈力量大於等於自身的個體，還能使用「操血術」（操控血液做為武器或炸藥）。但仍無法壓抑捕食的慾望。

大貴族

肉體素質、再生力、壽命與黑魔法親和力大幅提升，還自然擁有念力、精神控制、分身、幻術、重力操控等固有技能，實力極強。始得克服捕食慾望，就算不進食也不會虛弱。

血祖

血脈力量達到「血祖」，血脈純度也會提升至「血祖」。壽命極其悠久，幾乎無法被殺死，擁有超大規模的念力與精神控制能力，能一定程度掌控時間與空間，但並未涉及法則之力。

血脈純度階級

殘血

通常為被賜血的眷屬，膚色蒼白且乾枯，牙齒尖銳，有明顯赤瞳。

殘血所進行的「賜血」只能將對方變成無自我意識可操縱的食屍鬼；所生後代為殘血。

真血

膚色白皙且滑嫩，高顏值，牙齒與赤瞳較不明顯。始得擁有幻化能力。

真血所進行的「賜血」可將對方變為「殘血」；所生後代為真血。

純血

顏值更高，散發獨特魅力。

擁有「純血血印」能力，獲得一個專屬自身的特殊技能。始得擁有長眠能力。

純血所進行的「賜血」可將對方變為「真血」；所生後代為純血。

聖血

顏值再更高，擁有「超凡魅力」，能選擇性地藉由捕食生靈來取得對方記憶。

擁有「聖血血印」能力，再多一個專屬自身的特殊技能，並使其它純血血印的能力無效化。始得擁有唯心連結能力。

聖血所進行的「賜血」可將對方變為「純血」；所生後代為聖血。

[魅力強度](%E9%AD%85%E5%8A%9B%E5%BC%B7%E5%BA%A6%2017bf38cb84a180bcbe2bd734db402f28.md)

血祖

血脈純度達到「血祖」，血脈力量也會提升至「血祖」。擁有極致的美貌和魅力，可自由控制牙齒與瞳色。

擁有「血祖血印」能力，又有一個專屬自身的特殊技能，使純血血印、聖血血印能力無效化。該血印還能與現有所有血族連結，可隔空抽取、賦予、轉移所有血祖以下血族的一切能力，包含生命。（除非有和自己同為血祖的個體干涉）

血祖所進行的「賜血」可將對方變為「聖血」；所生後代為血祖，但血祖的生育能力極低，近乎無法生育。
