# 大五行天道宗

由金虹門、紅蓮教、玄靈山、星河宮、化塵派等五大勢力組成巨型聯合宗。

其中每勢力單獨拉出來都能與「六宗」的其餘五宗相當，五大勢力聯合後，大五行天道宗成為修真界最大勢力。

此外，大五行天道宗也六宗十二派宗唯一能和六大貴族、天輝山、皇室叫板的勢力，甚至有許多人認為大五行天道宗應被歸類為主流勢力。

大五行天道宗內部

大五行天道宗勢力雖龐大，但其五大勢力實則並不團結，經歷過不少次分裂危機。

五大勢力擅用各自的五行之力對敵，而當他們聯合禦敵，能施展許多五行大陣、五行聯合術法神通，這也是他們選擇結盟的重要原因。

大五行天道宗彼此經常有弟子共同試煉、弟子切磋大比、論道交流會、共同任務，甚至短期交換弟子等活動。

金虹門

擅金屬性靈力，門內以劍修為主。重視劍意之修煉，重視悟性、論道。不擅劍道者會成為體修，刻苦磨練體魄。

門內弟子部份性格堅毅，有些剛毅木訥；也有部份喜歡耍帥，刻意斥巨資修煉萬劍歸宗等戰鬥模式。

紅蓮教

擅火屬性靈力，亦擅於煉器、祭煉法寶，以及各式攻伐用術法神通。

門內弟子脾氣火爆，熱愛戰鬥，經常派門下弟子去其它勢力踢館。

玄靈山

擅木屬性靈力，亦精通煉丹、煉藥，以及靈植培育和御獸、御蟲。

門內弟子性格內斂，專注修煉，較少與人起衝突。

星河宮

擅水屬性靈力，亦精通識海的修煉及運用。部份擅於天命道的卜卦、推衍之術，也有些弟子擅於音妙道，做為樂師、舞師。

門內弟子無論男女，姿色都驚為天人，是其它勢力的主要追求對象。

化塵派

擅土屬性靈力，亦精通陣法、符籙、傀儡之道，以及靈脈的管理與開採。

門內弟子較博學多聞，對靈力和修煉的理解普遍高於其它勢力。但也有許多弟子性格怪異。
