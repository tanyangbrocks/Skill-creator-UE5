# 人族對獸的實力分級與分類

人族對獸的實力分級（只適用無靈智存在）

無危害：普通孩童也能輕鬆應付。

極低危害：普通成年人可應付。

低危害：五名以下成年男性可應付。

弱級：五~八名普通成年男性即可應付。

初級：個體可獨自摧毀小村落。

中級：個體可輕易屠殺諸多村落。

高級：個體襲擊城鎮將造成巨大傷亡。

領主級：個體有屠城之能耐。

君王級：個體可能造成國家覆滅。

禍難級：個體需各大勢力聯合討伐。

災殃級：唯有請動對應強者，否則對全體人族造成重大危害。

劫災級：若不及時派出對應強者，可能導致人族覆滅之大劫。

滅世級：有毀滅世界之能耐，「滅世級獸」當前只適用於「蒼究沉浮」。

📌通常強大存在都會擁有靈智，因而使用「星盟戰力標準」。因此，實力達災害級卻仍無靈智者極少。

白獸
就是一般動物，體內沒有引發變異的特殊能量。地球上約870多萬種生物都可套用至此處的「白獸」。

這裡也存在許多地球沒有的「一般動物」，因此總白獸數量約1200萬種。

大多數的生物都無法開啟靈智，或是獲得魔力、靈力、妖力而變異，因此世間的「白獸」還是占「獸」的絕大多數。
