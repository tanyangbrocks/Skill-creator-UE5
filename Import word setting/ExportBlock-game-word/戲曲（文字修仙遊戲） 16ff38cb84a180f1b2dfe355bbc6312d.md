# 戲曲（文字修仙遊戲）

《小妖報恩》的故事！

傳說有名農夫曾於於路上發現一隻受傷的小犬妖，將其帶回房內療傷後放其離開。
十年後，農夫與妻兒在外面旅行時遭遇妖獸襲擊。此時一名強大妖獸救下他們，原來正是當初那隻被救助的小犬妖。

即便人妖殊途，亦有真情道義。這感人肺腑的故事令眾人熱淚盈眶、無以言表！

《窮兒登仙》的故事！

傳說有名窮孩子為養活重病的母親和兩名年幼的妹妹，每日去市集擺攤、去工地搬磚，不辭辛勞地拼命賺錢。而在夜深人靜之時，終日忙碌的他竟前往深山練劍，不放棄任何精進自身的機會。

其堅毅之心似乎感動上蒼，某天他忽然引氣入體，開啟修煉生涯。隨後他一路破境，最終成就道極，飛升成仙了！
