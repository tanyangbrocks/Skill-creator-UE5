# 森精族

既是類人體系，又具有精怪體系的特質，與精靈契合度極高。

膚色普遍白皙，顏值極高，多居住於森林中的祕境，喜愛種植花草樹木，善用弓箭。

敵族眾多的森精

森精由於外貌極美，經常被人族與哥布林族綁架，故極為痛恨這兩族。（由於是不完全的類人種族，故在人類社會地位也普遍較低，基本被視作奴隸或異族）

此外，身為類人體系、精怪體系的森精也同時是鬼族、妖怪支、妖厄支的食物。

不僅如此，森精部落內的生命之泉、世界樹等珍寶也經常受異族覬覦，森精可說是眾多種族都想劫掠的可憐對象。

森精的居住環境與文化

為保護自身，森精總是隱居在族人創造的祕境中，平均部落勢力堪比人族小國。

森精雖肉身較為孱弱，但普遍具有極高的魔法天賦，五感與動態視覺極強，能以卓越弓箭術護衛家園。
