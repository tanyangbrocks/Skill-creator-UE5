# 次元物種(裡世界物種)

根據史書記載，次元物種共有四族，分別為：

—–玄階—–

影族：具魔，為附體術創始族，附體術天賦高。居住在「影之世界」中。

[影族](%E5%BD%B1%E6%97%8F%20188f38cb84a1801a80bac94ab314819b.md)

[影之世界](%E5%BD%B1%E4%B9%8B%E4%B8%96%E7%95%8C%207b4f72268d2e4d5d837b4b9d92628db7.md)

幻想蟲族：具魔，「擬態」模式可大幅增強實力。能藉由吞食其它種族而獲得其能力，本身肉體能力極強、繁殖力高。居住在「次元蟲洞」中。

[次元蟲洞](%E6%AC%A1%E5%85%83%E8%9F%B2%E6%B4%9E%205fc78180e8444d08bd4d82d6d3f9a1fa.md)

機械族：

[機械族](%E6%A9%9F%E6%A2%B0%E6%97%8F%20bfca2e5a5c1041fda725ee56149b7e13.md)

元素靈族：

具魔，其靈智相對較低，體內蘊含大量元素能量，經常被魔法使召喚獻祭以發動更強大的元素魔法。居住在「元素位面」中。

[元素靈族](%E5%85%83%E7%B4%A0%E9%9D%88%E6%97%8F%20187f38cb84a180ceb89ec660457dd688.md)

—–地階—–
幻形族：具魔，能在一定程度上複製對手的能力和形象。居住在「鏡像世界」中。

[鏡像世界](%E9%8F%A1%E5%83%8F%E4%B8%96%E7%95%8C%202df4082b6d7e4960a3e4a3134d316d6a.md)

深淵族：具魔，擁有能污染、腐敗一切事物之力，以其它位面生靈為食。居住在「深淵」中。

[深淵](%E6%B7%B1%E6%B7%B5%20183f38cb84a180c68812c394cd068cdf.md)
