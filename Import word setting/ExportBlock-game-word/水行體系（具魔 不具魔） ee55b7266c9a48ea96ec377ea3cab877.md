# 水行體系（具魔/不具魔）

根據史書記載，肢節體系共有兩千兩百三十一族，以下為常見種族：

—–黃階—–

海魚族：誕生靈智的強大魚種，有眾多支派。

深海魚族：誕生靈智的強大深海魚種，肉體力量普遍強大，較為古老神秘，有眾多支派。

蟹族：誕生靈智的蟹，外殼堅硬、肉體強悍，擅使用兵器。

蝦族：誕生靈智的蝦，速度極快，擅操縱海底水流。

軟爪族：包含海螺種、章魚種、魷魚種等多種支派。通常具強大妖力，每支派也各有特殊的種族固有技。

魔棘族：包含海星種、海膽種、海參種等多種支派。通常具強大魔力，每支派也各有特殊的種族固有技。

刺胞族：包含水母種、海葵種、珊瑚種等多種支派。通常具強大妖力，每支派也各有特殊的種族固有技。

—–玄階—–

巨鯨族：主要居住於深海，性情平和，無欲無求，體內蘊藏無盡能量。多數巨鯨族為魚龍族之高階眷屬。

九爪族：主要居住於深海，體型龐大，四散於海中各處，喜歡自己主宰一片小海域。全身能高速再生，動作靈敏，部份使用魔力，部份使用妖力。

領地意識強，可能主動襲擊路過船隻，或向異族勒索寶物錢財，又被稱作「深海惡爪」。

蜃族：主要居住於深海，能製造幻境迷惑獵物，將其困死後捕食。擅用魔力。

智力偏高，鮮少露面，且牠製造的幻境能投射至極遠處，故難以尋獲其本體。部份蜃族效忠魚龍族，會為其製造大面積聲光特效，以提高魚龍族的排面。

—–地階—–

魚龍族：擁有部份龍的特徵的深海種族，其地位僅次於巴哈姆特族。但由於巴哈姆特極少露面，故通常魚龍族對大海種族的影響力最大。

巴哈姆特族
生活在海中的巨型魚龍種族，為大海主宰者。
