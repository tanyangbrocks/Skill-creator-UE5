# 仙御（強力御物）

同源複合能量：

[仙道（靈玉）](%E4%BB%99%E9%81%93%EF%BC%88%E9%9D%88%E7%8E%89%EF%BC%89%20ce0ca431b91d4df88398686bc515dc85.md)

[法道（識海）](%E6%B3%95%E9%81%93%EF%BC%88%E8%AD%98%E6%B5%B7%EF%BC%89%202f1d95bdc702424699350aa38db85158.md)

---

使用「仙御」驅動各種仙器、符籙所產生之效果大於純粹的「精神力」。

此外，使用「仙御」時也能將使用者自身靈力特性帶入當前驅使之物體。

附靈師

所謂「附靈師」即是運用仙御的技巧，使物體擁有特殊靈力，或是對法器進行改造。
