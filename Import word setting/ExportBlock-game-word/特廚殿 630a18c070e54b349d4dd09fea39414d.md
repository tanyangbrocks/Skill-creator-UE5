# 特廚殿

由特別喜愛料理之生靈組成，其成員超越了種族間的隔閡，含人族、妖族、龍族、海魚族等。

特廚殿廚師考核

特廚殿每年舉辦針對廚師之考核，針對各方面評價綜合廚藝。

特廚殿美食評鑑

特廚殿食獵部

特廚殿專門獵取世界各地食材，包含肉食、植物、調味料等，據說越美味的食物越難以取得。

-

特廚殿暗狩部

特廚殿專門狩獵「種族生靈」做為食材的部門，只會接取以料理為目的之高價獵殺委託。通常獵殺執行者會和委託者為同一種族。

由於特廚殿明面上禁止獵殺種族生靈，故「暗狩部」是不被特廚殿官方承認的組織。但實際上，暗狩部也是特廚殿殿主本人所創，他認為追求美味是至高無上的慾望，超越一切倫理道德。

PS：「妖怪」經常委託暗狩部獵殺人族，此乃暗狩部最大生意，因此暗狩部也有許多妖怪。
