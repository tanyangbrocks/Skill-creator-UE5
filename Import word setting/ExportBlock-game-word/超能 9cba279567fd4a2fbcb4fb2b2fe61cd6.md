# 超能

基因之亂數，非正規力量；人為激發的超能力只能算科技或源能的應用。

「天生的」超能力來源：

超能常出現於「力量薄弱」、「周遭環境險惡」、「命運強大」之個體，常被視為上天的恩賜。

（當然，這並非絕對。這實際上是）

不過超能力依循上述定律，常出現在「純物質界」、泰克拿勒等以凡人為主，反物質能量較少的社會中。超能力者一旦發現就被視為異類，這股力量經常反而為宿主帶來災禍。

超能力本身能量來源可能是「概念」的弱化版，由宿主腦內本身之意志演變而生，這過程稱為「超念實化」。

超能力體系：

超能力之能量無法被歸納為一種體系，只能按其效果做簡單分級、分類。

紅色警戒：如穿越時空、能力複製（掠奪）、掌控生死、改寫命運，涉及法則操作或無法預估後果之能力。

橙色警戒：如精神控制、重力操作、物質崩毀、憑空造物、召喚軍團，可能對社會造成重大危害之能力。

黃色警戒：如瞬移、不死、空間傳送、毒氣擴散、附身，具有難以壓制之強大力量之能力。

藍色警戒：如噴火、冰凍、雷電、風暴、怪力，具有明顯攻擊性與威脅性之能力。

綠色警戒：如心電感應、念力、隱身、加速、飛行，對外界影響較小之能力。

灰色警戒：如智力提升、肉體強化、過目不忘、透視等，不直接影響外界之能力。

_

人為激發之超能力：

研究員雖沒有概念之力，但能利用源能刺激受試者大腦，引發其「超念實化」。

此做法又成為「覺醒手術」，能讓受試者有40%機率覺醒超能力、10%機率留下腦部後遺症、1%機率使大腦直接崩潰造成腦死亡。

由於其具死亡風險，故該手術受到政府嚴格管制，無法普及。

📌常人的大腦無法負荷多種超能力，但若透過「開啟腦竅」、「培養精神力」等方式提升大腦效能，或許就可容納多種超能力。

定向激發之超能—編號異能

部份特殊藥物能使特殊族群覺醒固定異能，這類藥物受到政府嚴密控管，多用於科研或軍事用途。

指定覺醒的超能力受科學方法不斷調整、優化，通常會具有較強效果與較低副作用，又被稱為「編號異能」。
