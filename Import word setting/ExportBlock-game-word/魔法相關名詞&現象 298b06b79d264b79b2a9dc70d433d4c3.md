# 魔法相關名詞&現象

魔素：魔素即是物理法則的對應，與物理法則共生共滅，是一種必然的存在。因此，魔素也是一種「反物理能量」。

魔素可以粒子、固態或能量等各異型態存在。

魔力：魔素的能量型態，也是魔素被使用、操縱時的模樣，有各種千奇百怪的效果。

瑪納：粒子型態的魔素，通常逸散在空氣或儲存於生靈體內。逸散在空氣中的魔素，會形成相對穩定的「瑪納」。空氣中的瑪納可輕易被能掌控魔力者使用。

生靈體內的瑪納總量可決定其戰鬥持久性，魔力量則決定其單次輸出的威力。

魔素晶體：固態的魔素，可由魔石或蘊含魔素的事物開採、提煉出，經由特殊儀器可再度轉化為魔力或瑪納。

魔力災害

乃瑪納絮亂導致魔力異常的現象，又細分為以下兩者。

－

瑪納潮湧：環境瑪納異常充沛，可能導致魔獸實力大幅誕生、強化，甚至引發魔獸潮。

魔力風暴：環境魔力混亂且不可凝聚，導致無法發動魔法。

魔威：釋放魔力的威壓，強者釋放魔力時可利用魔威導致弱者恐懼、昏厥，甚至當場暴斃。

演算：操作魔力時進行的運算與推演。操縱魔力的過程需集中精神，凝聚精神力以進行演算，無論是否編織術式。

PS：許多魔法使也會修行修煉六道中的「法道」，以大幅強化精神力，但若沒有靈力輔佐，通常修煉速度極慢。

魔力感知：凝聚精神力感應周遭的魔力波動，用於偵測環境、敵人、預判對手術式、感知危險、蒐集魔力相關情報等。

魔力屏蔽：隱藏自身魔力與氣息，讓自己不被他人感知，用於埋伏、隱身、躲藏。

瑪納壓制：魔力使用者壓制體內瑪納，使展現出的瑪納量比體內實際瑪納量小，可用於隱藏實力，使對手誤判情報。

術式反噬：倘若在編織術式的演算過程中發生意外，如因分心、編織嚴重錯誤、當前魔力嚴重不足，可能造成術式魔力暴走，引發術式反噬。

越高階之魔法越容易反噬，施術者也需承受越多魔力衝擊。據統計，發生術式反噬者有86%機率當場死亡、12%機率大腦永久受損，術式反噬乃新手魔法使最常見死因。

術式改造：精通術式編織原理者能對既有術式進行改造，如改變威力、規模、特性，開發出延伸用途。但倘若編織術式時出現錯誤，也有可能導致術式魔力暴走，引發術式反噬。

靈魔互斥定律：由於魔力是極端的「反法則能量」，它與通往法則、試圖感悟法則的靈力相互排斥。故靈力、魔力通常無法結合，也不太可能同時存在於單一個體。

魔力功率：單位魔力量所能釋放出的「奎因」（純能量體系破壞單位）。

魔力適應性：物體或生命體適應魔力的程度。魔力適應性越高，施放魔法的過程越順暢，功率也越大，但遭受魔力侵襲時受到的效果也相對較顯著。

魔力適應性指數為0%~100%，

0%即為完全阻隔魔力流通（但可以被魔力產生的能量破壞）

100%即為完全零阻力

一般而言，空氣會有約88%的魔力適應性，具魔的人體約74%左右，但也有少數人能到90%以上。

魔力抗性：物體或生命體抵抗魔力作用的程度。雖無法抵禦「奎因」的直接破壞，但在抵抗控制、魔咒和屬性負面狀態效果上有奇效。

通常個體的魔力適應性越高，魔力抗性就越低，但也可透過煉能武裝、魔抗裝備等方式來單獨提升魔力抗性。

魔力抗性指數為：1-100（魔力適應性）

「魔力適應性/魔力抗性」的高或低都有各自優點，故兩者位於中間值反而最弱。

魔力疾病

部分不祥的魔力可能變質為詛咒，甚至演變為疾病在民間傳染開來。

蒼究魔力網路

散布在重疊裡空間中的強大魔力網路，內含大量純潔瑪納，從中汲取力量將獲得極為強大的魔力。

未論魔法

能夠藉由生靈想像或才能而激發出的魔法，但本人不知曉其運作原理，也尚未有人成功解析。
