# 夢幻族

可依靠想像力改寫現實與法則，是當今蒼究世界種族血脈力量最強大的種族。

種族能力：

其所有種族能力都可直接影響法則，但法則控制權在神話種族之上的「尋能」、「神力」、「願」、「概念」、「超格技能」、「星能與其複合能」皆可干涉其種族能力效果。

自我唯心

自身當前狀態全憑自身想像力，相信自己不會受傷就不會受傷、相信自身不會死亡即永恆不變。

自身外貌、聲音、氣息、能力也可隨意改變，故沒有固定型態。

自我唯心的

若要殺死幻想族，只能使其相信自己會被殺死，或用「尋能」、「神力」、「願」、「概念」、「超格技能」、「星能與其複合能」來干涉其力量。
