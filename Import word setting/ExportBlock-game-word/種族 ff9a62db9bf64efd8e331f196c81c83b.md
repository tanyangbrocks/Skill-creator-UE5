# 種族

種族進化：

種族進化後，不僅自身血脈基因完全改變，甚至「魂」與生命層次也可能提升。

許多「種族」及「獸」可以進行進化，各族進化條件不一，難度、進化過程也各不相同。

（萬獸體系全族種皆可進化）

「種族進化」本身之原理是「業力價值」的積累，導致生靈魂的蛻變。

種族位階&體系：

種族依「初始生命層次（魂之位階）」分階級，分為……

黃階：普通存在，世上99.6%以上種族皆為黃階血脈。

玄階：整體力量遠高於黃階血脈，通常戰力極強、壽命較長。

地階：初始生命層次接近神之領域，得天獨厚的高貴血脈。

天階：初始生命層次已達神之領域，只要稍加修煉便可成神。

★生命層次，又稱「命階」，代表生靈當前魂的昇華程度。

★命階達到神之領域者可成神，而當魂的「價值」達到神之領域時，命階也會提升至神之領域，使該生靈成神。

獸的類別

無靈智之生靈統稱為「獸」，但一般不包含沒有能量體系的「非動物界生靈」（如普通植物、蕈類等）。
