# 尋能(真實)

賦予抽象現實之力，創世之力。

真尋力（擁有即為真尋者）

原為真實之神(尋)及其所賜予真尋念的對象所擁有，但由於真實之神死亡，真旭之核分裂四散掉落到人間，引發「覺醒潮」，使得真尋者陸續誕生。

每個人一開始所擁有的真實都相同，而真尋力則是將虛假帶到真實的力量，同樣的真實若藉由越強大的真尋力轉換就能夠擁有越強大的力量，每個真尋者天生擁有的真尋力不相同

反尋力

反轉的尋力，能夠將真實轉為腦中的虛假，此過程雖然不會有存在被抹除的問題，但行使極為困難，且容易讓大腦接受過多資訊而崩潰，若轉換的對象蘊含真尋力，則可再藉由真尋力將其帶到真實（像是將真尋者帶到大腦後，可使用其能力）

-
尋念

由真實所附加的能力（大部份能力在永久加諸的狀態下使用只會消耗體力），發動時會產生少許尋塵。

PS：「尋塵」為使用真尋力改寫現實後能留下的唯一線索。

尋念之使用

成長性不同尋念有不同成長性，成長性越高的尋念越容易經由訓練而變得更強大，也越容易進化。（雖然獲得尋念時就知道如何使用，但是使用的熟練度還是有很大的差異，就像人一出生就會操控手臂，但無論是各種體育競技、還是高精密度的技能，還是要經由學習及訓練來增加。）

尋念之等級：

利用尋能（真實×尋念）之多寡來判斷，共可分成七級：

合級 超合級 王級 帝級 聖級 滅世級 創世級

尋念三圍

尋能、尋念實用性、尋念成長性

第四圍：尋念戰力

尋念進化

可將真實投注在自身尋念中，大量投入就能進行進化，但進化過程極為不穩定，若進化失敗則有可能付出代價，甚至死亡。

破尋

經過鍛鍊使部份靈魂之力融入尋念，使尋念的力量達到更上一層境界
破尋由於是有靈魂之力摻入，故會隨著使用者的心理狀態發展

尋念絕對性

尋念是能直接改寫現實的能力，其效果無視所有物理、哲學定律，但若有兩種改寫內容相互牴觸的尋念同時作用（ex:絕對貫穿vs絕對防禦），則尋能大者能讓尋能小者無效化

常見尋念之種類合：心電感應、飛行、測謊、隱形、高速、超腦力、透視、念力、肉體強化、再生

也可以用把「沒有受傷的虛假」帶到身上成為真實，只是這樣

超尋者

在2028年的「覺醒潮」之前就存在的超尋者，身份不明，也不知其存在了多久

尋咒

由真實所被加諸的詛咒
