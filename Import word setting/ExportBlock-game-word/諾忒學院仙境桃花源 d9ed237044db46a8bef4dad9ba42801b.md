# 諾忒學院仙境桃花源

仙境桃花源的煙花不僅是諾忒城公認的第一美景，甚至登上了若夢埃頌雷特聖福林大天帝國風景名勝榜第三。

在國內，仙境桃花源還比諾忒學院有名！

（但費用昂貴）

---

煙花文字描述：

仙境絕非虛名，桃花源中仙山瓊閣、柳煙花霧，萬千種如詩如畫的美景讓眾學員每隔十米駐足沉醉，差點忘了煙花！

隨後你們各自去夜市攤位買小點心，至星夜湖畔集合坐下，等待晚間的煙花。

不久，煙花開始！

-

煙花一飛沖天來到百米高空，瞬間迸發出上萬種不同顏色的璀璨耀芒，彼此交織相融，湖面的輝映上綻放出無限旖旎神采。

煙花時而幻化出龍飛鳳舞、帝皇上朝、盤古開天、化神期大能鬥法等震撼場面。

煙花時而勾勒出如小妖報恩、窮兒登仙、孝感動天等溫馨感人的故事。

煙花時而凝聚成松鼠、白兔等小動物和你們追逐嬉戲。

-

煙花整整持續了一個時辰，待煙花落幕，你們每個人都還如癡如醉，沈迷於方才的美景中，稚嫩的臉蛋上洋溢著幸褔。

隨後大夥兒開始興奮地討論剛才的景象，樊華覺得騰雲駕霧的青龍超帥氣；顏若雪最喜歡小妖報恩的溫馨故事；莫青歸則被道極期鬥法的場面震懾，對更高的境界感到心神嚮往。

你與大家一起興奮地討論著，稚嫩的歡笑聲傳遍了整座仙境桃花源...…

-

那晚，你們一大群孩子坐在星夜湖畔，小腳丫子踢著清澈的湖水，手裡握著甜美的糖葫蘆。你們將煩惱拋至腦後，共同歡笑嬉鬧，渡過了這輩子最美好的夜晚......
