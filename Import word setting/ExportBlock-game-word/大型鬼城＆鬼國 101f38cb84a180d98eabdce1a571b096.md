# 大型鬼城＆鬼國

亂池鬼國

鬼王亂池上陰所建立的鬼國，位於西域與南域之間，為赤鬼派系。

國民淫欲極為旺盛，市面流通大量鬼族性侵人類的影音檔案，高階鬼族亦會在家中圈養人類。街道和設施上都繪有大量人族圖樣做為美的裝飾，人類的慘叫與呻吟不絕於耳。

皇室機構著重訓練具備隱匿、潛行能力的智鬼，並與鬼影會合作，得以較低價格大量收購科技體系人族做為性奴。

鮮牙鬼國

魔鬼奇茲卡厄所建立的鬼國，位於南域靠近東域處，以紫鬼派系為主，亦有赤鬼、青鬼，乃最強大的鬼國。

極度喜愛吃人，對人肉重度上癮使得他們一日三餐皆必須吃人族血肉。出於對人族血肉近乎瘋狂的癡迷，他們極少交易手中的人類，居民間的互動以妖法切磋、交流捕食與享用人族的心得為主。每日皆會前往人族地區襲擊路人或捕捉幼童，與東域、南域勢力產生極大衝突。

由於奇茲卡厄採取完全放縱、鼓勵獵殺之政策，吸引大量高階鬼族遷入並繁殖。國內設有大量人族養殖場、妖力訓練所，並模仿人類政體建立鬼族社會，鮮牙鬼國日漸強大。

參喰鬼國

魔鬼參喰樂道所建立的鬼國，位於西域的低開發邊際產業國附近，以青鬼派系為主。

由於低開發邊際產業國容易侵入、獵食人民。許多高階青鬼喜歡與捕捉而來的孩童、學生玩奇怪的遊戲，盡情折磨、玩弄後再大快朵頤；或是潛入住家、校園或宿舍、遊樂場、商場，以能力瘋狂追殺附近的學童。其活動創造大量恐怖而血腥的都市傳說。

曾將繁華都市內某間國小整座傳送到鬼國內，進行集體圈養、捕獵。

黑樓鬼城

位於東域天北百州的小型鬼城，混雜不同體系的低階鬼族，喜愛獵食附近村民。
