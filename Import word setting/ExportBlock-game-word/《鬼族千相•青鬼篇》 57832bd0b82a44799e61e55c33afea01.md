# 《鬼族千相•青鬼篇》

📌低階鬼族個體除了本身型態所固有之能力外，還會擁有自身獨特S鬼域。

[鬼域](%E9%AC%BC%E5%9F%9F%20a5f2afa1b6f8492b84d69e9816b7ec77.md)

📌此處所述為尚未經分裂生殖之原始型態，分裂生殖後能力會自然削弱。

以下紀錄常見型態，這些型態能力較為強勢，故可大量繁殖：

寶箱鬼（獸鬼）

本體像是有兩顆頭，臉上只長各一隻眼睛和耳朵，嘴長在肚子的哥布林。能夠幻化成附近人類當前最想得到且合理的物品。

像在剛弄丟佩劍的劍士身前幻化為其佩劍；在剛輸光財產的賭徒身側幻化為路邊錢包；在森林找食物的孩童身旁幻化成水果。而在獵物觸碰到其身軀時，寶箱鬼會直接化成鮮紅的人類巨口，在對方反應前爆發出極強吸力將其吸入，再瞬間咀嚼、消化。

至於其最知名的手段，就是幻化成超界者隊伍攻略各場景的寶箱，並於對方中招後一口氣獵食整支強大隊伍，直接進化成微智鬼或智鬼。

除非有手段能看破其偽裝，否則無論何時都要留意寶箱鬼。

隱匿型（獸鬼）

在人煙稀少的地區，會從人類周遭的黑暗環境或恐懼情緒下誕生於其附近，模樣為該人內心深處最恐懼的事物，隨後偷偷來到其身旁驚嚇、獵食之。若能順利獵食創造牠的目標，將會獲得隱身能力並穩定其存在；若半小時內無法獵食對方，則會自動消亡。

成功獵食創造者的隱匿鬼會偵測四周心靈最纖細、脆弱的人類，隱身接近他，並在其面前化為他最恐懼的事物獵食之，如此反覆。

雖能化為他人最恐懼的事物，但實力上限仍在獸鬼層次，超界者只要不被其表象蒙蔽或陷入恐懼，大多能輕鬆擊殺。但普通人，尤其是野外的冒險者、迷路孩童都極容易觸發隱匿型誕生條件，因而被獵食。由於隱匿型很少曝光在大眾面前，比較像是都市傳說，許多知曉鬼族資訊者也不太瞭解其誕生機制。

人面蟲型（獸鬼）

頭部為巨大嬰兒臉，軀體為粗壯青綠色蟲身，以蠕動、彈跳方式移動。鑽地能力強，平時於地底高速移動，鑽地過程幾乎不會產生聲音、震動，能隱蔽自身氣息，還能極為敏銳地偵測障礙物、能量波動和地表人族的氣息與活動狀態。

鎖定目標後會從人族腳下瞬間鑽出，咬住目標後快速拖至地底，同時雙眼噴射出特製泥狀物將鑽開的洞補平，泥狀物會自動融入環境掩蓋痕跡。獵物被拖至地底後，人面蟲型會分泌大量續命唾液，側邊冒出大量嬰兒手臂緊抓住獵物，一口一口地蠶食、折磨獵物，持續數日，獵食完成後，頭部人面會轉化為前一個獵物的模樣，並重現其死前痛苦扭曲的臉龐。

人面蟲型戰鬥力低，但其不會在地面和人正面戰鬥，在地底也只鎖定不具戰鬥力的普通人，特別是鮮嫩又柔弱的女人及孩童，甚至專挑他們落單時、熟睡時進行突襲。牠也會主動避開強者和危險區域，在地底進行繁殖，就算是超界者被其拖入地底，也容易受突來的黑暗密閉環境影響而被獵食。專業獵鬼士會在大量女人孩童失蹤地區偽裝自身，以自身為誘餌被拖入地底，再將其反殺。

人面蟲型很少被發現，但其實數量極多，在野外、村莊或落後城鎮有無數女人孩童成為牠的美食。即便知曉其存在，還幾乎沒有方法預防，隨時都可能被吃，因此牠是許多女人孩童的噩夢。

暗影蟲型（半獸鬼）

人面蟲型約一半會進化為暗影蟲型，此時身為青鬼的暗影蟲型已具備約5歲孩童的靈智，在原始人面蟲型之基礎上大幅增強軀體縮放能力與移動速度、隱蔽氣息能力，並能短暫虛化軀體。此外，被牠咬住的生靈若不具能量體系，能被牠縮小並含入口中。

暗影蟲型多數會前往科技更發達的都市地區，捕食更鮮嫩的先進都市人，獲取更多養分。他們能夠縮小潛入管線、牆壁、電梯、交通工具，鎖定都市中生活最優渥，最鮮嫩、幸福的女人及孩童，包括高樓層住宅區，在其睡覺、如廁、洗澡、唸書等落單時機瞬間虛化竄出，將獵物含入口中後虛化回地底，再用續命唾液盡情折磨獵物，細細品嚐，隨後藉交通工具迅速轉換獵場，只要謹慎行事，幾乎不可能被強者抓到。雖仍不具戰鬥力，但卻能享受每日大塊朵頤，長久下來還能透過大量獵食再次進階。

暗影蟲型已無法由常規手段處理，只能由仙道大能的靈識、高階級魔法等遠超半獸鬼層級的存在來強制搜索並捕捉。然而人族強者幾乎不會出現於科技體系，更不會發現任何異常。

所以都市中許多根本不知鬼族存在，在和平世界養尊處優的女人、孩童只能在無盡驚恐和地獄般痛苦中成為暗影蟲型的鮮美饗宴。

寄生型（獸鬼）（不是中階能力的「寄生」！）

寄生型軀體形似大蟑螂，擁有翅膀，本身實力甚至不如人類，且只要持續四十八小時沒有成功寄生於人體，便會因機能衰退而死去。欲寄生時必須飛向人體並鑽入其皮膚，對象的體質和精神越弱小，寄生越容易成功，故目標多為孩童、病患、心情低落或身心疲憊者，且通常不具備能量體系和戰鬥力。寄生成功的瞬間會直接摧毀宿主人格，並奪取其靈智和記憶，始得使用鬼族共通初階能力來捕食獵物，以及專屬S鬼域。

雖實力不強且寄生條件較嚴苛，但寄生成功後能直接擁有完整靈智，能夠隱藏自我，相對不容易被人察覺。

歡愉精靈（獸鬼）

本體有一顆非常巨大，眼球凸出的青色頭顱，軀體比例較小。扭曲面龐上的血盆大口有多層牙齒，口部空間巨大，擁有恢復效果特強的續命唾液。

專門獵食孩童，製造出只有孩童能看見的小精靈投影，向四處擴散飄蕩。歡愉精靈看見孩童時會逗他們笑，並引誘他們靠近，當孩童因好奇而觸摸投影時，孩童將無視距離被直接傳送至本體身旁。

本體喜歡將孩童放入嘴中持續咀嚼，續命唾液的效果讓孩童不會死亡；孩童將持續被嚼成肉泥再恢復過來，甚至會被吐出來再吃一次，直到歡愉精靈嚼膩為止。

本體居住地極為隱密，故人族只能宣導不要觸摸歡愉精靈投影，但每隻歡愉精靈投影的聲音、樣貌會受本體和目標孩童的願望影響而變化。

電子蟲型（半獸鬼～微智鬼）

本體是一團長有血盆巨口的漂浮肉球，除了初階能力和S鬼域外，擁有能簡單移動物品的念力。

專門獵食科技先進的都市地區之人，也是極少數能在泰克諾爾獵食的低階鬼。可進入任何電子用品之電路，並以能連網的任何產品為媒介進入網路世界，以特殊檔案形式存在。電子蟲進入電子設備時，也能發動念力移動外界的物品。

電子蟲型可從網路中獲得大量資訊，除社交軟體外，還包括每個人的雲端相簿、私人筆記等，並能鎖定資料的IP位址。利用此法能極高效地找到自己滿意的目標，且能幾乎無視距離條件，選擇肉質最鮮甜軟嫩，又對社會影響相對不大的獵物。

鎖定目標後，傳送至對象的手機、平板、筆電等。在對方開心滑手機時突然跳出詭異訊息，營造驚悚氣氛，用念力操縱對方房間內的物品，待對方恐懼達到極致時，驟然從螢幕竄出驚嚇之，將其瞬間捕捉後拖入網路，再從自身基地設備連同獵物一起傳送出來。接著就能盡情享受、折磨自己喜歡的獵物。

電子蟲型特別愛獵食孩童、年輕學生，以及生活幸福、才華洋溢者，有時會選擇奴役、囚禁他們來取樂或販賣。狩獵效果還更勝於鬼影會。

電子蟲型的狩獵幾乎不會被察覺，也很難預防。就算在短時間獵食大量獵物，由於失蹤人口分散，故也不會引起太大關注。此外，別說是科技體系，就連修煉及支配體系都鮮少有人知道電子蟲型的存在，就算知道也不會想插手。因此電子蟲型在科技體系可說是能吃到飽，想獵食誰就獵食誰。
