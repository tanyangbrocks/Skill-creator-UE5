# 宗教信仰

蒼究所信仰的神皆是真實存在的，部份強大神明藉由散播信仰，賦予神聖力給自身代行者來獲取「價值」。

不同於水、火等可以用能量體系掌控的元素，蒼究之神更信仰虛無縹緲的概念、渴望來做為神祉。
