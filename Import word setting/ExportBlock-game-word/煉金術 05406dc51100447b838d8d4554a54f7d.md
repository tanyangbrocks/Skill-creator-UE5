# 煉金術

探索理之真諦與應用，乃研究能量轉換之科學。

亦為「反物理能量學派」的主要學門，所有魔法使都理應具備煉金術的概念。

初級煉金術只需使用魔法、魔導學、法陣學，但中、高級煉金術需使用魔力與源能結合之「虛能」。

煉能武裝

需使用「虛能」，融合斑雜能力成特殊型態，形成煉能武裝烙下刻印後可隨時開啟，分為：

攻擊裝

通常散發紅光，特化攻擊並弱化防禦。

防禦裝

通常散發藍光光，特化防禦並弱化攻擊。

極限裝（似卍解）

短時間轉為另一型態，極大幅度增強力量，但通常無法維持太久。

突破裝（使用後有重大代價）

透支身體，短時間突破至更高境界。通常為一次性，且使用後需付出重大代價。

煉成矩陣轉換

煉成矩陣由一個個小矩陣組成，需使用「虛能」，煉成矩陣分一階二階三階......
有轉移矩陣（調整矩陣使其到穩定狀態）、旋轉矩陣、鏡像矩陣（火鏡像變水）等，功能千奇百怪。

但煉成矩陣必然遵守以下三大定律：

一、萬能轉換定律

理論上可進行一切不涉及「魂」與「法則」的轉換。因萬物皆可分解成「虛能因子」，再結合為新的事物，但魂與法則受到世界限制，無法觸及。

二、因子守恆定律

轉換物之「虛能因子」總量相等，煉能轉換絕不可能憑空增加或減少總虛能因子。但若轉換失敗或其它特殊原因，轉換中的虛能因子可能逸散。

三、差態連虛定律

不同物質或能量的「分解虛能」與「組合虛能」不盡相同，且轉換物差異越大，矩陣越複雜，運行煉能轉換所消耗之虛能也越龐大。

ex:柳揚月後來弄的種族轉換

究造：（似法道，但原理和創造物不同）

部份需使用「虛能」，究造師使用以魔力與技術創造物品，種類如下：

融合究造

將法寶或魔具經過特殊究造後進行融合。

魔道具製造

如契約羊皮紙、護身符、魔力吊墜等。

卡牌製造

以魔力製作魔法或技能存放之卡牌，包含下列種類：

魔法卡：存放魔法於其中，將其打出可瞬發，越強大的魔法越難存放。

傳承卡：存放知識技能於其中，使用後可學習其中蘊含的知識技能。

特殊效果卡：持有者將其打出，能為自身添加獨特增益效果。

怪獸之決鬥卡：利用映像魔法製作的虛擬決鬥卡牌。不具備實際戰力，但其怪獸與牠們的特殊能力精緻有趣，廣受支配體系人士喜愛。

基因改造

洗點、種族外觀或能力改變。

藥物製造

如魔藥、藥粉、秘藥、注射型藥劑等。
