# 精怪體系勢力

半天之巔

樹妖族、花妖族之主要領地，而地精族、蕈菇族與其它小型精怪族群則為兩者眷屬。

時界神廟

位於眾生之源核心地帶的神廟，乃器靈族主要據點，亦有大量仙靈族、魔靈族、建樓族的存在。

夢之殤

光族與七色族的領地，族內成員十分排外，故夢之殤乃人間禁地之一。

---

各大森林

各種面積較大的森林，通常都有小型的樹妖族、花妖族村落，如眾生之源。

各大古墓、劍冢

古墓、劍冢內經常出現神兵族、器靈族、仙靈族、魔靈族。
