# 鬼族素材性（人族與鬼族的伴生制約）

**1.鬼族心臟—強化魔力漩**

「魔力漩」只能用「鬼族心臟」強化（支配體系獵鬼風氣最盛）。

**2.鬼骨粉—藥引**

人族服用之「丹藥」普遍需用鬼族骨粉做引子。

**3.鬼生素—維持生命**

鬼族血液內之「鬼生素」乃維持普通人健康的必須營養素（如同鹽巴），每日需攝取少量。

ps：鬼生素無色無味，但加入料理能帶給人族些許滿足感。

**4.鬼肉—超界突破、強化肉身、食糧**

「鬼肉」對人族而言極度難吃，有如腐爛的嘔吐物混雜糞便的味道，若食用實力遠超自身的鬼族軀體會產生精神污染、錯亂，引發「癲鬼症」。

但鬼肉營養價值高，還能使人在一定程度上獲得超凡的身體素質、再生力，攝取特定蘊含妖力的鬼肉可以獲得妖力，藉由大量食鬼抵達超凡境界者稱為「食鬼人」。

家境稍好之人堅決不吃鬼肉，反而養活許多以鬼肉為食的貧苦家庭。

對於完全沒有武學、魔法、靈力天賦或資源者，食用鬼肉反而是最容易抵達超凡境界的方式。

PS：人族也會養殖鬼族，高級養殖場利用對其相對營養的豬肉或人類奴隸、罪犯做為飼料；普通養殖場則餵食蟑螂、螞蟻、蚯蚓等。

**5.高階鬼族心臟—地源**

仙道若要達到築基以上境界，需感悟「天源」或「地源」，其中地源可用高階鬼族的心臟製作。

**6.鬼油—頂級保養品**

鬼族提煉的油脂本身包含能讓肌膚鎖水、滋潤補水、強抗氧化、抑制黑色素生成、修復屏障、強力刺激膠原蛋白增生、強化肌膚自我修復基礎等效果。

還能讓膚況逆齡，真正做到近似返老還童的功效，是對人族而言最頂尖的保養品。
