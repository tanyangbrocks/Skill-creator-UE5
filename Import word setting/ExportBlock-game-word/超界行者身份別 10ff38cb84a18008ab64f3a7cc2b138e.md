# 超界行者身份別

超界聯盟身份別

經超界聯盟認證的超界行者，可直接進入與超界聯盟合作的各大戰鬥相關公會，經後續簡易測驗、手續後領取相應執照。

常見超界行者有：「冒險者」、「獵鬼士」、「斬妖將」、「異族殲滅者」、「異獸討伐者」、「傭兵」、「獵人」、「攻略者」。

冒險者執照

冒險者為最普遍的超界行者職業，組成討伐隊並登記隊伍名，完成冒險者公會之任務。

任務性質五花八門，從幫居民掃廁所到捉拿通緝犯都涵蓋在內。由於都以賞金做為報酬，因此冒險者又被戲稱為賞金獵人、聯盟打工仔。

PS：任務難度分級與隊伍實力分級相似，只是每級又細分為上、中、下。

冒險者隊伍分類（非官方定義）

冒險者民務團：專門接取非戰鬥委託之隊伍，隊伍總實力多在超凡級。

冒險者素材團：專門獵取野外常見素材之隊伍，隊伍總實力多在征戰級。

冒險者血戰團：專門接取高難度委託之隊伍，隊伍總實力多在主將級以上。

獵鬼士執照

獵鬼士專門狩獵鬼族，由低至高分為一星到九星。其中最低階的獵鬼士也要具備個體征戰級以上實力，並克服對「恐懼鬼氣」的恐懼（若克服不了，無論實力多強都會被獵殺）。

獵鬼士還細分為應對低階的紫鬼、赤鬼、青鬼之獵鬼士，少數菁英能與高階鬼族戰鬥。

獵鬼士的報酬在各區域差異極大，其在六龍聖天的報酬較低，在戴洛艾色的報酬較高。因此獵鬼士主要分佈於戴洛艾色，其同時也是戴洛艾色極為熱門的超界行者職業。高階的獵鬼士可能會對鬼族勢力發起進攻。

斬妖將執照

斬妖將專門屠戮妖怪支，由低至高分為一星到九星。其中最低階的斬妖將也要具備個體征戰級以上實力。

主要負責斬殺威脅到人族的妖怪，較少進攻妖族領地。主要會花大量時間在城內巡邏，尋找躲在暗處的妖怪，只要某偏遠小鎮有一名斬妖將駐守，妖怪就不至於太猖狂。

妖怪多出沒於東域及靠近東域的南域，由於妖怪普遍實力不如鬼族棘手，故斬妖將整體實力較獵鬼士弱。不過東域城鎮對斬妖將的需求大於獵鬼士，畢竟妖怪較能暗中獵食城中平民百姓，對眾人而言更加恐怖。

異族殲滅者執照

異族殲滅者專門屠戮對人族危害極大之異族，排除特別強大的鬼族與妖族，殲滅對象包含哥布林、歐克、鬼兔、獵蛛、血蚊、土蚯、巨翅蟑等，還會再細分不同專業。

由低至高分為一星到九星。其中最低階的異族殲滅者也要具備個體超凡級以上實力，且通常會集體行動。

異族殲滅者在南域較熱門，因東域已被許多妖族佔據，反而壓迫到這些異族的生存空間。

異獸討伐者執照

異獸討伐者專門討伐強大異獸個體，其中又分為魔獸、靈獸、妖獸等三種專業。通常弱級以上就需呼叫異獸討伐者處理，若要應付特別強大的異獸，異獸討伐者會以團體攻略方式討伐之。由低至高分為一星到九星。

專門處理白獸之人姑且也算異獸討伐者，但可能會被同行恥笑（畢竟要戰鬥的最強對手也僅是普通獅子、老虎、野熊等。）

傭兵執照（通過超界行者考核後可領取）

傭兵主要受僱於商人、政府、貴族，或加入某方勢力，做為護衛、打手、刺客等。

傭兵是除冒險者外第二普遍的超界行者職業。

傭兵沒有具體考核內容，實力標準完全以個人履歷和超界行者階級衡量。

獵人執照

獵人主要以蒐集難以取得的珍稀資源做為收入來源，依照在自身領域所做成就可分為一到九星。

獵人種類繁多，通常為個人獨自行動。常見有寶藏獵人、食材獵人、魔獸獵人、靈獸獵人、奇珍獵人等。

獵人沒有明確的主要收入管道，多半以追求個人興趣和自我實現為主要目的。

獵人的純戰鬥能力不一定強大，但皆是某領域的超級行家。

關於超界聯盟&超界行者

1.一名超界行者可以同時擁有多重身份，但各執照所累積的成就積分皆個別計算。

2.超界行者沒有年齡、身份、種族限制，只要實力夠強，皆可成為超界行者；同樣的，只要實力不足，無論身份多尊貴，都無法改變超界行者等階。
