# 泰克諾爾簡介

永恆聯邦泰克諾爾，是科技高度發展的先進國家！
人民就業類別以工業及服務業為主，國風也並非實力至上，普遍以學歷、權勢、資本做為社經地位依據。
電子產品與互聯網發展出豐富的資訊媒體、娛樂、影視產業。而人工智慧、互聯網、虛擬實境、人體基因改造等高端技術更讓產值飆升！

-

再加上嚴謹的法律、教育、政體，以及大量高便捷基礎設施，人民平均所得與生活品質遠高於其它兩國。
軍事面，該國採邊際自動化防禦系統、機戰神軍隊等高科技系統嚴密保護，使他們較少受妖族侵襲，反而更注重國內經濟與治安問題。

-

然而，除了少數使用【X機鎧】之【機戰神】、進行基因改造者、或嘗試接觸他國體系者以外，人民皆養尊處優，不具戰鬥能力。
頂尖實力也只有核武、禁忌導彈可匹敵修煉體系之元嬰期、魔法體系之七階法神等存在，因此才沒有將另外兩大勢力併吞，形成如今三國鼎立局面。
「發達而不戰」—這就是泰克諾爾！
