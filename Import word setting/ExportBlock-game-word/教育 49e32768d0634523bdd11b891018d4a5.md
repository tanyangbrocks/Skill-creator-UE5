# 教育

這世界的學生所學知識難度與複雜度，普遍為地球的100倍以上。

不過他們有強大的演算能力、過目不忘的精神力、驚為天人的悟性等，還是有許多學子能夠駕馭這些知識。

教育情形：

修練體系（東域）

在這以修練、武力為尊的世界。雖然民生產業能獲得的利潤不高，讀書幾乎是唯一能發家致富的途徑，但大家仍認為只有鍛鍊出實力才能自保。

對貧苦家庭而言，靠讀書來賺錢是條耗費資源、虛無縹緲又高風險的路，寒門書生多半因太好欺負而被惡霸凌虐至死。

普通知識多為書院或皇室教育等機構，是給權貴唸的；但亦有大量培育武道、仙道之學院或宗派，乃窮苦百姓的去處。大家都嚮往成為武者或修練者，但實際能如願者乃鳳毛麟角。

支配體系（南域）

教育分為「學術派」與「實戰派」，學術派認為必須依靠知識、學問，才能帶給大家更好的生活；實戰派認為必須增強人民的戰鬥實力，才能抵禦異獸、異族的侵襲，以保障人民安全。

兩派勢均力敵，通常沒有魔法者或魔法不出眾者，只能步向學術派一途。

科技體系（西域）

西域普遍施行二十四年義務教育，學生們雖不需鍛鍊武力，但課業壓力極大。

社會風氣上，各地區對不同領域的重視度不盡相同，但「戰鬥」相關領域普遍為最下等。

有部份人民極端功利主義，重度歧視另外兩大勢力。

表面上工作機會供過於求，人民生活富裕，但眾人相較之下卻背負極大心理壓力。
