# 魔陣之杖

外表與魔杖相似，法杖內可安裝許多的「法陣模組」，在有足夠魔力供應下，魔力會從最尾端的法陣模組一路流動至前端，自動發射魔陣之杖內紀錄的固定術式。

其優點是使用者完全不需自己編織術式，甚至可以不會操縱魔力，只要在有瑪納儲備的情況下，就能輕鬆發動複雜的魔法。

缺點是魔陣之杖和法陣模組的價格昂貴，且魔陣之杖的發動需消耗大量魔力，若使用者瑪納量不足，需時刻安裝魔晶充填。此外，在不更換法陣模組的情況下，法杖內儲存的魔法固定，使用者不便應付各種突發狀況。

★基本上魔陣之杖的客群完全針對部份不擅用魔法，卻又缺乏自保能力的貴族，以及想假裝自己擅用魔法的、彰顯身份地位的貴族。

**法陣模組**

又分為「術式法陣模組」和「調整法陣模組」。

術式法陣模組

儲存規格化的直擊術式。

調整法陣模組

對前面的術式法陣進行改造，例如增加術式強度、射速、範圍，或是讓術式附加其它元素屬性或特性等，但可能會影響魔力消耗。

如：加速、威力強化、雙重射擊、迴旋、鎖定、漫射、附加毒元素、節能等。

★若使用者非常不熟悉魔力運作，使用危險的法陣模組搭配，或魔陣之杖發生故障，很有可能導致法陣模組運轉失靈，甚至讓魔陣之杖自爆。
