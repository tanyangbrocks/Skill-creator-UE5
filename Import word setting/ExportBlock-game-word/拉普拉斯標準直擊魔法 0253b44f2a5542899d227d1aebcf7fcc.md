# 拉普拉斯標準直擊魔法

魔法界第一法師訂定，共224種直擊魔法，前108為通用魔法（旋環皆可），108～200為環魔法，最後24種為雙重魔法。
每種魔法皆有標準的詠唱、魔印或冥想等編織方式。

按照理論，透過冥想可編織此處所有直擊魔法，只是寰界數越高的魔法門檻越高。

📌注意！這僅為人類歸出的直擊魔法。
📌括弧內數字代表該魔法寰界數，「含印」代表有固定標準魔印，反之為「無印」。

📌編號只代表術式複雜度，威力或效果還是看個人實力

📌在通用魔法中，魔力環更適合操縱類魔法，魔力漩更適合破壞類魔法。
