# 幻神聖朝戴洛艾色

原為：神聖皇朝戴內色堤

幻神聖朝戴洛艾色是結合魔法體系和特殊信仰的奇幻國家。
鍛鍊魔法雖也極度依賴天賦，但入門門檻極低。不僅戴洛艾色血統者有超過七成能操縱魔力，就連其餘兩國也有不少人能夠使用魔法。
修煉者之頂點雖遠高魔法使，但由於修煉實在太難，魔法往往被視為比修煉更務實的選擇！

-

此外，戴洛艾色種族文化極多元。有諸多宗教如。教、教等，部分教會權力勝過皇室，信徒也可用信仰力發動神聖魔法。
還有異界種族，如天使、惡魔、精靈等，他們藉魔法與該國人溝通，建立契約。
當人類與異族結婚育子，後代為亞人，如哈頓族（惡魔）、長耳族（精靈）、瓦爾基里（天使）、矮人族（混雜）等。

-

可以說，戴洛艾色乃三大勢力中種族最多元的國度。
然而，該國表象和樂，暗地卻因種族多元衍伸歧視、犯罪問題，再加上民眾普遍能使用魔法，治安難以控管。因此戴洛艾色也是三大勢力中最混亂的國度。
「精彩而混沌」—這就是戴洛艾色！

📌一些補充事項

1.戴洛艾色平均壽命為43歲，因獵鬼士的興盛，使得眾多年輕人選擇與鬼搏殺，容易產生傷亡。

2.戴洛艾色人自然髮色多樣，且通常與自身親近元素相關（如紅髮擅火、藍髮擅冰等，但可藉由染髮欺敵），瞳孔顏色亦和髮色相似，身材個體間差異大，但平均體格略高壯於東域。

魔法使由於長年待在室內學習且穿著法袍，通常膚色較為白皙；騎士則與之相反。

天生平均身高體重—

男：180cm/70kg

女：165cm/57kg

3.戴洛艾色崇尚魔法，不具魔法者普遍被認為下等人，易遭歧視。但由於學術派和實戰派勢均力敵，故從事學術領域的無魔法者也不至於太慘。（因此，「無魔法者只能走學術」是大家的普遍思想。）

PS：有魔法但天賦弱者，通常會成為使用鬥氣的騎士。

4.因戴洛艾色的種族、體系、宗教、身份複雜，治安混亂，故大家普遍對外地人或陌生人保持警覺，哪怕是小孩子也會學習一定的防身能力。此外，容貌太出眾者容易引來禍端，故他們總是進行易容或遮住面部。
