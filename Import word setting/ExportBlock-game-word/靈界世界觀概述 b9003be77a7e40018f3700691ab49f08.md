# 靈界世界觀概述

[靈界種族（特殊存在，全具魔）](%E9%9D%88%E7%95%8C%E7%A8%AE%E6%97%8F%EF%BC%88%E7%89%B9%E6%AE%8A%E5%AD%98%E5%9C%A8%EF%BC%8C%E5%85%A8%E5%85%B7%E9%AD%94%EF%BC%89%20d20036fd46024a02bb16a65bf286bf1d.md)

靈界由下界法則之神所造，乃虛無空間，無邊無際。

用於存放未能達到神之領域，卻又不願重入輪迴的極端強大之魂。

順便平衡世間之「魂」的能量。

原生的靈界物種：

靈界會因生靈的情感而自然產生天使、惡魔；會因法則的不穩定而產生精靈。

但自然產生之靈界生物通常智力與戰力較低，多為外來者的眷屬或奴隸。

外來的靈界物種：
經歷過「死亡」的靈魂在進入輪迴前將趨於不穩定，在這期間：

吞噬足量靈魂者可以成為惡魔（通常是魂修之路，起始惡魔貴族）
幫助足量靈魂進入輪迴者可以成為天使（通常是死神、冥界人員之路，起始能天使）

以此狀態保留666年有兩種情形：
1.永久消亡，靈力回歸虛無
2.與自然結合，成為精靈（起始中位精靈）

魂到達極端後會變為天使或惡魔，將脫離輪迴成就永生，並進入靈界生活，但不具備肉體。

📌天使與惡魔在靈界對立，分別構築了天使島與惡魔城。
📌靈界居民可藉由靈界門干涉任何界面，但需受肉才能長久存在於世。

📌外來的靈界物種通常都是高階存在，統治著靈界。

📌靈界物種有無限接近「神之領域」的特性，故難以被消滅。
