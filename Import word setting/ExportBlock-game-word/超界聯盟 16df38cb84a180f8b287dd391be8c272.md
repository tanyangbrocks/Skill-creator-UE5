# 超界聯盟

超界行者
泛指一切透過修煉、血脈、機緣等因素而超越人類正常戰鬥實力界限者。為極度模糊、概括的統稱。

超界行者可前往「超界聯盟」進行考核。

超界聯盟
由封聖峰會成員所創，其勢力星盟同等規模，乃許多超界行者的主要收入渠道。

超界聯盟在整個天輝大陸地區，甚至其餘可能有人類活動的區域皆有成立分區總部、小分部等。建築、設備與服裝樣式會與當地文化結合，但盟徽皆固定。

超界聯盟小分部—非分區小分部

除了各分區總部可另建小分部，在其餘偏遠地區、乃至異族領地，皆可能存在由合夥單位所建造之非分區小分部。

但受限於人手不足、物流不完善、安全因素等，容易有設備不齊全，甚至考核主辦方違規等情形。

PS：為推廣超界聯盟相關產業鍊，超界聯盟之加盟與審核之門檻極低。不僅建築沒有固定樣式，考核標準與服務品質也可能不一致。

長久以來的推廣雖使其成為天輝星盟最知名的實力驗證機構，卻逐漸缺乏公信力。

超界聯盟常見設備

低階設備是所有分部必備，中階設備可能有部份偏遠地區的非分區小分部未實裝，高階設備基本只有分區小分部、總部、大總部會實裝。

測驗型

分級考核測驗場：初階設備，用於考核超界行者的綜合實力。

基礎六維測定儀：中階設備，能測定基礎六維，並依測定結果做為考核分級依據。

事務型

身份登記處：初階設備，登記超界行者的考核後身份別，並處理其相關行政事務。

公會接洽處：初階設備，不同超界行者身份別之公會派專員進駐於超界聯盟，便於超界行者認領、轉換身份別，並執行該身份別之事務。

戰鬥訓練型

訓練室：高階設備，進入後將由高級源能人工智慧掃描使用者，並根據其需求計算出一套最適合的訓練方案。

訓練室能模擬訓練環境、情境，創造敵人來供使用者訓練，甚至會鞭策之。

對戰室：中階設備，進入後將傳送至通用對戰大廳，可於大廳內和人約戰，或隨機匹配相似戰力者，前往大廳內的專用擂臺戰鬥。

通用對戰大廳內，任何生靈死亡後將自動傳送回對戰室，正常情況不會真正死亡。

擂臺：中階設備，視分部規模大小存在一到十個擂臺。擂臺上的守擂者需隨時接受他人挑戰，勝過守擂者即可上擂臺守擂。

守擂者只要待在擂臺上，就能定期得到超界聯盟給予的獎金。

生活型

超界旅館：中階設備，便於執行任務的超界行者中途居住。部份旅館中會有舞廳、夜店、賭場、電影院等娛樂設施。

超界商會：中階設備，販賣知名超界行者週邊、生活物資、常用道具。

道具市集：中階設備，許多商人會在超界聯盟旁販賣各體系超界行者常用物品。
