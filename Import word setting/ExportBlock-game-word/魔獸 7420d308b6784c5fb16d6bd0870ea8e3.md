# 魔獸

善用魔力，以「血脈」區分之等級（同樣分天地玄黃）。

根據史書記載，世間魔獸共有三萬四千五百九十四種，以下為常見種族：

📌橙色文字表示對人類有攻擊性，需加以討伐

生活魔獸
大白：白色大團子，力大溫馴，通常被拿來搬運或拉車。
小黑：黑色小團子，通常被養來處理雜物。

大白與小黑是經由人工改造後的「彈羊」。
