# 奪舍

魂修或尚存靈力的個體奪取其它生物肉身的行為。

奪舍注意事項：

1.被奪舍者精神力越低，越容易使其軀體無法承受而崩潰。所以修士奪舍凡人的難度反而比奪舍修士高。

2.被奪舍者若精神力高出奪舍者許多，或有特殊法門。有可能排斥、重創，甚至反噬奪舍者。

3.若奪舍不同能量體系者、能量過強者、種族血脈差異大者，很可能導致肉體與靈魂排斥。

若不能解決此問題，將招致自我毀滅。

4.無論奪舍成功與否、對象為何，奪舍本身都會對魂魄造成巨大損傷。故奪舍成功後通常需耗費較長時間修養神魂。

★一般而言，奪舍並非修士的正常修煉手段，多數名門正派也極力抵制奪舍行為。

若奪舍他人被發現，可能遭各大勢力追捕。
