# 仙武（輔助修煉）

同源複合能量：

[仙道（靈玉）](%E4%BB%99%E9%81%93%EF%BC%88%E9%9D%88%E7%8E%89%EF%BC%89%20ce0ca431b91d4df88398686bc515dc85.md)

[武道（神兵）](%E6%AD%A6%E9%81%93%EF%BC%88%E7%A5%9E%E5%85%B5%EF%BC%89%20373c35151dd2416484519c27eb5b187f.md)

---

主要用於開啟「神魂五竅」。

對靈力天賦不高者而言，仙武在體內的流動可能比純靈力還快上許多，因此有時混合出仙武能大幅提升修煉速度。

由於仙武融合了真氣，故會隨經脈運轉，其威力遠大於普通真氣，但無法進行「靈力運用法」。
