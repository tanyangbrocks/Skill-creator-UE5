# 半神體系（具備神性、更能領悟法則）

具備神性、更能領悟法則之稀少神秘體系。

根據史書記載，半神體系共有五十二族，以下為常見種族：

—–玄階—–

千翼族：外型似人，背生千翼，擅空間法則，多分佈於聖域、高空雲層。

千手族：外型似人，體生千手，擅時間法則，多分佈於聖域。

無常族：沒有固定的外型實體、顏色，魔力極為強大，擅以精神支配其它個體。

—–地階—–
邪根族：

[邪根族](%E9%82%AA%E6%A0%B9%E6%97%8F%202a4f38cb84a18060ba5efab93a365d07.md)

天狐族：又稱九尾狐族。擅幻術，可操縱部分因果、造化法則，能改變運勢，多分佈於森林、河川。

天狗族：善於使用肉體、靈力、真氣、妖力、神聖力，具有極強威嚴的種族。後背翅膀可助其高速飛行，個體有少許神性。
種族固有技「神通力」能助其發揮更強大的力量。
