# 制衡譚：伴生種族誕生

蒼天的煩惱

「蒼天」創造蒼究世界除了滿足自身興致與心願，還有另一項重要目的—「積累價值」。

[神界基本設定](%E7%A5%9E%E7%95%8C%E5%9F%BA%E6%9C%AC%E8%A8%AD%E5%AE%9A%209c0ee12bbc7c491289a18d200cc78dec.md)

但，蒼究世界持續著破滅與再生，如此根本無法為蒼天帶來價值，這令蒼天極為苦惱。

蒼天的想法

既然種族間的衝突無法避免，那就試圖將衝突分割成數塊，為各族塑造專門且合適的敵人，來使其無暇侵略他族，也不易變得過度強大，更難以大規模掠奪資源。

雖然如此可能讓族種間衝突增加，但都是一對一、彼此間距遙遠的衝突，反而產生一種「制衡」。

制衡譚：伴生種族誕生

蒼究在某次「眾生的呢喃」中加入「伴生種族」設定。

它讓所有種族與其「伴生種族」一同誕生，並藉由調整部份基因法則，使伴生種族彼此呈「與生俱來的敵對關係」。

由於蒼天在發動「眾生的呢喃」時並非完全沒有生靈存活，因此該事件也被紀錄下來。做為蒼天本尊的行動，後世又將其稱為「制衡譚」。

---

人族的興起

起初，由於鬼族個體普遍比人族個體強大，人族還是被大量捕食，處於極度弱勢。但由於人族此時的敵人只有鬼族，因而能有足夠時間建立據點、制定策略、發展體系，還成功向鬼族勢力發起大規模清剿。

最終反倒是鬼族依靠其頑強生命力和可怕繁殖力才得以生存，此後兩族達相互制衡關係。

人族勢力日益強大後，選擇與其它種族交好、共榮，以此避免後續的戰亂，畢竟文明與和平才是人族強盛的最大途徑。
