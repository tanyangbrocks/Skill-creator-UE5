# 頓悟

修士最大的機緣之一。

當修士在修煉、參悟、觀想、感悟、消化思緒時，有極低機率進入「頓悟」狀態。

頓悟作用

頓悟狀態能使該名修士在極短時間內快速領悟大量訊息。

如在一瞬間參悟原本需耗時數年的神通術法，在十幾天內領悟原本需花數十年領悟的大道進度，甚至於修煉主功法時進入頓悟，讓所修功法跳好幾階，導致修為瞬間暴漲。

頓悟風險

頓悟持續時間通常較短，短則一剎那，再長也通常不會超過一年。

修士在任何時刻都可能進入頓悟，頓悟時身體將呈現打坐狀態，毫無防備。一旦此時被人偷襲，輕則被迫退出頓悟錯過機緣，重則當場被擊殺。因此，若修士看到自己的好友頓悟，通常會在其身旁為他護法。

頓悟發生率

頓悟狀態發生率與頓悟回報取決於「悟性」，悟性高者可能很容易觸發頓悟，悟性低者可能終其一生都無法體會頓悟。
