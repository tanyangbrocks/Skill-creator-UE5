# 翼神龍族VS瑞神龍族

兩族的伴生制約

翼神龍族與瑞神龍族本出同源，因此有諸多相似之處，如軀體構造、種族固有技、進化路徑、血脈分支、高貴自尊心與強大實力等。

但由於兩族所使用之能量體系、理念大相逕庭，導致兩族互看不順眼，再加上兩族所需資源大幅重疊，故經常發生鬥爭。

此外，兩族主要在爭奪「龍的正統性」，以維護自身血脈之尊嚴。

兩族實力比較

由於靈力之實力上限普遍強於魔法，加上瑞神龍族的祥瑞與信仰力強大，故同等瑞神龍族大多略強於翼神龍族。

但由於翼神龍族有十分之一個體能使用「䵻龗」，再加上有異常強大的妖魔龍，故翼神龍族頂尖實力強過瑞神龍族。

此外，由於雙方都能掌控法則，且肉體、生命層次強大，故很難有個體於戰爭中消亡。
