# 各種最上位系統

利亞系統（贓物λεία系統）（希臘語）

系統靈：阿勒疤堤鞏（捕食者αρπακτικό）

形象是一可以用各種方式偷取他人系統，可將沒用的低階系統轉為能量石。
可偵測系統位置、等級，協助系統間互動。
盜賊系統背叛了本源核心基地

取得系統方式

1.透過殺戮直接取得

2.直接吸收無主系統

3.偷取四步驟

（1.讓對方說出：「盜賊系統，開！」（後可用言靈魔法輔助）

（2.強制經歷他人的過去

（3.說出對方最想聽見的話

（4.完成對方許下的一個願望，失敗將失去此系統

萬能升級系統（）

可透過鍛鍊、受傷、戰鬥來提升等級，增強各式屬性點

力量、速度、體力、再生等，甚至對局部的器官進行強化或改造，提升顏質、皮膚、視力等。
甚至修煉體質、魔法體質也能改造出來。

無限技能系統

透過重複特定行為來取得相應技能，並提升等級。

發動「技能奪取」來從對方記憶中、系統中的相應能力提取出技能（一切被系統判定為「技能」者）。

技能奪取發動條件：讓對方感到恐懼並失去戰意，被奪取者精神會受損
可分解技能以取得強化點，增強部份技能，或是融合、進階等。

分為

技術：廚藝、劍術、

被動技能

主動技能（極消耗體力）

奪取來的中低階系統可化作被動技能！（亞空間口袋、記憶庫）
就算初階系統也是能輔助超越人類極限的力量

魔法本源系統

魔力源於心靈之力，可吸收他人情感能量來增強魔力。

各屬性需不同情緒：

火 憤怒與憎恨

水 思念

木 嫉妒

風 快樂與自在

土 平淡與乏味

光 希望與積極

暗 絕望與頹喪

可自行創用魔法，或是進入魔法塔學習魔法，強化魔力。
