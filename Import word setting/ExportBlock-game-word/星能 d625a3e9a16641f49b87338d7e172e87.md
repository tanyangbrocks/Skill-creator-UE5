# 星能

星能性質：

最為純粹、皎潔，古老神秘種族可吸收星辰力量，變幻無常，可用於替代任何神力以外能量。

其干涉法則的優先權比神話種族高，故為神話種族所懼怕，又最想支配的能量。

星能應用：

星轉

將星能轉換為魔法、靈力、奧術等其它體系能量。轉換對象能量等級越高，單位可轉換量越少。

星能複合

讓星能融合其它能量（如魔法、靈力），提升該能量之威力與層次。

星能高階複合

將複合能量體系（如鬥氣、奧勒）再融入星能，提升該能量之威力與層次。

PS：由於融入星能的能量層次僅次於神力，故星能使用者多半可立於不敗之地。

星能嬗變

將物體、靈體注入星能或浸泡於液狀星能中，物體將發生變質而產生特殊進階、變異、崩壞或拆解，甚至誕生靈智。

星能嬗變為不穩定刺激，若非星能使用者，很難預估嬗變後結果。

---

星能使用者：

星之子

通常為與世界之誕生有重要聯繫，乃該世界獨一無二的個體。其體內甚至本就存在星能，且品質與力量極為出色。

神話種族

龍、鳳、泰坦等部份神話種族天生就擁有掌控星能之力量。

摘星者

不符合上述兩種，而是以特殊手法取得星能者，通常星能相對較弱。
