# 神界種族（皆為神族）

任何達到「神」之領域並來到神界的物種皆保有原先種族特性。只是會受神力、職階的影響而成為不同存在，並都能得到永恆壽命（但可以被滅殺）。

[神界基本設定](%E7%A5%9E%E7%95%8C%E5%9F%BA%E6%9C%AC%E8%A8%AD%E5%AE%9A%209c0ee12bbc7c491289a18d200cc78dec.md)

天神

領取職位之神即為「天神」，神格保底有下位神。由於他們能掌控自身神職之「法則」，實力較一般下位神強大許多。

此外，執掌凡間事物的他們更容易累積價值。

PS：無論價值有多高，只要沒有向下界法則之神成功申請掌管任何界面的任何事物，皆無法成為「天神」。

佛

若生靈之心境與磨難達到一定程度，可選擇在達到神之領域時「成佛」。

在成佛的瞬間，神格保底有中位神。

菩薩

「救贖功德類價值」單獨達到神之領域時，可選擇成為菩薩。菩薩雖實力不強，但在神界廣受眾神景仰。

成為菩薩的瞬間，神格保底有下位神。

魔神

「罪孽類價值」或「屠殺業力」單獨達到神之領域時，可選擇成為魔神。

魔神受眾神排斥，因而獨自開闢「魔神界」來生活。大多數的魔神則是獨來獨往，喜歡派使徒禍害人間。

（但也有少數魔神是善良的）

成為魔神的瞬間，神格保底有下位神。

-

「神話物種」之神

神話物種只要修煉足夠時日，便能順理成章地成神。他們就算沒達到上述條件，也保底有「下位神」之神格。

-

星辰
累積價值達到「神」之領域卻沒有上述情形者，將成為「星辰」。

大多數的神界物種皆為星辰，神格階級乃「準神」，由於實力較弱，可能會遭受其它神歧視。

-

但星辰可以憑藉累積價值來成為「星座」，神格將被提升至下位神。若持續累積星座價值，也可能成為中位神、上位神……

異神

價值尚未達到神之領域，但實力足以突破神界障壁；靠自身力量從凡界闖入神界者。

他們雖無神格，但力量足以與神並肩，並且死亡後很可能因「突破神壁」之成就而成為高位神。因此眾神雖然看他們極不順眼，卻也不敢輕易招惹。

PS：但若異神在神界做亂，眾神也可組織軍隊進行討伐，並向「至高神壇」申請剝奪該異神死後獲封神格之資格。

神種

在神界出生的生靈。

其誕生之當下，神格等同於生育者當中最高者，種族則受多重因素影響。

神仙（未必有神格）

由於仙界與神界同層次，在仙界的仙道修煉者可來到神界定居。這類修煉者又被稱作「神仙」。

它們透過修煉來達到神之領域，但其累積之「價值」未必足以擁有神格，因此一旦境界修為倒退，有可能被神界排除。

PS：通常只有仙界大能才敢進入神界，境界較低的仙道修煉者若擅入神界，很可能觸怒性情高傲的神，並被其滅殺。
