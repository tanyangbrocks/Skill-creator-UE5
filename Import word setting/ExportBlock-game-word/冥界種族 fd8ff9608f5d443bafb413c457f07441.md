# 冥界種族

根據史書記載，冥界種族共有九十九族，以下為常見種族：

骷髏：以魂力驅動、或是靈魂回歸的骸骨。若能在成為骷髏後保留靈智，便可透過修煉魂力來提升實力。

殭屍：受殭屍病毒感染，或是怨念極深的屍體。以屍體生前同族為食，智力低下，對進食有強烈渴望。獵食越多血肉，將能使肉體產生更多變異，並恢復越多靈智。

木乃伊：以特殊方式下葬的死者，死後被詛咒的魂魄仍控制著軀體，具備相當強的魂力。

幽靈：生靈死亡後到轉世或回歸天地前的過渡狀態。

鬼物：凡間或冥界怨念深重的幽靈，形態不定，可能成為詛咒人的咒靈或啃食生命體的怪物。

鬼物按實力分級為「整、亂、奇、域、淵」。

克爾柏洛斯：又稱地獄犬，負責看守冥界入口。

亡靈：奇以上之幽靈蛻變而成，魔力和魂力天賦極強，可吞噬異火、詭氣、奇冰做為自身本源來維持自身存在。

鬼怪：妖力、魂力、靈力都極強，還具有受肉後肉身強大的種族特性。

死神：亡靈或幽靈透過修煉魂術、考核、魂灌等方式接受死神之力而成，具備專屬的死神鐮刀。負責收割停留在世間的幽靈、亡靈等鬼物，將其移送至冥界；或做為維護界域秩序的戰鬥人員。
