# 聖靈體系（遠古物種，全具魔）

根據史書記載，聖靈體系共有七十二族，以下為常見種族：

——黃階——

耀精族：軀體為一顆飛行眼球，兩側各長三根觸手，體膚偏紫，通體散發柔和金光。天生具備高速飛行能力，甚至可短距離瞬移，智力普通，肉體偏弱，善空間屬性魔法與幻術。

幻靈惡族：軀體是一個形似燈籠，散發粉紫色光芒的半透明生靈，沒有眼睛，但有著始終保持露齒微笑的嘴，漂浮在半空。

天生具備隱身能力，能在虛實間轉換，且能一定程度吸收各能量體系之能量並化為己用，從口中釋放高能射線。

智力偏低但性格頑劣，喜歡聚在一起惡意屠殺、凌虐異族，被各族稱作「聖靈的黑暗面」。

——玄階——

獨角族：軀體似一匹通體晶瑩潔白，長有獨角和雙翼的馬，散發白芒，移動時會拉出一道彩虹軌跡。

智力偏高，性格溫順且正義感強，會主動照顧、憐愛弱勢種族，極度愛好和平。

能從獨角釋放所有基礎元素與部份進階元素之魔法，其目光有治癒生靈的效果。

歌訟族：外形似人族，但通體潔白柔軟，無嘴無毛，身高低於一公尺，四肢皆為一顆圓形小肉球。

不具備戰鬥能力，喜愛歌唱，能以意念傳播其美妙歌聲，讓所有聽見歌聲的生靈放下憂愁與對立，和平共處。

但由於其肉質鮮嫩美味，食用後甚至能改善生靈體質、延年益壽，故被各族大肆捕獵，或被強行奴役。約莫在大分裂晚期時，野生歌頌族幾乎絕跡，倖存的族中生靈也不再為眾生歌唱。

——地階——

超負義族：外型為一個金色巨型鳥喙，鳥喙四週有八顆白色眼球環繞，漂浮在半空。

智力普通，能透過「信仰自己」來獲得神聖力與價值，對神聖力的掌控極強，甚至能自行模擬神力效果（但不是真的神力）。

自稱「超正義族」，但其心中的正義就是自己的慾望，會理所當然地以正義之名掠奪異族，故被其它種族命名為「超負義族」，負為「自負」之意。
