# 源導設備

日常

- 全息智慧手環
    
    最常見的源導設備，泰克諾爾境內幾乎每人都有。（類似手機）
    
    手環會在眼前投影出一道光幕，光幕連接網路，並擁有多核心處理器，本身就是臺通用超級電腦。
    
    手環內建AI助手、身體數據監測、定位、代入虛擬實境、萬波充電等功能。多數功能僅需耗費電力，少數功能需使用源能，故有兩個插孔。但因為有高效萬波充電儀，所以通常不需特意充電。
    
    可自行決定由心念操縱、手動觸控操作、聲控操縱，並自行「開啟/關閉」操作模式。（心念操控會消耗較多源能，而充源的價格遠高於充電，故較少人使用之。）
    
- 充源插座
    
    每個充電插座旁都有一個充源插座。雖然源能傳輸零損耗，但源能本身比電能昂貴數倍，故日常活動較少使用充源插座。

- 源導挪移器
    
    建立於大城市的巨型挪移建設，民眾事先預訂目的地後，前往並踏入源導挪移器中，瞬移至另一座大城市的源導挪移器。
    
    源導挪移器發動頻率不固定，尖峰時段發動率最高可達十分鐘一次，每次發動都有時程表所列之固定目的地，一次發動最多可容納數千人。由於扭曲空間所需源能較龐大，故源導挪移器票價相對昂貴，是富人或出差者趕時間的手段。
    
    跨域源導挪移器
    
    泰克諾爾在東域、南域、北域甚至其它大陸也有佈置源導挪移器，是企業家、旅行者及政府人員出境的主要媒介。外界的挪移器受到嚴密保護，若其遭到破壞或竄改，會將訊息即時傳回泰克諾爾。
    
    此外，外界挪移器若要送人回國，需經內部相關機構同意。
