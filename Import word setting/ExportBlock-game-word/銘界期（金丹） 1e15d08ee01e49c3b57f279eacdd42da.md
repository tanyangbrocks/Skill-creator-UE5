# 銘界期（金丹）

金丹始可自行施展遁術，遨遊天際。

金丹條件：

1精神力化實，足以凝聚元神

2.吸收道種

「結丹」之舉本是逆天而行，絕非水到渠成的過程。必需吸收此方界面一絲天地之力，以此為核心、導引，才可凝聚金丹。
而築基期修煉者要吸收天地之力僅有一法，即吸收【道種】！

所謂【道種】，泛指具備天地之力，又恰能被人體吸收之物。然而，世間【道種】大多靈性劇烈，生命層次較低的築基期修煉者一旦攝取，必定爆體而亡！

根據歷史記載，已知靈性較溫和、能被築基期修煉者吸收之【道種】僅有九十九種，【道種】品質分為「天」、「地」、「玄」、「黃」四階，【道種】階級會影響日後金丹銘文
