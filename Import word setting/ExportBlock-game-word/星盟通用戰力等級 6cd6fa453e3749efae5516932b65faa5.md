# 星盟通用戰力等級

[天輝星盟](%E5%A4%A9%E8%BC%9D%E6%98%9F%E7%9B%9F%20206f38cb84a18077af64f98bb47896ff.md)

個體、隊伍實力分級

由天輝星盟與冒險者公會聯合制定之跨種族、跨體系的統一實力分級，主要由冒險者公會與其分部進行測試。

測試分為個體實力測試與隊伍整體實力測試。兩者使用相同的分級，但「徽章」僅有個人實力測試達標才會發放。

徽章實力分級如下：

[徽章通用能力](%E5%BE%BD%E7%AB%A0%E9%80%9A%E7%94%A8%E8%83%BD%E5%8A%9B%20282f38cb84a1806ca5d9fb3a4d7ebe12.md)

學徒級（灰徽章）：未達組建隊伍的最低要求。若單一個體為學徒級，也僅比正常人強大一些，為合格超界行者的最低門檻。

個人必須達到學徒級以上才算成為超界聯盟成員，始可創建或加入隊伍。

超凡級（黑徽章）：以隊伍而言，是可組建冒險者隊伍的最低實力門檻。若單一個體為超凡級，無論多少普通成年男子都無法與其為敵。

[黑徽章特權&能力](%E9%BB%91%E5%BE%BD%E7%AB%A0%E7%89%B9%E6%AC%8A&%E8%83%BD%E5%8A%9B%2010ff38cb84a1808493f5d5ad8596caa4.md)

征戰級（綠徽章）：普通探險隊伍水準，可敵初級異獸。單個體征戰級者通常能輕易毀壞建築。

[綠徽章特權&能力](%E7%B6%A0%E5%BE%BD%E7%AB%A0%E7%89%B9%E6%AC%8A&%E8%83%BD%E5%8A%9B%2010ff38cb84a180cd98c8cbacb1e73900.md)

主將級（藍徽章） ：主將級個體可作為攻擊主將的實力，乃冒險者中的強大存在，可敵中級異獸。

[藍徽章特權&能力](%E8%97%8D%E5%BE%BD%E7%AB%A0%E7%89%B9%E6%AC%8A&%E8%83%BD%E5%8A%9B%2010ff38cb84a1805b908ef0f66ea843d6.md)

群星級（黃徽章）：少數精英強者，已可成為明星的強大存在，可敵高級異獸。

[黃徽章特權&能力](%E9%BB%83%E5%BE%BD%E7%AB%A0%E7%89%B9%E6%AC%8A&%E8%83%BD%E5%8A%9B%2010ff38cb84a18004ba74c63c9d4a5040.md)

耀日級（紅徽章）：必須前往分區總部或大總部測定。單體耀日級有一騎當千、屠城之能力，公會測試員水準，可敵領主異獸。

[紅徽章特權&能力](%E7%B4%85%E5%BE%BD%E7%AB%A0%E7%89%B9%E6%AC%8A&%E8%83%BD%E5%8A%9B%2010ff38cb84a1808ab49bdc26a50a3ddb.md)

巔峰級（紫徽章）：必須前往分區總部或大總部測定。單體巔峰級為超乎常理的怪物，巔峰級以下無論多少人都無法與其為敵，可戰君王異獸。

[紫徽章特權&能力](%E7%B4%AB%E5%BE%BD%E7%AB%A0%E7%89%B9%E6%AC%8A&%E8%83%BD%E5%8A%9B%2010ff38cb84a180d0b249e8ec82a9ec2b.md)

**巔峰級以上（客製化專屬徽章）**

巔峰級以上沒有官方測定標準，只能透過事蹟紀錄、名聲等決定，或是前往封聖峰會申請測定考核。

巔峰級以上成員可向超界聯盟訂製客製化專屬徽章，徽章上會標注該成員的姓名、階級、稱號、成就等。

特異級：宛如傳奇般的稀世強者，可敵禍難異獸。

超特級：實力遠強於普通特異級，但未達封王級者。

-

封王級：常為各大勢力的王者，可敵災殃異獸。

超王級：實力遠強於普通封王級，但未達封帝級者。

-

封帝級：常為超級勢力的王牌，可敵劫災異獸。

超帝級：實力遠強於普通封帝級，但未達封聖級者。

-

封聖級：世間頂點存在，舉手投足都會引起世界動盪。聖級以上強者受到同級監視、相互制約，不可輕易動用力量。

超聖級：能夠碾壓普通封聖級強者，但還不是當今世上最強大的第一梯隊。

-

滅世級：已擁有獨自滅世能力的個體，用於形容傳說中的幾位存在，如拉普拉斯、蒼究沉浮、人祖等。通常擁有此種實力者早已前往其它世界。
