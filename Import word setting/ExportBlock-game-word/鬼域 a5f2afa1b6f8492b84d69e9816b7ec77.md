# 鬼域

鬼族的種族固有技，每隻鬼有自己的鬼域，能力各不相同。鬼域越高階，能力越強。

此外，鬼族也可藉由修煉、捕食等方式讓自己的鬼域更加強大。

捕食鬼域（S鬼域）

鬼族個體出生即擁有的基礎捕食用能力，常見如……

極黏唾液（12%）：續命唾液變得極具黏性，用於製作陷阱或控制對手。

伸縮觸手（20%）：額外擁有能協助捕食的觸手，觸手速度快且可再生。其中又有30%會變為帶有毒、火、雷等屬性觸手，10%會變異為隱形觸手。

穴風口（18%）：張開巨口產生極強吸力，將前方獵物吸入口中。

肉翅（10%）：讓自身得以飛行，可透過訓練提升速度。

基礎提升（15%）：隨機強化任意數量的鬼族共通初階能力，強化幅度也隨機，最高能強化至5倍。

麻痺聲波（5%）：瞬間釋放能使人神經麻痺，暫時無法動彈的音波。出其不意之下能越階擊敗遠強於自身的敵人，是最強的常見S鬼域。

-

其餘10%皆為稀有S鬼域，部份能力強大到不可思議，如……

初生靈智（0.01%）：在獸鬼階段便直接擁有靈智。若提升為智鬼，則智力會大幅增強。

惡食者（<0.000001%）：擁有「惡食之人」與「惡食之鬼」、「惡食之魂」等三大能力。

惡食之鬼：

獵食人類後，能奪取其靈智、記憶，可幻化為該人模樣，完美模仿氣息，並獲得其所有力量和技術。本身有3~5次使用次數限制，使用後恢復一次「惡食之人」的使用次數。

惡食之人：

獵食鬼族後，能奪取其型態能力、專屬鬼域、靈智、記憶、力量、技術，可幻化為該鬼模樣，完美模仿氣息，可隨時切換已擁有的型態。本身有3~5次使用次數限制，使用後恢復一次「惡食之鬼」的使用次數。

惡食之魂：

藉由上述能力，若重複獲得靈智可提升靈智，重複獲得相同型態或鬼域也可將之提升。此外，能選擇將吸取到的事物轉為「惡食力」，用於強化自身技能與基礎屬性。若透過捕食獲取到另一個「惡食者」，「惡食者」將往未知方向進化。

此為現今已知最恐怖的鬼域，該存在將給世界帶來災難。但不僅出現機率極低，一旦持有者被發現，人族和鬼族都會全力將其滅殺。

異能鬼域（P鬼域）

達到智鬼解鎖，常見如隱匿存在、影界轉移、化塵、超速移動、幻術瞳、迷魂煙（讓獵物被催眠朝自己走來）等強力輔助能力。

王虛鬼域（D鬼域）

達到魔鬼或王鬼解鎖，如操血術、控冰術、死靈術等具系統性之強大能力。

特殊鬼域（F鬼域）

遇特殊機緣解鎖，每個體能力差異極大。鬼族個體額外激發之F鬼域數量無上限。

常見特殊鬼域解鎖法：

1.食用超高品質或特殊的人類，如：奇異肉、仙道天驕、魔法天才、泰克諾爾人、高階血脈人。

2.成為魔王或王鬼時機率附帶

3.昇域化時機率附帶

4.瘋狂暴食人族時機率產生
