# 晶煉期（築基）

築基始可御器飛行，始可辟穀。

凝聚築基方式：

1.氣海化靈晶（通常需築基丹輔助）

2.**功法特殊築基法**

開闢築基後續之路：

築基前必需先感悟「天源」或「地源」，才能達到築基之上的境界。否則晉升築基後就是修煉終點。

天源只能等待天地自然生成，地源除了自然生成，還能用高階鬼族的心臟來製作。

築基過程：
將氣海化為靈晶層層疊合，可創造實質靈力。

凝聚十層以上可構築靈山。

凝聚二十層以上可構築靈月。

凝聚三十層以上可構築靈日。（誕生靈日有觸道之資，六靈日代表六道，為築基極限。）

構築靈日將引動劫雷。

築基境界

晶化境（前期、中期、後期）

晶變境（前期、中期、後期）

晶玄境（前期、中期、後期）

晶煉特殊神通

始可御器飛行、辟穀，脫離凡間疾病。
