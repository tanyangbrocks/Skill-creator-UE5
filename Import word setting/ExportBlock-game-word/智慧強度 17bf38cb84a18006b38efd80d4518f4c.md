# 智慧強度

智慧強度包含五大指標：

決策力

包含演算分析、聯想力、應變力、智商、決策制定之綜合能力。

洞察力

包含觀察力、敏銳度、經驗、直覺。

學習力

包含記憶、智商、悟性、耐心等要素。

創意力

包含創意發想、創造力，可能激發各種出奇不易的戰鬥手段。

意志力

包含心理強健、情緒管控、毅力根性、冷靜清明等要素，與靈魂強度高度相關。

---

智慧強度數據化

上述每個指標被星盟以萬分比1~9999做為衡量，9999以上即代表測定對象的該項指標在10000人中超越9998位測試者，故無法將其數據化。
