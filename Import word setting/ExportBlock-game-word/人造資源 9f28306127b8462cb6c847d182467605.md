# 人造資源

裝備

不同地區所製造出之裝備性質差異極大，如泰克拿勒製造之高科技武器、戴內色堤製造之魔劍、六龍聖天製造之法器。

「顧氏拍賣行」單純以裝備之性能、實用性、耐久度做為指標，為單一武器進行測試、評分，以定義其價值（不考量使用條件及成本，但會在拍賣時額外說明）。

顧氏裝備評分：性能、實用性、耐久度各0~1000（分數越高，每一分的價值越大）

顧氏裝備等級

通製：未達精良的一切武器，皆為普通。

精品：製造品質足以勝過市面大多數裝備，通常價格極為昂貴。

特質：難以在普通店家看見的珍稀裝備，拍賣行主要的裝備拍品即為特質級。

傳奇：無法正常交易的強大裝備，唯有世上真正的強者能持有。

超格：常做為各大超級勢力的底牌，乃戰略級資源。

夢幻：其存在能造成世界轟動，扭轉蒼究局勢，價值無法估量。

神器：力量足以威脅神的裝備。
