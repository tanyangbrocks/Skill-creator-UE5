# 兩儀棋（不完全）

兩儀棋分陰面和陽面。

「陰面」輸入水、火、風、光、血，陽面輸出。

（對應法道、意道、仙道、詭道、武道）

「水、慧」

露水製造「慧水」，可以滋養神魂，提高識海力量，增加悟性。
後來發現它可以用來滋養契約靈獸，使其提早開智，力量變更強。也可以讓不具意識的事物開智。

「火、情」

創造一條條沒有實質的情緒絲，使人情緒變得更為極端，影響精神。（可用來對敵，只有空間類才能防禦）

熱火使人瘋狂，可將當下情緒放大，使人失去冷靜。寒火使人迅速鎮定，也可使敵人戰意全消。

「風、香」

棋接觸「風」會製造靈氣。可藉由輸入真氣來製造靈氣。用來修煉或快速恢復。

「光、時」（只能用自然之光）

半徑五米內時間加速最高十倍，可用來訓練or催熟植物。光輝籠罩範圍可轉化為持有者去過的環境。陽光、月光：五倍

「血、肉」

棋接觸「血」會創造血主人的肉身，分身狀態變化可以選擇傳給主人（會不斷吸血，而且吸很快）。
（可利用此法開啟五竅）

神獸的精血也可創造神獸的肉身、只要血持續供應就不死不滅，但消耗精力或被攻擊都會增加血的消耗。鮮血用盡了就消失。

📌在陽光、月光最盛時練法術、劍術，其它時間灌真氣修煉，真氣用完泡水、需要跑腿時泡血。隨時取血以備不時之需。
📌火只有幫助入定、穩定心魔，戰鬥時或急需鍛鍊意道才使用。
