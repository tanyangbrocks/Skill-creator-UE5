# 修煉境界（下境界）

修練境界形容（舊版）：

【通靈】：靈氣引為靈力，凡間至尊，修道始
【煉氣】：靈力凝為氣海，呼風喚雨，掌似仙
【築基】：氣海轉為靈晶，遨遊八荒，靈不朽
【金丹】：靈晶聚為金丹，毀天滅地，念通神
【元嬰】：金丹孕育元嬰，虛無飄渺，不可滅
【道極】：元嬰觸及天地，扭轉虛空，近乎道

修煉境界形容（新版）：

【通靈】：靈氣引為靈力，凡間至尊，修道始
【虛海】：靈力凝為氣海，呼風喚雨，掌似仙
【晶煉】：氣海轉為靈晶，遨遊八荒，靈不朽
【銘界】：靈晶演化天地，毀天滅地，念通神
【界源】：天地重鑄界身，虛無飄渺，不可滅
【道極】：界身融入本源，扭轉虛空，近乎道
