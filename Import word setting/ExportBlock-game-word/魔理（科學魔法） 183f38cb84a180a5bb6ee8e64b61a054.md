# 魔理（科學魔法）

純粹利用對魔力、元素、物理學、其它能量體系和特殊裝置的精密操作搭配，因而完成的複合魔法，廣義上統稱為魔理魔法。

魔理魔法變化極多，不一定需要編織術式或刻印迴路，但可能會有些自定義的發動條件。

魔理魔法之難度和危險性極高，通常只有精通魔力性質且知識淵博的魔法使才會嘗試，因此它較為少見。

但也因其稀少性，在戰鬥時對手難以看破自定義魔理魔法的效果。

魔理魔法舉例

武器破壞術

利用高速旋轉的魔力搭載電流，形成龐大高頻交流電磁場，破壞或移除對手金屬製武器。

---

魔理魔法寰界等級

魔理魔法原則上沒有明確的「寰界」等級定義，但經驗豐富的魔法使能就其複雜度、效果來大致推估其寰界等級。
