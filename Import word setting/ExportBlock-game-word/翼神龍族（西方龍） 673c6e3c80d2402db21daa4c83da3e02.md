# 翼神龍族（西方龍）

肉體極強，魔法、妖力、精神力也極強，壽命悠久，乃法體雙修最強種族，幾乎所有個體天生都能「掌控魔法」。

約有十分之一個體能使用靈力，同時具備魔法、靈力者能自然使用䵻龗。

翼神龍族大多數個體都能成神，是蒼究世界最強大的種族之一。

翼神龍族支派

五大支（翼神龍族主要支派種類）

綠龍：五大支排名末位，擅操控風、毒與植物。

冰龍：五大支排名並列三位，擅操控寒冰、海洋與氣候。

火龍：五大支排名並列三位，擅操控火焰與大地。

紫龍：五大支排名次位，擅操控雷電與空間。

金龍：五大支排名首位，擅操控重力與時間，喜用劍戰鬥。

雙王支（能輕鬆駕馭五大支擅長之領域）

黑龍：死亡、毀滅的象徵，通體漆黑。

白龍：生命、創造的象徵，通體潔白。

特殊支

妖魔龍：妖力強大的墮落支派，會四處獵食各族生靈來快速變得強大。

翼神龍族進化

龍蛋

龍蛋能以精神力感知外界，也可傳達意念給外界生靈，並利用天生的靈力、魔力來進行移動、攻擊。至少具備主將級戰力。

幼龍

翼神龍從孵化到成年間的狀態。這段時間由於實力相對弱小，易被異族捕殺，故大多留在族內修煉至成年。

此時若不修煉，至少也有耀日級戰力，但幼龍能修煉的手段極多，因此實力大多介於特異級到超王級之間。

成年龍

翼神龍通常能在300~500年間發育成年，實力至少封王級，修煉後多在超王級到超帝級之間。

巨龍

當翼神龍修煉足夠久，即能成為巨龍。巨龍的生命層次已達神之領域，還擁有操控法則之力。巨龍至少超帝級實力，大多為聖級。

龍王

成為龍王需經特殊儀式「加冕」，並繼承龍王之位，同時代只能同時存在一名龍王。龍王至少超聖級實力。

翼神龍族種族固有技

龍言

以言語的力量支配其它個體的行動，其效力遠強於「斷罪聖旨」，強者甚至能言出法隨。

龍威

龍的氣息，能夠震懾、壓制比自己弱小的血脈。使龍族在面對異族時基本能立於不敗之地。

幻化

翼神龍族能自由幻化為其它種族，但不一定能模仿其種族固有技。在當今時代，翼神龍族通常以人族的面貌現世。

龍爪技
瑞神龍族可藉由修煉來增加自己的龍爪數，有四到九爪不等。每根龍爪分別寄宿一種強大的個體固有技。

龍之吐息

凝聚體內能量釋放毀滅一切的致命射線。若以原始龍型態釋放，其威力可最大化。
