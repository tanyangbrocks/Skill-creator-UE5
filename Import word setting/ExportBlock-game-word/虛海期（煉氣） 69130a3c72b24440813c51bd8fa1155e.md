# 虛海期（煉氣）

達成煉氣前三步驟：

1.吐納：藉由吐納吸取靈力、調整身心

2.通靈：感知到靈力後，運轉之並嘗試凝聚氣海

3.入靈：成功引氣入魂、凝聚氣海

★通靈可以用靈力揍人，凝聚靈力球、靈力盾等，但到虛海才能用法術。

煉氣每層名稱：
一：靈璇

二：塑靈

三：養神

四：塵斷

五：通明

六：識地

七：識天

八：元生

九：元合

十：道門（始可築基）

十一：道臺

十二：道雲

十二層以上由修煉者自定義

一到十層除了凝練靈力，還要斬斷凡塵，修身養性，每個小突破皆有阻礙、心魔。

可分為一到十層，十層始可築基，十一層以上者，越往上越艱難，若氣海凝煉度不夠，或容納範圍不夠，可能爆體而亡。

（天才一般十二層，頂尖者十六層，可力抗築基。而主角抵達練氣三十六層。）
