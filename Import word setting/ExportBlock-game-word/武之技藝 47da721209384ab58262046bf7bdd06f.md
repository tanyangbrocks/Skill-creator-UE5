# 武之技藝

武具之技藝（武器技藝）

兵器類：劍、匕、刀、戟、槍（長槍）、棍、盾、鏢（暗器）、弓（弓箭）、鎚、鞭、投（投擲）

工具類：斧、鋤、鎬、鑽、剪、鋸、刷、扇、

器械類：槍、車、騎、機（機台）

操作之技藝（頭腦技藝）

理（演算）：高速邏輯運算推演能力，主要由支配體系修煉，部份學術、學習力強大者可能修煉。

悟（學習）：快速學習精熟的能力，各大領域的佼佼者都可能修煉成。

（不會戰鬥的純學霸也可能利用「理意」、「悟意」來構築「無能量武之主觀」）

-

引（驅動）：主要由精神力強大者修煉。

控（控制）：主要由精神力強大者修煉。

觀（觀察）：部份特殊職業修煉，如獵人、特務、情報機構等。
