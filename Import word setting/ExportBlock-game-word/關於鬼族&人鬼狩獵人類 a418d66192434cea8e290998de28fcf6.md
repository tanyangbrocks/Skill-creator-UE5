# 關於鬼族&人鬼狩獵人類

以下為鬼族視角……

鬼族的狩獵慾望

人族血肉對鬼有著極強吸引力，若看見人族在眼前，很難違抗強烈捕食本能。尤其吃過人族血肉後會上癮，其戒斷症狀更是令鬼族生不如死。

此外，人族的氣味對鬼有著誘惑、人族的聲音會使鬼興奮雀躍、人族的驅體和容貌被視為美的藝術品。可能會用人皮做為衣服、床、並以人骨做為器皿。

PS：當一名鬼擁有越多人類特徵，則代表其越美麗、越高貴、越偉大。

PS：食用人族是鬼族最快提升實力的方式，且它們在戰鬥中能藉由食用人族血肉來快速恢復傷勢與力量。

鬼族的狩獵指標

一、營養價值

修煉者：擁有靈力之修煉者對鬼族營養價值極高，吃掉一名可抵數十、乃至數百名普通人的營養，且靈力蘊養後的肉質異常鮮美。

魔法師：擁有魔力之人營養價值次之，吃掉一名可抵數名、乃至數十名普通人的營養，且魔力洗滌後的肉質異常鮮美。

奇異肉：約千分之一人天生體質為奇異肉，這對本人而言無任何影響。但對鬼族而言肉質極為鮮美，且能抵至少百名普通人的營養。

奇異肉散發的氣息容易被鬼族感知，並引來大量鬼族獵殺，且奇異肉者若有靈力或魔力，將產生更大效果。

人族稱此為「引鬼體質」，鬼族稱此為「奇異肉」。

高階血脈：高階類人體系血脈如瓦爾基麗、龍人、不滅族、古蘭族、阿修羅族等，肉質對鬼族而言極為鮮美，且能帶給其更強力量。（但捕獵難度極高）

此外，不滅族可為鬼族持續提供超高品質的血肉，地淵就有關押兩名不滅族孩童做為永續肉品。天官鬼老則是在胃裡持續消化一名不滅族男童—小耀。

二、肉質味道
環境&身份：如泰克諾爾民眾。該人民雖不具特殊力量，但其生活品質極為良好，肉質軟嫩較為鮮美。同理，鬼族通常優先獵食女子、孩童及年輕族群。

幸福者：心靈幸福之人族對鬼族而言肉質較為鮮美，還會散發特殊氣息。

因此哪怕是生活艱辛、皮糙肉厚的鄉野村民，只要心靈幸福，也易引來鬼族獵食。

才華與能力：才華與能力出眾之人，對鬼族而言其大腦與肉質更為鮮美，也會散發更甜美的香氣。

驚嚇、痛苦與恐懼：人族在驚嚇與恐懼狀態下會分泌腎上腺素與皮質醇，肉質會對鬼族而言更為鮮美，因此鬼族通常會在啃食人族前折磨、驚嚇之。對心靈脆弱的孩童效果更佳。

許多鬼族擁有特殊共感力，可以“品嘗”到對方的恐懼與意識，所以獵殺時非常講究獵物當前的狀態。鬼族會盡可能讓獵物感受到最大的痛苦和恐懼，直到最後一刻才取其性命，其中青鬼最擅長折磨。

身體部位：對鬼族而言，人族最美味的部位是臉蛋，其次是眼球和腦袋，再來是四肢，再來是內臟，最後是骨骼。但可能會因獵物狀態或個體喜好而有所變化。

個體年齡：對鬼族而言，人類第二性徵出現前的肉質最鮮嫩，出現後至性成熟前為次等；性成熟後，膠原蛋白開始大量交聯、流失，肉質為最次等。

三、個體偏好

普遍偏好：多數鬼族偏好獵食相貌較美、較可愛、顏值較高者，以及性格純真善良、心思細膩或膽小懦弱者，這將帶給其強烈征服慾。如長耳族等類人也因此容易遭受捕食。

獨特癖好：鬼族在獵食的當下，若能滿足其內心對獵食、獵物的獨特慾望，能額外得到更高的滿足感，以及更多的營養。

例如特別享受折磨獵物、驚嚇獵物、玩弄獵物情感，甚至花時間和獵物培養感情等，或是挑選自己情有獨鍾的獵物。一旦成功，獵食的當下越滿足，就能獲得越多營養。

四、多重要素

若有同時滿足諸多條件者，如嬌貴的魔法使、清秀白淨的修煉者、初入仙途或魔法的幼嫩學童、泰克諾爾的奇異肉則會被鬼族重點針對。

鬼族的獵食選擇

三者相較之下，崇尚武道且生活品質差的六龍聖天雖然民眾實力弱，但獵物品質低。屬於頂級獵物的修煉者數量極少，且往往結伴行動。

戴洛艾色頃向獵捕體質嬌弱的年輕魔法使，使用劍與鬥氣的精壯冒險者相較之下非其所好，但仍有極強捕獵慾望；

泰克諾爾人雖然肉嫩鮮美且居民毫無抵抗力，但鬼族難以突破其國界防禦，多為泰克諾爾之鬼影會成員暗中獵食。而西域邊際低開發國之居民不僅也肉嫩鮮美，捕食難度還大幅降低。

其它

異界召喚：部分強大鬼族有可能召喚異世界的人類（如地球人）來圈養、食用。主要勢力為地淵。

同類相食&異族獵食：低階鬼族普遍能吃到人類的機會並不多，因此多半會在極度飢餓時忍不住獵殺同類，也有可能獵殺其它物種。但這並不能從根本解決其飢餓問題。

外型越接近人類的同類或異族越容易遭因飢餓而失控的鬼族獵殺（但高階鬼對低階鬼有壓制氣息，不會成為其攻擊對象）。

當獸鬼吃不到人時，森林中的哥布林、獸人等種族也經常被鬼族獵殺。
