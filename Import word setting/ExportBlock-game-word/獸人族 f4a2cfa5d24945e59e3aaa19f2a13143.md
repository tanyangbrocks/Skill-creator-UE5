# 獸人族

有眾多分支，外形接近類人體系，文明發展度高。肉體力量強，擅使用真氣、靈力，少數可用妖力。

注重實力和名譽，亦為戰鬥民族。

獸人族常見分支

鋼爪支：狼獸人種、獅獸人種、虎獸人種、豹獸人種、熊獸人種、犬獸人種、貓獸人種。

★多分布於平原，戰鬥慾望最強烈，多半討厭使用兵器，喜歡以純肉身力量戰鬥，是將「狂獸化」發展到極致的分支。

★部份還會有嚴重的中二病。

鱗尾支：蛇獸人種、鱷獸人種、蜥獸人種、

★多分布於沼澤，喜用冷兵器作戰。

天翼支：鷹獸人種、雞獸人種、鴨獸人種、雀獸人種、鵝獸人種、孔雀獸人種、蝠獸人種、鷗獸人種、鴿獸人種。

★多分布於高山或森林，肉體相對薄弱，但對妖力的使用造詣極高。

影蹄支：牛獸人種、豬獸人種、馬獸人種、羊獸人種、鹿獸人種、駝獸人種。

★多分布於平原，戰鬥慾望相對低，智力較高。多從事士農工商等行業，常和類人體系接觸。

巨體支：象獸人種、犀獸人種、河馬獸人種、長頸獸人種。

堅牙支：兔獸人種、鼠獸人種、

比蒙獸人：

擁有比蒙血脈的獸人或類人體系，乃是最高貴的獸人族。

狂獸化

強大獸人經鍛鍊後可狂獸化，將短時間內大幅激發自身野性基因，急據提升某方面能力。如獅獸人的狂獅化、熊獸人的熊化等。狂獸化會消耗大量體力。

爆裂•極限•二段狂獸化！！

部份有天份且經過刻苦訓練的獸人，可進行更強大的二段狂獸化。

究極•超越•三段狂獸化！！！

少數天資卓越者可再次超越極限，完成三段狂獸化。

無敵•必殺•霹靂•滅世•超神•四段狂獸化！！！！

極少數存在可再再次超越極限，達成第四段的狂獸化變身。

目前沒有任何研究表示狂獸化的最高上限為何，甚至多數理論支持狂獸化沒有極限，故未來可能出現更高段的狂獸化。
