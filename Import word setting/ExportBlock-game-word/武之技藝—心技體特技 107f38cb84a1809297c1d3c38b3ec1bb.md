# 武之技藝—心技體特技

當部份種類武之技藝達到特定武之領悟時，可延伸出特別的「心技體特技」。

常見的心技體特技有以下種類：

心技體特技—心技體外放

當武器類、動態類技藝的武之領悟達到「法」後，可運用自身能量體系產生更強的「心技體外放」。

外放境界為：氣、罡、元

如劍氣、劍罡、劍元，指氣、指罡、指元等，皆可使用魔法、靈力、真氣等不同媒介發動。大多數武者因難以觸及意之領域，故會多加鑽研心技體外放來提升實力。

心技體特技—通覺

「意」覺醒「通覺」：能「聽到」顏色與味道，可以「看到」聲音與氣味，可以「品嘗」到對方的恐懼與意識。

心技體特技—謄印

達「悟意」者擁有快速精熟眼前所見技術之恐怖才華，又稱為「謄印」。
