# 靈界種族（特殊存在，全具魔）

[靈界世界觀概述](%E9%9D%88%E7%95%8C%E4%B8%96%E7%95%8C%E8%A7%80%E6%A6%82%E8%BF%B0%20b9003be77a7e40018f3700691ab49f08.md)

需找到契合度高的容器才可在凡間「受肉」而長久停留於凡間。或是當凡間聖靈使用附體術時，可暫時寄居於對象體內提供力量。

低階種族可藉由累積歷史、力量來使自己升階。

天使

光明與純潔之「魂」---天使

根源乃七美德的化身，每種「美德」乃職位的一種，其上各自依附強大權能，天使可相互爭奪。

[天使](%E5%A4%A9%E4%BD%BF%2073ac303f6c0c46ffbf1869e7e73f5b7e.md)

惡魔

黑暗與渾濁之「魂」---惡魔

根源乃七大罪的化身，每種「大罪」乃職位的一種，其上各自依附強大權能，惡魔可相互爭奪。

[惡魔](%E6%83%A1%E9%AD%94%205f00563b2c0548029e6b569dc5a39170.md)

精靈

自然與法則之「魂」---精靈

根源乃自然力量的化身，有十二大元素的支派（基礎、高階元素），每種「元素精靈王」乃職位的一種，其上各自依附強大權能，精靈可相互爭奪。

[精靈](%E7%B2%BE%E9%9D%88%20630a23b810ba45588039ca9c044345b9.md)

英靈

部分神話人物、歷史人物、故事中人物。

形成英靈原因極多，如本身即為「生靈意念」的虛構體、死後事蹟才被世人傳頌、價值達英靈以上成神未滿、價值足以成神卻選擇成為英靈等。

[英靈](%E8%8B%B1%E9%9D%88%2076cb5815f75d4dd39b656e84496ed974.md)
