# 《鬼族千相•赤鬼篇》

📌低階鬼族個體除了本身型態所固有之能力外，還會擁有自身獨特S鬼域。

[鬼域](%E9%AC%BC%E5%9F%9F%20a5f2afa1b6f8492b84d69e9816b7ec77.md)

📌此處所述為尚未經分裂生殖之原始型態，分裂生殖後能力會自然削弱。

以下紀錄常見型態，這些型態能力較為強勢，故可大量繁殖：

奇懼型（獸鬼）：

軀體似成年男性殭屍，皮膚赤紅，頭顱比例巨大，面部五官扭曲，除鬼族共通能力和專屬鬼域外，還具備怪力與高速移動之能力。

鎖定目標後，會幻化為對方內心深處最反感的人類模樣向其衝去。制服目標後，口中滿溢的續命唾液將塗滿目標全身，隨後會將身體和目標纏住，持續磨蹭、舔舐、啃咬、吸允、抽插，發出刺耳、淫蕩、令人不寒而慄的詭異怪笑，透過精神波動將自身的慾望、邪念與惡意露骨地傳達給目標，享受目標的驚恐、哭泣與反抗。強暴持續三到七日後，奇懼型會射出大量精液，精液會滲透進人類皮膚或七孔，一旦接觸精液，無論何種性別、年齡、狀態之人類皆會懷孕。

奇懼型多分佈於平原、洞穴、叢林，並在有人類氣息的區域徘徊。

奇懼型對目標產生之性慾程度標準約和其它鬼對獵物的鮮美度相似，若同時有大量目標聚集，奇懼型產生的性慾會疊加，部份極品目標甚至能讓奇懼型在見到的瞬間射出大量精液濺滿全身。

最常見的赤鬼，數量約占赤鬼總數的百分之四十，相對實力不高。

巨淫型（獸鬼～微智鬼）：

軀體似兩到三倍放大的人型，頭骨不規則畸形，腐爛的臉部長著三顆詭異眼珠，巨大豬鼻，嘴巴裂到耳邊，嘴唇極厚，內有4根分岔舌頭。皮膚赤紅且如水泥般粗硬，全身毛孔有大量白色蛆蟲蠕動，鑽進鑽出。身型健壯，有四隻手臂，可能會將人類的殘破衣物穿戴在身上，或將新鮮的人類殘肢縫在自己身上，一根能高強度縮放的生殖器在胯下擺盪。

擁有怪力、超速再生，智力接近青鬼，當前被勾起的性慾越高，力量、速度、防禦越高，魔力、靈力抗性能提升至最高70%，若因性慾臨時進化到微智鬼，必能暫時獲得「中階能力」。

其五感靈敏，能透過氣味獲得大量資訊，從遠方鎖定目標氣息後會立即前往，生殖器會膨脹至最多半米。制服目標後，口中滿溢的續命唾液將塗滿目標全身，隨後會將身體和目標纏住，持續磨蹭、舔舐、啃咬、吸允、抽插，發出刺耳、淫蕩、令人不寒而慄的詭異怪笑，透過精神波動將自身的慾望、邪念與惡意露骨地傳達給目標，享受目標的驚恐、哭泣與反抗，其續命唾液的強化敏感、增加痛苦恐懼、修復體力傷勢的效果也會隨性慾提升而增強。強暴持續三到七日後，巨淫型會射出大量精液，精液會滲透進人類皮膚或七孔，一旦接觸精液，無論何種性別、年齡、狀態之人類皆會懷孕。

巨淫型多分佈於平原及叢林，會在留有人類氣息的區域徘徊，或直接騎上體積較大的紫鬼，或攀附於人面蟲鬼上。

巨淫型對目標產生之性慾程度標準約和其它鬼對獵物的鮮美度相似，若同時有大量目標聚集，巨淫型產生的性慾會疊加，部份極品目標甚至能讓巨淫型在見到的瞬間射出大量精液濺滿全身。

多數生產後的目標會直接成為巨淫型食物，部份巨淫型會捨不得吃掉極品目標，可將其存放於生殖器空間，隨時放出來盡情縱慾，而目標也會在續命唾液的效果下持續痛苦驚恐，不會輕易死亡，體力不會枯竭，精神更不會麻木，每次都像第一次。

為相當常見的赤鬼，數量約佔所有赤鬼的百分之二十，也是人們最不想遇見的低階鬼族。隨著性慾提升，實力最強能達到微智鬼層級，群星級以上強者才可不畏懼之，群星級以下只能祈禱自己對巨淫型的吸引力較低，或事先準備專門降低巨淫型性慾的藥物。

PS：性慾標準和鮮美度指標相似

觸型（獸鬼）：

軀體由一攤紅色黏液所組成，透過伸縮、蠕動、分散、聚合來移動。黏液中會生成許多細小的人類嘴和舌頭，感知到美食會分泌續命唾液。黏液本身有敏感的神經，受刺激後會長出許多人類男性生殖器，噴出大量白色精液。

感知能力敏銳，能夠縮入細小縫隙藏於各種物品，巧巧入侵人類居住地。鎖定獵物後噴射出，包裹其身，利用全身小嘴盡情舔舐、啃咬、磨蹭，使其身上沾滿唾液，同時會發出令人不寒而慄的享受呻吟聲，透過精神波動將自身的慾望、邪念與惡意露骨地傳達給獵物。獵物無論再怎麼驚懼、痛苦，都無法擺脫，衣服會被溶解，身上的孔洞和傷口也會被觸型趁機入侵，觸型也會開始啃食、強暴其內部血肉和臟器。最終觸型將長出大量陰莖噴灑精液，待對象無法再生育後將其吃得一乾二淨。

本身戰力不高，但超界者被偷襲、包覆後也難以應對。
