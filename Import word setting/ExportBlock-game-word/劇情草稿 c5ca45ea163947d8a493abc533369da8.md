# 劇情草稿

主角原是扒手，被有技能系統的罪犯綁架（他綁架一群人），罪犯抓各種人來獲取技能，並強化自己的「殺戮者」。
主角臨死之繼，盜賊系統誕生。身旁一名高中生（萬能升級擁有者）阻止了罪犯，與其戰鬥。
盜賊偵測到兩個最上位系統。
兩人兩敗俱傷，擁有軍火的罪犯略勝一籌；主角依照盜賊系統指示前去掠奪高中生之系統。
主角當時無法殺了高中生，對方身體太強，打不進去。
碰高中生肩膀，騙他說你的萬能升級系統有隱藏系統，跟我一起喊：「盜賊系統，開！」
經歷了對方的生活後，得到了系統，結果因為對方身體也是系統的一部份而變成對方的樣子。
用系統的力量殺掉了殺戮者使用過度而體力不支的罪犯，獲得無限技能系統和他累積的能力。
高中生雖然沒死，但失去了系統力量而整個人外貌變醜，變平凡。
他感謝主角救命，但又想報復主角，許下的願望是：「請你用我的身份過生活，而你必須聽從我的一切指令！」
「我是個自由的小偷......」
啪刷！主角砍了高中生的頭。
「傻孩子，你應該趕緊命令我不能傷害你才對！」
你是個可憐的好孩子，但我不會放過任何威脅到我的人。抱歉啊，叔叔不是什麼好人......
如果主角能一次獲得所有最上位系統，甚至能讓地球躍升為高等文明，開發並統治數十個星系，並反咬其它界域一口

主角可以嘗試圈養他人的系統，把它們關進異世界，讓他們變得強大後再奪取其能力。

（進學校，發現自己高中同學回來當這高中生的班導師）
高中生轉學後，用另一個身份轉進來
他的摯友認出了一切，後來也發現
摯友有一個能從對方記憶中複製出對方人格的能力，透過對視、觸摸......（也）
盜賊頓時感應到對方同等系統的能力（原本感應不到存在）
他的系統感知到對方感知到後拿刀刺入主角心臟，主角也抓過去，對方同時跳開
複製分身能和本體共享訊息
