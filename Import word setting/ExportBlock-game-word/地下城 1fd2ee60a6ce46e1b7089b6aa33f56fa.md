# 地下城

形成原理—地下城化

群居性異獸或異族的領地，通常會有一個或數個統領的Boss，並存在許多珍稀資源。

當同質性高且意志統一的大量異族或異獸的魔力、靈力或妖力等能量體系開始改變周遭環境法則，領地本身也將「活」過來，成為名為「地下城」的特殊空間，這就是「地下城化」。

★擁有能量體系的人類也群體也能產生地下城。但人類很難意志統一，只有少數宗教能依靠神聖力來真正產生地下城。而宗教人士會將其稱為「信仰聖城」。

地下城功能

地下城會使領地能量充沛，發展更加迅速，成員也能迅速變強。

而該勢力的統領boss將擁有掌控此空間的權柄，能自由設置陷阱，轉移成員等。

常見如哥布林洞穴、毒蜘蛛窟、龍巢等都歸類為地下城。
