# 仙道（靈玉）

使用靈力，需具備靈玉。

仙道象徵物：靈玉

[靈玉](%E9%9D%88%E7%8E%89%200f1d4cb5888e414fa7d740d8544c8b43.md)

---

靈力

又稱生命能量，能夠提高「魂」之層次，甚至追尋「法則之力」。世間所有生靈體內都具有靈力，但唯有具備「靈玉」者才能掌控並修煉之。

靈力運行方式

靈力修為平時儲存在「魂」內，只有在修煉或欲使用靈力時才會立刻進入「丹田」，並按特定路線在體內運轉。

靈力運轉路線視所修功法而定，功法練成後將在體內開闢出「靈力經脈」，靈力經脈並非實質（乃魂的外延現象），故其路徑不受骨骼、肌肉所阻擋。

PS：倘若魂的靈力量超過丹田、經脈，就可能承受不住靈力壓迫，爆體而亡。

修煉仙道

修煉仙道能夠昇華生命層次（命階），滋養壽元、肉體與精神，還能驅動靈力發動各種超乎想像的術法神通，而能夠進行修煉之人即為修煉者。

傳聞百人中只有一人能感應到靈氣(有靈玉)，而能夠感應靈氣者僅有不到百分之一能做到引氣入魂(靈玉品質夠高)，進行修煉。因此萬中無一的修煉者是六龍聖天帝國最崇高，也最珍貴稀缺的人物。

修煉除了增強實力，還有讓生命層次提升而延年益壽，以及養顏美容、通明神智等效果。

修煉功法種類

主功法：用於開闢經脈，有些會附帶特殊靈力性質或仙術神通，主功法只能同時修煉一種。轉修前需先「散功」，使現有靈力經脈消失。（散功不會使魂內的靈力修為倒退）

元嬰主功法：元嬰後，可讓元嬰修煉第二個主功法。但由於第二主功法也同樣會開闢靈力經脈，影響修煉效率、改造靈力性質，需注意它是否能與現有主功法相容。

副功法：不開闢靈力經脈，而是用於單純積累靈力、改變靈力性質、修煉新術法神通等輔助用途。副功法沒有修煉數量限制，但通常修煉副功法需花費不少時間。

心法：透過冥想或其它方式，修煉某種特殊技能或天賦。通常不直接影響修為和靈力。

修煉功法類型

功法按其功能導向又區分為多種類型。

修煉型：主要能提高修煉效率。

神通型：主要能習得強大神通。

靈改型：主要能改造靈力性質。

煉神型：主要能強化精神力。

煉體型：主要能強化體魄。

中庸型：各層面都可加強，但都不突出。

靈力三大本源性質：曉、霸、極

曉：靈力輕盈且靈動，較能精細操作並適應各種環境與術法。常見於術法神通與法道修士。

霸：靈力霸道且厚實，蘊含強大氣勢與破壞力。常見於劍修、體修等需要硬拼力量的修士。

極：靈力隨使用者意志走向極端，自成一套特殊運作形式，通常是受「意道」影響發生靈力變異所致。

常見仙道修煉方向

劍修：以劍為主要攻伐手段，能以較少的精神力、靈力來發動強大攻擊，或佈置劍陣，實力強勁。

但由於需要購買較多精良的劍器，且劍器日常保養昂貴，再加上劍修需花費大量時間練劍、感悟，故劍修多半較為貧窮。有時劍修會成為其它修士專門的打手。

道修：以功法神通做為主要攻伐手段，以修煉功法為核心，一心求道，穩扎穩打。

整體實力取決於功法性質，若為「修煉型」功法，則戰力低微；「神通型」功法較能具備強大實力，但其手段單一，易被克制。

法修：修習多種千變萬化的術法神通做為主要攻伐手段，戰鬥能創造層出不窮的變化，缺點是靈力消耗較大。

另外，法修需花大量時間修煉術法，通常精神力和肉體都較弱，相對不善於近戰。

體修：以鍛鍊肉體做為主要攻伐手段，一力破萬法。缺點是手段單一，且通常需近身作戰。

[煉體（武道煉體&仙道煉體）](%E7%85%89%E9%AB%94%EF%BC%88%E6%AD%A6%E9%81%93%E7%85%89%E9%AB%94&%E4%BB%99%E9%81%93%E7%85%89%E9%AB%94%EF%BC%89%20dcab40563eb147bbb61fc7725c94d076.md)

邪修：又稱魔修，為達極高修煉效率或超越同境的強大實力，採用非正規、非人道的方式修煉或煉製物品。

邪修功法通常會影響心性，易走火入魔，還會被諸多正道修煉者討伐。

陣修：以陣法做為主要攻伐手段，若能將敵人困於陣法，或使用克制對手的奇陣，能發揮遠超本人境界的戰力。

但佈陣往往需要時間，且佈陣動靜明顯，易被敵方阻撓。即便專精陣法，也很難在戰鬥中神不知鬼不覺地佈陣，這通常需搭配縝密的戰術或隊友掩護。

御獸：御使靈獸作為主要攻伐手段，若一次放出許多靈獸能壓制同境。缺點是培養靈獸花費時間、金錢、精力極高，自身靈獸還可能戰死。此外，御獸師本身通常是最大弱點，靈獸還需保護主人不被偷襲。

御寶：驅使各種法寶做為主要攻伐手段，憑藉特殊法寶機制掌控戰局。但驅使法寶所需精神力消耗較大，法寶也容易在戰鬥中損毀，且持有強大法寶者可能被人盯上。

混修：沒有特定流派。有劍用劍，學到法術用法術，撿到法寶用法寶，通常是散修或經常得到大機緣者會成為混修。混修的好處是手段多樣，能打得敵人措手不及，但需花費心力精修各種領域。
