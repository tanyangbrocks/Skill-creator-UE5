# 界源期（元嬰）

元嬰過程

1.碎丹

2.心魔

3.孕育元嬰

要從金丹孕育出元嬰有三種方式，而這三種方式也分別對應至三條修煉之路：

1.本源道嬰—天道之路

嘗試以自身道種直接突破元嬰！
成功孕育本源道嬰者其心念與天道共鳴，實力極為強悍！
然而此法必然引動天罰劫雷，自古渡劫成功率極低。一旦失敗，則肉身消亡、魂飛魄散，因此鮮少有修煉者採用此法。

[本源道嬰修煉之路](%E6%9C%AC%E6%BA%90%E9%81%93%E5%AC%B0%E4%BF%AE%E7%85%89%E4%B9%8B%E8%B7%AF%2096307594393c40598f927d5d801c1055.md)

2.伴生奇嬰—外道之路

吸收伴生靈物破境，通常選擇道行千年以上之妖皇魂。
妖族為人族伴生種，對人族而言極為滋補，妖皇以上魂魄更能刺激金丹孕育元嬰。伴生奇嬰實力受吸收靈物強度影響，若是道行超過萬年之妖聖魂，元嬰後實力甚至可匹敵本源道嬰！
由於此法成功率相對較高，也是當前最多修煉者採用的元嬰之道！

[伴生奇嬰修煉之路](%E4%BC%B4%E7%94%9F%E5%A5%87%E5%AC%B0%E4%BF%AE%E7%85%89%E4%B9%8B%E8%B7%AF%204c5491688b5946a9ba2c63c7498e7395.md)

3.魂火邪嬰—魔道之路

屠殺數以萬計的人類，焚燒其冥蝶取得魂火，並於金丹上銘刻魂火紋。然而魂火紋銘刻成功率低，踏上此道者必須持續殺戮直到魂火紋銘刻成功！
此道實屬極端殘忍、喪盡天良，被正派修士及皇室、天輝山等勢力明文禁止。但由於此道元嬰成功率極高，元嬰後實力普遍等同伴生奇嬰，選擇此道者亦不少！

[魂火邪嬰修煉之路](%E9%AD%82%E7%81%AB%E9%82%AA%E5%AC%B0%E4%BF%AE%E7%85%89%E4%B9%8B%E8%B7%AF%20f75df130660a43609e723c15f9e0b572.md)

特殊元嬰：

主角隨著先前的路走。五行嬰、魂火嬰、伴生嬰、意嬰、真元嬰都要達成。
主角接受天宮之仙道本源之力元嬰。

元嬰修煉

證源期（育嬰期）

遵循結嬰時所選之三條修煉之路，修到極致，即可踏入嬰變期。

界身期（嬰變期）
三條育嬰期圓滿後皆會進入「嬰變期」，嬰變期可自行選擇適合自己的金身，只要將一種金身修煉至圓滿，即可度過嬰變期。

★嬰變期五大特殊界身

靈動身：身體感知、直覺超大幅強化，識海強化數倍。

金剛身：肉體強度超強化，誕生特殊體質。

化羽身：突破肉體枷鎖，使肉身接近靈體，可虛實轉換。

不死身：可滴血重生的肉體恢復力，壽命大幅延長。

通天身：誕生本命神通，且使用神通的資質提高。

一般而言一人只能擁有一種特殊金身，不同金身會互相排斥。但道體沒有金身排斥現象，五種金身皆取造就「五道聖體」。
