# 瑞神龍族（東方龍）

肉體極強，靈力、真氣、神聖力、精神力也極強，壽命悠久，乃仙體雙修最強種族。雖無法使用魔法，但其仙道靈力極度強大，可輕易支配法則。

瑞神龍族大多數個體都能成神，是蒼究世界最強大的種族之一。

瑞神龍族進化

龍蛋

龍蛋能以精神力感知外界，也可傳達意念給外界生靈，並利用天生的靈力、魔力來進行移動、攻擊。至少具備主將級戰力。

幼龍

瑞神龍從孵化到成年間的狀態。這段時間由於實力相對弱小，易被異族捕殺，故大多留在族內修煉至成年。

此時若不修煉，至少也有耀日級戰力，但幼龍能修煉的手段極多，因此實力大多介於特異級到超王級之間。

成年龍

瑞神龍通常能在300~500年間發育成年，實力至少封王級，修煉後多在超王級到超帝級之間。

巨龍

當瑞神龍修煉足夠久，即能成為巨龍。巨龍的生命層次已達神之領域，還擁有操控法則之力。巨龍至少超帝級實力，大多為聖級。

龍王

成為龍王需經特殊儀式「加冕」，並繼承龍王之位，同時代只能同時存在一名龍王。龍王至少超聖級實力。

瑞神龍族種族固有技

龍言

以言語的力量支配其它個體的行動，其效力遠強於「斷罪聖旨」，強者甚至能言出法隨。

龍威

龍的氣息，能夠震懾、壓制比自己弱小的血脈。使瑞神龍族在面對異族時基本能立於不敗之地。

幻化

瑞神龍族能自由幻化為其它種族，但不一定能模仿其種族固有技。在當今時代，瑞神龍族通常以人族的面貌現世。

龍爪技

瑞神龍族可藉由修煉來增加自己的龍爪數，有四到九爪不等。每根龍爪分別寄宿一種強大的個體固有技。

祥瑞

瑞神龍族自帶祥瑞氣運，不僅運氣及修煉效果極佳，也很難遇上災禍。

龍之信仰

瑞神龍族自帶強大神聖力，可供人民信仰，並回饋龐大氣運。
