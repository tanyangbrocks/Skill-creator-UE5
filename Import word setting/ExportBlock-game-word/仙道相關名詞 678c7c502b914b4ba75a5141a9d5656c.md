# 仙道相關名詞

靈香：每個生靈的靈力都有其獨特的氣息，熟悉彼此者通常可迅速辨認對方靈香。有些修士會刻意改造、培養自己的靈香，塑造個人魅力。部份靈香甚至能蘊含特殊效果，提供戰力加成。

[魔香&靈香](%E9%AD%94%E9%A6%99&%E9%9D%88%E9%A6%99%20281f38cb84a18066976dfdea34526f04.md)

仙氣&靈光異象：許多資質不凡的修士體表會散發氤氳之氣，或微微散發光暈，給人聖潔、朦朧之感；部份特殊體質則有祥雲、瑞獸、金光、經文流轉等更浮誇的體表異象。

★雖然異象能用術法模擬，但很容易被同為修士者識破。此外，先天自帶的異象也能憑個人意志開關。

靈威：釋放靈力的威壓，強者釋放靈力時可利用靈威導致弱者恐懼、昏厥，甚至當場暴斃。

吐納：修煉者進入冥想狀態後，透過特殊呼吸方式吸收天地靈氣，進而積累靈力修為，通常要搭配功法以大幅提升效率。吐納為最主要修煉方式之一。

★「吐納」包含吸收靈氣與冥想，與武道的「呼吸法」是截然不同的事物。

辟穀：修煉者達到築基期後，可透過修煉替代進食、排泄、睡眠等生理需求。辟榖後也將免疫一切凡俗疾病。

閉關：修煉者為專注積累自身修為，經常與世隔絕，避免外界干擾。此時將能進入深度冥想，且一閉關通常就是許多年。

傳功：修士將體內部份靈力轉移給另一人，以使其修為快速晉升。但此舉容易讓快速晉升者根基、心境不穩，最終自取滅亡。

走火入魔：修煉時可能因分心、修煉方式錯誤、遭遇心魔等情形導致走火入魔。走火入魔若不及時妥善處理，輕則修為倒退、經脈受損；重則爆體而亡、神魂俱滅。

[心魔](%E5%BF%83%E9%AD%94%20279f38cb84a180df932cda8a02689699.md)

法號：強大修煉者通常會自立「法號」以堅定道心，並提高識別度。雖然沒有明訂，但通常需達金丹期才能立法號，否則將遭眾人恥笑。

三花聚頂，五氣朝元：修煉功法大成之異象。每次經歷此過程，都可洗經易髓，修煉者天資越高，異象越明顯。

護道人：部份大勢力為保護自家天驕，會暗中派高手時刻隱匿於其身旁，避免其受到突襲暗殺，或在遊歷時意外身亡。

但為訓練天驕自保能力和危機處理能力，護道人不會隨意出手，甚至會對天驕隱瞞自身存在。

靈魔互斥定律：由於魔力是極端的「反法則能量」，它與通往法則、試圖感悟法則的靈力相互排斥。故靈力、魔力通常無法結合，也不太可能同時存在於單一個體。

靈脈：靈力濃郁的天然區域，能夠大幅加快修煉速度，部份靈脈之靈力會蘊含特殊屬性。

心境：修士的心境與修煉速度密不可分。如果心境低於當前境界，則事倍功半，反之亦然。倘若心境遠低於當前境界，或進境速度太快，可能直接誘發遠強於自己的心魔誕生。
