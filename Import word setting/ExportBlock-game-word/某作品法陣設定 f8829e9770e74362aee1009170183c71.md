# 某作品法陣設定

—<魅魔，不，是食人魔>

魔法阵角的数量，代表了对魔力节点的运用。
通常，在施展成功的前提下，数量越多，效果越强。
理论上来说，效果最强的魔法阵，应该是纯粹的圣圆——代表无数个角集合。

最基础的魔法阵，一般是从三个角开始的，魔法阵学是所有法师需要掌握的学说，也是魔法之基础。
这三个角分别代表了魔法的最初性质。
储蓄（注入）、稳定、释放。
而典型的四角魔法——爆裂魔法，则只需要在其中加入一个‘压缩’就行。
这些角边并不能胡乱添加，不然完全可能会出现相互抵消的现象，那样施展的魔法，实际生效的部分就少的可怜了。

通常来说，复数式的魔法阵会更加稳定一点。
而一个召唤魔法阵，至少需要六个角。
这六个角分别是——
储蓄、稳定、转换、定位、通道、链接。
元素位面的召唤阵。
通常一般还会再加两个角——元素、塑造。
当然，也有法师为了加强控制，将用于沟通的‘链接’换成‘奴役’的；只要你的精神力足够强大，那就不需要沟通。

欧格拿起羽毛笔，徒手在书纸上画出了一个完美的七芒星，以此证明自己。
魔法阵的每一个角距离，最好趋于平等。
不然单方面过强，也可能会造就整个魔法阵的崩坏。
七芒魔法阵并不受欢迎的原因，很大程度上是因为角节点的间距不太好把握。
相比之下三、四、五、六、八，就受欢迎的多了。
学习成本也小。
其中最受欢迎的还是伪六角与伪八角（两个三角形叠加与两个四角形叠加）。
这种伪叠加式魔法阵可以旋转叠加的三角或四角进行魔法性质变化。
比如——重叠两份爆裂魔法，就可以完成连环爆裂，或是大爆裂魔法，双重爆裂魔法等。
“天才的想法！这么看来，艾尔老师用的是沟通召唤阵咯？”罗奥一番夸奖后，再度询问了相关细节。

召唤系的魔法阵通常分为三大类。

奴役、沟通、献祭。
最常用的是前两种，第三种出现的情况通常是魔力或精神力不足的法师使用的。
