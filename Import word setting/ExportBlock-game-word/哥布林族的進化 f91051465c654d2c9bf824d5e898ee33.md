# 哥布林族的進化

哥布林族極為容易進化，會受「精熟業力價值」影響而進化出不同職階；也會受「捕食業力價值」影響而繼續進化，且由於時常和異族交配，故血脈複雜。

PS：「精熟業力價值」乃「持續進行某行動直至精熟」的價值，部份種族以此為進化依據。

[特殊價值](%E7%89%B9%E6%AE%8A%E5%83%B9%E5%80%BC%204fdecaf0814a4de483af5e3e55bb2bd1.md)

哥布林幼崽

剛出生至一個月的哥布林，已有相當攻擊性。

成年哥布林（個體超凡級水準）

哥布林出生一個月即成年，體型約等同成年女性，但力量遠大於成年男性。

大哥布林（個體征戰級水準）

少數天資卓越的哥布林能自然成長為「大哥布林」，只有「大哥布林」可進化至二階以上。哥布林與類人體系交配之後代有高機率成為大哥布林。

常見一階進化哥布林（個體征戰級~主將級水準）

哥布林盜賊：能靈活使用短刀與盾，身手敏捷之哥布林。

哥布林薩滿：能使用簡單妖術之哥布林。

哥布林射手：能使用弓箭，有強大洞察力之哥布林。

常見二階進化哥布林（個體群星級~耀日級水準）

哥布林大魔將：肉體力量強大，部份甚至能使用妖力、真氣，普遍使用刀劍武器。

哥布林領主：智慧與統帥力極強，擁有強大精神力，戰鬥力也極為不俗。

哥布林大祭司：哥布林薩滿上位種，妖力極強，通常還能驅使其它較弱哥布林。

常見三階進化哥布林（個體耀日級~巔峰級水準）

哥布林王：哥布林領主上位種，能統領、鼓舞其它哥布林。自身戰力也極強，智力更超越普通人類。

哥布林勇者：能在進化瞬間獲得強力追加技能，每個體所獲之技能不一。同時速度極快，並擁有超速再生能力。

哥布林聖騎士：攻擊力強大，能使用強大神聖力與真氣，體表凝聚堅硬鎧甲，幾乎不可撼動。

-

特殊：哥布林修士

約有百萬分之一的哥布林能成為修煉者，這類存在通常會脫離族群生活，但不一定會改掉其貪婪、淫亂的本性。
