# 概念

賦予存在意義之力，賦予&否定、關鍵詞、誓約、贈予等。通常由「次元型家族」所掌控，少數低界面個體能使用。由於該力量能直接影響法則，故受到神界重視。

賦予概念

賦予或強化某事物其原先不具備之概念，如：賦予活體「死亡」的概念，將導致對方死亡。

否定概念

否定或弱化某事物其原先具備之概念，如：否定亡者「死亡」的概念，將導致對方復活。

📌同時能「賦予」或「否定」概念者，被至高神壇視為能完美改寫現實，將被其重度關切。

---

關鍵詞

掌握住某法則對自身的影響，將自身狀態與該法則綁定。其使用效果之上下限極高。

如：每「背叛」一次，實力大幅增強、能感知到所有「不正義」、每次「下廚」就能提升顏值等。

誓約

利用「概念」之力對自身或某對象（對方需同意）發動「誓約」。

誓約內容包含「規則」、「效果」及「懲罰」。規則越嚴格、懲罰越重，單位「概念」所能產生的效果越強大。

已經屬於有條件的「心想事成」。（類似武之主觀的形塑）

贈予

贈予其它生靈操縱概念的能力，通常為高階概念能力者所擁有的權能。

概念強度

當兩種被執行的概念相互矛盾時，會進行概念強度比較。若概念強度完全相等，則兩種被執行矛盾的概念相互抵消。
