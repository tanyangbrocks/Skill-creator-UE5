# 附體術

原為影族種族天賦，後被拉普拉斯解析並將使用方式編入書籍。

利用魔力連結英靈殿，號召有緣之英靈，為較偏門之應用。

會附體術者極少，因為需要較高天賦，且具有極高風險，很容易被英靈反噬，甚至奪舍。

附體術效果

附體可暫時對象繼承部份能力值、技能，產生靈體形象虛影。

附體徵召對象權力

1.鬼物或英靈可拒絕他人的「附體徵召」要求，但通常會同意。

2.附體徵召對象隨時可憑自身意願脫身，也能決定自己要給予對方多少力量。

附體術限制

1.附體術使用者實力不可比「附體徵召對象」低太多，但若附體對象是自身前世，皆可附體。

2.附體術使用者可事先與徵召對象簽約，以便日後更順利發動附體術。

---

附體徵召對象類型

鬼物：凡間或冥界怨念深重的死靈。

等級分為整、亂、奇、域、淵。

英靈：部分神話人物、歷史人物、故事中人物、契約玩家。

等級分為軼聞、響名、上古、傳說、神話。
