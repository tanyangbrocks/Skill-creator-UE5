# 信徒種類

人們信仰宗教後，進行不同程序或達到特定信仰程度後，所獲得之效果和反饋之價值亦有不同。

[宗教信仰](%E5%AE%97%E6%95%99%E4%BF%A1%E4%BB%B0%2040ae611d6e1140ecbaa8ed5044ab5fea.md)

泛信徒

在心中相信該神明存在，並敬畏其事蹟、偉力。但若遭遇苦難，或其它神明能給予其更大的恩惠，隨時會拋棄信仰。

★不需進行入教程序，亦不會得到神聖力。

★會反饋微量價值給神明，但並不穩定。

正信徒

泛信徒進行過入教程序（除非該信仰無入教程序），熟悉神諭或教義，並盡可能不觸犯戒律。將會成為正信徒。

除非生活或利益被外物嚴重影響，否則不輕易拋棄信仰。

★能得到少量神聖力，相當於褐袍/銅甲。

★會穩定反饋一定量價值給神明。

★每隔一段時間，低機率孕育「信仰之穀」給神明，神明可將信仰之穀種植於某塊信徒分佈地，使該區域人民反饋的信仰之力提升。

虔信徒

正信徒嚴格遵守神諭、教義、戒律，將信仰視作生活重心，將自我與信仰連接在一起。背棄信仰相當於拋棄自我，基本不可能發生。

★能得到大量神聖力，為灰袍/銀甲以上階級，各大教會人員多半為虔信徒。

★能持續反饋大量價值給神明，部分能和神明對話接取神諭，或成為神明代行者。

★每隔一段時間，高機率孕育「信仰之穀」給神明，神明可將信仰之穀種植於某塊信徒分佈地，使該區域人民反饋的信仰之力提升。

★每隔一段時間，低機率孕育「信仰之弦」給神明，此為「神聖祭壇」、「真神像」等施展神蹟媒介物的主要材料。

狂信徒

將信仰視為世界真理，甚至超越自身生命和一切事物。眼裡只有神，以及對其無盡的癡迷，可能會為了信仰做出瘋狂的舉動。除非遭受精神控制、洗腦，否則不可能拋棄信仰。若察覺自己可能會背棄心中的神，將會直接自我了結。

★虔信徒成為狂信徒並不會直接提高神聖力，但神明可主動灌注該信徒軀體所能承受極限之神聖力。

★能持續反饋極大量價值給神明，神明可直接操控其意識、軀體，甚至獻祭狂信徒的靈魂、肉身來進行儀式。

★狂信徒的存在可能使世界混亂，故神明們相互約定不會培養狂信徒，並會介入處理意外產生的狂信徒。但部份魔神教派可能暗中大量製造狂信徒。

★不會孕育「信仰之穀」或「信仰之弦」。
