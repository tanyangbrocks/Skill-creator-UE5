# 自然資源

🔴生物資源

石油

多由「混亂時期」大量戰死的生靈屍體經高溫高壓變質而來，可用於燃燒做為能源、發電等。但在源能興起後逐漸被取代。

植物

等級：普通、稀世、靈植、奇珍、天精。

★「普通」又分糧食作物和經濟作物；「稀世」以上通常具備魔力、靈力等特殊能量。

★另有《曠世本草榜》為各種植物做註解、分類，並為其力量及效果排序。
