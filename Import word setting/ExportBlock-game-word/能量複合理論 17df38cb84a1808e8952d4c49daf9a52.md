# 能量複合理論

蒼究的生靈力量主要皆由「肉體」、「靈魂」與「外能」所組成。

修煉體系主要精進肉體與靈魂，支配體系主要精進外能。

同源複合

將單獨內在或外在的力量挖掘到極致，較為失衡。

[十五種支配同源複合體系：](%E5%8D%81%E4%BA%94%E7%A8%AE%E6%94%AF%E9%85%8D%E5%90%8C%E6%BA%90%E8%A4%87%E5%90%88%E9%AB%94%E7%B3%BB%EF%BC%9A%2072ad194d9be7496e84790b70a032288e.md)

異源複合

能夠將內外之力交融，較為均衡。

[二十九種異源複合體系（劇情不全出現）：](%E4%BA%8C%E5%8D%81%E4%B9%9D%E7%A8%AE%E7%95%B0%E6%BA%90%E8%A4%87%E5%90%88%E9%AB%94%E7%B3%BB%EF%BC%88%E5%8A%87%E6%83%85%E4%B8%8D%E5%85%A8%E5%87%BA%E7%8F%BE%EF%BC%89%EF%BC%9A%2006443e8352cb4754bf2bed159e998e75.md)

---

因此，若排除掉能量層次、元素、法則、武之心得等要素來看，異源複合能量的效果對生靈而言更均衡、更強大。
