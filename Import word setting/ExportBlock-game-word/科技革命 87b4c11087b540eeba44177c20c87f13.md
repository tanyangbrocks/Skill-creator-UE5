# 科技革命

源能被發現，人類科技能源有重大突破。

機械族的內部構造強行讓人類科技水準提升好幾個世紀，

-

許多擁有高度智慧與能力，卻因缺乏「魔法」、「修煉」等才能的知識份子開始發起「科技革命」。

以。。。為首之人欲推翻當前以武為尊的社會體制，認為當前人類實力已足夠強大，沒必要全民皆兵。不僅該適性揚才始得強盛，更應致力於追求更高的生活品質。

況且，以科技實力發展出的槍、炮等熱武器可碾壓低級冒險者

公會槍擊事件

一支極端革命軍部隊手持步槍掃射冒險者公會，當下諸多冒險者反應不及而被當場射殺。

此事件做為支撐「槍比魔法強」之論點的重要依據，雖然那支部隊被隨後趕來的六環魔法使當場擊斃。

但「對多數人而言，科技是最強的武器」之聲量逐漸攀升。
