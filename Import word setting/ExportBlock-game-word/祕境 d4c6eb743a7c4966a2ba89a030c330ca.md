# 祕境

非天然形成，而是由生靈精心開闢、設計的小型界面，通常有特殊用途。

祕境種類：

試煉祕境

強者為培育弟子、後代所開闢的祕境，也是最常見的祕境。常見於各大宗門、學院，如各種試煉塔。

個人祕境

強者因個人雅趣或其它目的所開闢的私人小天地。主人死後，部份可能毀壞，部分可能顯化於世間。

奇珍祕境

為存放珍寶所刻意打造之秘境，由於其本意是不讓人破解，故通常極度危險。

傳承祕境

部份強者會在死前創造祕境做為陵墓，等待有緣人獲取其身家傳承。

為祕境攻略者主要目標。

封印祕境

為封印某強大存在而開闢出的祕境。

陷阱祕境

本身存在即是陷阱，偽裝成其它類型的祕境來吸引挑戰者。通常是為了蒐集獻祭材料、肉身、靈魂、能源、獵物等。

專業的祕境攻略者有許多手段辨別祕境是否為陷阱，但非祕境攻略者容易因貪圖暴利而中招。
