# 鬼族勢力&機構

大部份鬼族為野生鬼，但也有不少高階鬼族之勢力或機構。

地淵
鬼族主基地，位於天輝大陸地底之異次元空間，高階鬼族能力者可自由開關地淵次元門，大多數鬼王位於地淵十八層。

地淵勢力受到人族高層重點關注，並受和平協議影響，通常不會輕易發動攻擊；與此相對的，人族也不能輕易入侵地淵。

地淵為穩定取得人類，創造許多人類養殖場以大量圈養人族，也會從異界召喚柔弱人類做為食糧或玩物。

許多高階鬼族為享受獵食人類的樂趣，或追求更高品質的人類，會選擇離開地淵，自立鬼城、鬼國，或前往鬼族從屬國。

鬼城&鬼國

部份鬼王、魔鬼或智鬼會脫離地淵，在其它地區自立為王，建立鬼城或鬼國。

其通常勢力規模不大，且臨近人族小城、小村落，以便獵食。由於所在地距離人族主要勢力遙遠，該區居民只能委託冒險者、傭兵進行討伐，但仍經常出意外。

此外，鬼城&鬼國不受地淵管轄，可能時常進攻人族，因此被視為鬼族對人族最主要的危害。

鬼影會

鬼族第二大組織，沒有固定據點。成員多為高階鬼族，為鬼族暗中潛入人族社會之諜報機構。許多鬼影會成員也會在人族區域中捕食獵物、散播恐懼、刺殺天才，於各大人族勢力皆有分佈。他們也會暗中將外界鬼族放入人族市區。人族各大勢力中亦有鬼影會的臥底。

其中，鬼族市場內多數科技體系人都來自鬼影會的捕捉。泰克諾爾學生是鬼影會幹部重點捕獵對象，會於每次行動前分析獵捕難易度和價值。

鬼貴族

部分高階鬼族亦有育兒、組建家庭的行為，故產生部份鬼貴族。

鬼貴族直系從小每日皆可食用送上餐桌的白嫩孩童，因此實力普遍強大。鬼貴族多數聚集於地淵，也有許多鬼貴族會生活於鬼族從屬國。

人類養殖場

鬼族圈養人類之機構，透過捕捉許多有生育能力者，大量生產人類做為食物、玩物、洩慾交配工具，亦會奴役人類進行各種勞動。

會使用多種特殊催情藥物、生長激素、幸福計、營養品，如同人類養殖牲畜般，以機械化方式生產人類，但其品質頂多讓鬼族吃飽，遠不如在外捕獲真正的人類。

除了地淵與部分鬼城&鬼國有設置外，許多養殖場會設置在極為偏僻處，並輸送「貨物」至各大鬼族勢力。人族高層會在一定程度容許其存在，以避免鬼族因缺乏糧食而不顧一切進攻人族領地。

主要會餵食蛆蟲、糞便、屍體做為養殖低肉質人類的飼料。

反食人組織

部份不願殺害人族，或強烈信奉「天官鬼老」之高階鬼族拒絕傷害、捕食人類。其通常生活在遠離人族與鬼族之地，默默過著自己的生活。其成員多為青鬼。

跨界召喚所

鬼族召喚異界人類之組織，由於異界召喚需消耗不少能源，且成功率不高，故每次召喚都會嚴格篩選條件。召喚成功的異界人類多半會被獻給地淵鬼王。
