# 修煉仙道—半邪道

由於修煉艱難，不少修士不會完全使用正規方式修煉，而是藉由邪道功法快速進境。

只要不修煉邪功，以下這些方式皆不會有根基不穩、走火入魔之問題。

由於所犯殺孽相對較少，凡人又普遍被修士們視作螻蟻，故以下行為被修煉界視作「半邪道」。

除非是極為正派的修煉者，普通宗派或散修多多少少會走上部份半邪修之途。

1.煉聚靈人丹

由於人體為最契合人之靈物，故以人體入丹能增加積累靈力之效果，其中又以殘留先天之氣的幼童最佳。

★無論凡人、修煉者皆可用來煉製人丹，凡人易取得，修煉者若入丹則能大幅提升效果。

★許多入世測靈玉的修煉者其實是想補充人丹素材

[靈身寶](%E9%9D%88%E8%BA%AB%E5%AF%B6%209ed557cfc49c4fd2b479e6c7d69bc42e.md)

2.雙修爐鼎

若和凡人、修煉者行房事，可趁機吸乾對方精氣以提升靈力和修煉效率。

因此，容貌出色絕美的凡人容易被修煉者抓捕，並被其吸取精氣。

[雙修](%E9%9B%99%E4%BF%AE%20279f38cb84a180b0a9d6d29c3a3a6f5d.md)

3.煞氣

修煉亡魂、煞氣相關的強大功法時，犯下越多殺孽就越能提升實力。

4.奪舍靈玉者

修煉者可捨棄肉身、靈魂脫離，選擇具備靈玉之凡人或修煉者奪舍。

★雖然靈玉本身刻於魂上，但由具備靈玉之魂所衍伸之肉體，能讓奪舍者在奪舍後產生新靈玉。
