# 雙修

修士透過雙修之法採補對方體內精氣，以達到增進修為、加快修煉速度的好處。

★每個人在修煉或成長途中都會積累精氣，精氣的恢復需要時間或特殊藥物。而一旦精氣枯竭，將有生命危險。

雙修的類型

依地位

單向雙修：

其中一人單方面採補另一人精氣。被採補的一方被視作「雙修爐鼎」，其不僅無法獲得任何好處，若短時間內精氣被吸乾則有生命危險。

通常被採補的一方是被迫的，等同於被強暴的進階版。此舉理應為邪道。但由於透過單向雙修提升修為過於便利，早已普及，故各大勢力對此也睜一眼閉一眼。

雙向雙修：

雙方互相採補對方精氣，為最常見的雙修之法，通常是道侶在洞府中的例行事項。

雙向雙修能讓雙方都獲得好處，彼此損耗的精氣也較少。

★若低境界者能採補高境界者之精氣，將獲得遠超其當前境界需求的極大好處。但超快速的修為進境未必是好事。

依身份

通常男修士體內擁有至陽精氣，女修士體內擁有至陰精氣。但有時「魂」會發生變異，導致男修士蘊含至陰精氣，女修士蘊含至陽精氣，而這會在外貌、性格、氣質上體現出。

此外，12歲前的孩童擁有大量「先天精氣」，也能大幅輔助修煉，而先天精氣會於12歲後快速遞減，20歲後幾乎完全消失。

★凡人亦有這些精氣，但整體儲量少於修士，故相貌出眾的凡人經常被路過修士抓去單向雙修。

異質雙修

最常見的雙修類型。

至陽採補至陰，至陰採補至陽，以中和體內精氣，雙屬性互利互惠，達到修煉效率提升之效果。

同質雙修

擁有至陽精氣者採補另一位擁有至陽精氣者；

或擁有至陰精氣者採補另一位擁有至陰精氣者。

透過讓精氣走向極端，提升自己對單一屬性靈力的掌控度。若該屬性正是體內主要屬性靈氣，亦能加快修煉速度。

先天雙修

採補孩童的先天精氣以大幅提升修煉速度。

但與孩童進行雙修有違倫理，故此舉被名門正派列為邪道。

其它影響雙修要素

特殊體質：部份特殊體質能為雙修對象帶來更大的好處。

同時採補多人的精氣，為更極端的修煉方式。

但容易因體內精氣絮亂而自取滅亡。
