# 系統征戰構想

構想一是讓異界生物來搶系統，但感覺很難發展下去。
異界生物轉生為人類，但可使用力量。
（本源能量會隨機在某個低等文明開啟，以靈魂的型態儲存在生物體內高等文明若要參與本源之爭，必須獻祭自身界域的百萬族人來獲得參賽資格。）

（魔法界至尊天才法師、精靈王、魔族魔王
幽冥界亡靈法師、死神、始祖惡魔
天逆界人族、妖族、
天宮天使、神
泰坦巨人、
萬族界龍、鳳、聖獸並列統治十大界域）力量會受到壓制，無法造成大型破壞。

構想二是一開始系統擁有者還算少，後來才逐漸知道許多人擁有。
直到有強大能力者開始組織系統擁有者，逐漸掌控權利，以及可強化系統的「能量石」。
系統擁有者體內可萃取出能量石，越強大的系統擁有的能量石越多。
