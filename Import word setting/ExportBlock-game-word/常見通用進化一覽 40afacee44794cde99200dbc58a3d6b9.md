# 常見通用進化一覽

常見通用進化

蒼究世界存在許多生靈都能進行的進化方式，包含以下幾種：

昇域化

許多生靈可透過「捕食」來獲取獵物之力量，這過程必須滿足「親自獵殺」、「食用獵物」之條件，以便累積特殊的「獵食業力價值」。

只要累積足夠多的「捕食業力價值」，具備昇域化資質之物種就可昇域化。昇域化後之個體實力可能翻五到十倍不等。

目前所有妖獸，以及大多魔獸、靈獸皆可昇域化；萬獸體系、天行體系、邪靈體系之種族也幾乎都可昇域化。
