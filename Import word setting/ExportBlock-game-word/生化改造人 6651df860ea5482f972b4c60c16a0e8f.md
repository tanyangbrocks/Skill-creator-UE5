# 生化改造人

先天改造

超人類培育計劃

★魔力、靈力天賦與魂相關，無法直接由基因改造技術重現

集結科技和現有優秀基因進行融合、改造，研發出當前認定最完美的基因，並以此基因孕育出一批新生兒，每批約3000人。（但新生兒基因仍會產生突變，故彼此會有個體差異）

研發機構對這3000人進行各種培訓，包含超負荷的知識灌輸、體能訓練、藥物與刺激之耐受度等，直至年滿20歲，創造由國家培訓的超級菁英。

最終表現最優良的前1%將提供其基因，研究員對這30人的基因進行複合、改造，製造更優秀的下一代基因。而那30人將可直接擔任高官或下一代研究員，也可選擇離開。

實驗中，基因產生意外缺陷者直接剝奪人權，進行非人道研究後售賣給鬼族；其餘基因優良者開放外出，儘可能在國內繁衍後代，藉此讓泰克諾爾人民的整體基因越來越強。

★當前「超人類培育計劃」已進行到124代。

人民優良血脈計劃

民眾若有在學習、體能、外貌、生產力表現不如預期者，將受到政府關注。

政府會給予其一大筆補助金，希望其不繁衍後代，或使其前往「大地域」生活，並為其打理好身分、職位等，以維持泰克諾爾人的優良基因。但此舉並不強迫。

通常這類人身處「大地域」會得到比在泰克諾爾更好的生活，故會順應政府的政策前往「大地域」。

非人物種改造

混合其它物種之基因，創造、孕育出非人物種，通常用於各種特殊目的，或戰鬥兵器。

摻入其他物種或人工特製的細胞或器官，使自身具備特殊能力。
