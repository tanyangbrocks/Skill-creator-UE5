# 職業(世界職業)

部份高度發展種族能得到蒼天眷顧，擁有「世界職業」之力，獲得職業者將能有系統性地大量獲得技能。

職業的意義與象徵

職業是「才能的具現」，若能覺醒、轉職為某職業，則必定代表個體擁有該潛力天賦。

另外，職業本身不完全是種「能量體系」（沒有「職業能量」），而是能讓使用者更靈活運用自身力量的方式。

★對神話種族而言，職業的存在只代表該個體連特殊潛能都無法挖掘，是弱者的證明。

★職業力量有部分僅適用於蒼究世界，當擁有職業的個體跑到蒼究世界外，部份職業能力可能被削弱或無法使用。

覺醒職業方式

1.輔助覺醒：吸收「泰奧蒼古之心」

泰奧蒼古之心是一種能連接蒼天意志的稀缺資源，其存在分佈世界各地，沒有固定的開採環境。

人族當前持有的泰奧蒼古之心皆由星盟與冒險者公會保管。唯有實力達到「群星級」以上的強者始可用高價購買。

2.自行覺醒

不接收泰奧蒼古之心而自行覺醒者，放眼全蒼究世界都幾乎無人能辦到。此外，「自行覺醒」後獲得的職業能力與「輔助覺醒」並無二致。

3.轉生、穿越或系統持有者

當異界人進行轉生、穿越時可能會對靈魂灌注強大能量，或是系統對靈魂進行改造。

以上情況可能幫助個體挖掘潛力，覺醒技能。

職業的成長

等級：許多職業擁有者可透過特定行為累積「業力價值」來進行升級。等級越高越難升，目前已知最高等級為200級。

升級時將獲得全屬性提升，得到一定量的自由屬性點，同時恢復自身狀態。

[特殊價值](%E7%89%B9%E6%AE%8A%E5%83%B9%E5%80%BC%204fdecaf0814a4de483af5e3e55bb2bd1.md)

轉職：擁有職業者可藉由修煉來積累「職業業力價值」，當職業業力價值達到一定標準後就能前往「女神轉職臺」進行轉職。

若轉職條件為等級，則常見轉職等級如下：

10級 1轉

30級 2轉

60級 3轉

100級 4轉

150級 5轉（最高轉職次數）

★轉職後可能獲得新技能，且各項數值將獲得大幅加成。

同化裝備欄

若個體所持物品契合自身職業，可藉助職業之力同化手中物品，將其化為自己職業所能使用的「職業裝備」或「職業武器」。同化裝備後，職業裝備會對自身的戰力加成會數值化，還會產生各種職業裝備效果。

★職業裝備加成及效果需安裝於自身的裝備欄才可生效，每人同部位的裝備欄有限。

★物品被同化為職業裝備的過程不可逆，且當物品成為職業裝備，則僅有和該職業裝備相容的職業持有者可使用、裝備。

職業的調整

封印職業：倘若不滿自身職業效果，可進行「封印職業」來將職業影響完全屏蔽。封印職業屬於暫時效果，自己可隨時解封。

捨棄職業：將自身職業力量完全廢除，從此不再擁有該職業與其提供的任何效果。但個體捨棄職業後，仍然可以獲得新職業。

贈予&繼承職業：有時能將自身職業力量透過特殊媒介取出，並交付給他人。
