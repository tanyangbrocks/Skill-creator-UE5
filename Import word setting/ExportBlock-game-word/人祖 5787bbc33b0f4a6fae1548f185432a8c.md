# 人祖

從未透漏本名，出生自平民家族。為皮膚白皙、一頭烏黑長髮的稚嫩少年。

最終修練六道皆位於世間頂點，修練界無人能敵。麾下有「人祖八大眷屬」全部擁有接近準道極期之實力。

實際上，人祖幼時撿到仙界遺落之頂級仙器—「兩儀棋」，再加上本人的機智與堅毅，最終才能變得如此強大。

因看不慣人類遭受戰亂迫害，故與拉普拉斯一起建立「淒白輓歌」。其如今的狀態眾說紛紜，有人說其早已飛升至仙界、有人說其還隱藏在世上某處，還有人認為他已身亡。
