# 人、類人體系（當前凡間的主宰體系，具魔）

人、類人體系為擁有大量人族血脈之種族。

個體普遍極為弱小，未鍛鍊之初始戰鬥力也堪稱「最弱種族」。

但人族之數量和組織性有絕大優勢，鍛鍊後的個體實力上限極高，使整體族群實力最為強大。

PS：但人族也因幾乎什麼種族都吃，甚至經常同族自相殘殺，所以也常被其它種族成為「亞妖」。

—–黃階—–

人族：

最強智慧與魔力掌控。肉體偏弱，但十二大能量都有一定天賦。

召喚術、煉金術、科技概念創造者。

矮人族：與人族共存、身高較矮、鬍鬚濃密且耐火，酒量好，肉體能力、五感較強，擅工匠、商務。

侏儒族：矮人與人類血脈各占一半，又稱為半矮人，非常喜愛金錢。善貿易、商務、情報蒐集，智力偏高。

哈頓族：鼻子尖長、膚色偏紅，身高偏矮，智力與體能稍弱，具靈玉者靈力天賦高、魔法天賦較低。和惡魔契合度高，祖先有惡魔血脈。

哈頓族人若不具靈玉，則在各方面能力和人族相比皆有劣勢，教育程度與經濟能力較低，社會地位也偏低。

長耳族：長耳，通常膚色較白、顏值極高，壽命較長。魔力天賦強，與精靈契合度高。祖先有精靈血脈。

長耳族外貌與森精相似，但在人類社會中身份普遍尊貴，故鮮少看見長耳族奴隸。

紅獵族：白髮、尖耳、膚色赤紅，肉體強大，感官發達，喜愛戰鬥，是善於狩獵的特殊種族。目前分為親近人族社會的「蒼文派」與自成部落的「赤勇派」。

狼人族：平時與常人無異，但在月圓之夜會化身為力量強大的狼人，獵殺人類。狼人成功捕獵的人越多，自身抵抗狼人化的力量就越強，最終可能完全變回人類。

半血族：人族與血族的後代。其雖不懼陽光，但需依賴吸食血肉維生，普遍顏值極高、膚色蒼白，擁有特別強大的智力、肉體素質和恢復力，壽命也更長。

人鬼族 ：祖先為鬼族與人類的後代。外貌與人類無異，但以食人/類人為生。肉體、五感、智慧、外貌及操縱能量天賦都高於人族，可藉由吞食人類來快速獲得力量。通常暗中生活於人族領地，被人族發現格殺勿論。

PS：狼人族、血族、人鬼族雖為類人，卻對人類社會有極大危害，故又被人族稱為「類人異種」。

—–玄階—–
瓦爾基利：天使後裔，戰鬥天賦、正義感強，與天使契合度高，地位尊貴。情緒激動時，體表會亮起古老玄奧的光紋。

龍人：人與龍的後代，可使用部份龍的力量，且天生必定同時擁有魔力和靈力，並將其結合為䵻龗。

—–地階—–

不滅族：世界孕育的不死者，容貌俊美而空靈，唯一的弱點是精神攻擊。每個體擁有一個附加天賦。極難生育。

古蘭族：擁有堪比天球族的恐怖魔力，智力極高、肉體力量也不俗，平均實力甚至能比肩地階半神體系。

但每名個體天生背負一種可怕詛咒，如沒有情感、遭萬物生靈憎恨、終生沉浸於悲傷等，大多無法融入人類社會。

阿修羅：有六隻手臂，肉體力大無窮，善用真氣、靈力，生育難度高。

但該族性情暴躁好戰，男性貌醜且嗔怒心重、女性貌美但嫉妒心重，經常與異族發起戰爭。
