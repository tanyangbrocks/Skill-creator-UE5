# 詭道（圖騰）

世間奇人透過各種方式開拓出新的道，修行詭道者很容易產生意道的頑氣。

修煉體系中不透過武道、仙道、法道、意道、魂道等方式，卻能透過自身特定行為獲得力量之法，皆被歸為詭道。

詭道神秘且變換莫測，許多人會視需求搭配修煉。
詭道種類繁多，目前統計共有三百六十道。

---

詭道象徵物：圖騰

由於詭道是眾多相異之道的統稱，故沒有統一的象徵物。

但許多詭道修煉者會在其軀體刻劃該道的特殊圖騰，以展示身份。故大眾普遍將「圖騰」視為詭道象徵物。

最初圖騰由來：

知名詭道中的「逆天道」是真的會在體表產生「逆天紋」，而逆天道做為最早期的詭道派別之一，經常向其它派別的詭道修煉者炫耀自己獲得了實際象徵。

其它詭道修煉者對此感到羨慕，便也跟著在身上手動刻劃象徵圖騰，自此「詭道圖騰」的習俗流傳下來。

三大至尊詭道：

封神道、鑽研道、天命道

三大至尊詭道的力量極其強大，它們其實是對「魂」之「價值」的操縱。其概念早已超越修煉體系，甚至是能量體系的範疇。

但大多數人們無法得知這種資訊，故將這三種不可思議現象都歸類為詭道。

封神道

名字、稱號、封號、權威具備實質性的力量加護。

「聲望、信仰」可轉化為實質力量，行成獨有攻伐力。

權勢封號：神祉、供奉、帝王、魔王、王侯、霸主、教宗、宗主、老祖、聖子等。

★權勢封號會隨勢力興衰影響力量，也可封號給屬下對其進行加強。

成就封號：

無敵證道、永敗證道、不朽神話、

命數封號：

氣運之子、天命之子等特殊命數會得到相應封號。

境界封號：

元嬰期以上可透過「自封」來獲取封號，成就越高，封號力量越強。

★封號會烙印在靈魂中，也可將其外顯為異象虛影，封號越強、異象越盛。
★封號可重疊，因此也部份天驕做為封號獵手，專門獵取封號。

📌封神道是對「聲望業力價值」的系統性操作。

鑽研道

反覆練習同一件事至精，可發揮出強大力量。

千鑽境：對那件事完全熟練，除非刻意或外在影響，否則絕無失誤之可能。

萬鑽境：對某事達到出神入化，可以發揮額外功效。

十萬鑽境：舞動裊裊道音，產生道光異象。

百萬鑽境：技進乎道，觸摸道之雛形，週身浮現道紋，以及古今觸道者的姿態。

千萬鑽境：完全悟道，道下一切主宰。

億鑽境：超越該道，立於「道」之上，連道一併支配。

每一鑽不僅是次數，必須是有所明悟的一鑽才算數，若悟性或伴隨機緣，單一一鑽可能代表數百數千鑽之效。

📌鑽研道是對「魂」的「精熟業力價值」做出詮釋。

天命道

每人生來有其命數、氣運。

可透過奪天地氣運來改變自身氣運、命數。

對應氣運可幫助完成某目標，逆天而行恐遭天命反噬。

氣運等級：

微渺、云云、浮塵、逆天、祥瑞

好命數類：

祥瑞、長生命、王命、帝王命、天命之子、至尊主宰。

壞命數類：

災厄、災厄之子

其它命數：

結丹命、元嬰命、蛻凡命、

氣運可以用特殊方式滋養、掠奪，並能增加對應之「道」的感悟，增強實力。

推衍之法、卜卦之法，亦為天命道。

📌天命道是在窺探神所給予的「命運強度」。

逆天道

最初的詭道派別之一。

逆天道修煉者會向上天發出戰帖，並犧牲固有的「某事物」來開闢逆天之路。

體表會浮現血紅色經文狀的「逆天紋」，可更容易開發出自己的力量、開闢自身修行之路。
逆天紋開啟後也可修煉，透過逆天證道，增強逆天道威能。
