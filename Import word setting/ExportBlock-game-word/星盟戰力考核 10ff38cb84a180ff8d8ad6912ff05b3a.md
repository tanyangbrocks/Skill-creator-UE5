# 星盟戰力考核

戰力數值測定方式1—基礎六維測定

參考以下六維綜合值，按數值平均分數劃分至星盟通用戰力等級：

[星盟通用戰力等級](%E6%98%9F%E7%9B%9F%E9%80%9A%E7%94%A8%E6%88%B0%E5%8A%9B%E7%AD%89%E7%B4%9A%206cd6fa453e3749efae5516932b65faa5.md)

力量綜合值
純物理衝擊單位：龐頓
定義：普通人類拳頭一擊殺死藍色史萊姆，將其物理輸出定義為「1龐頓」。
結合肌肉力量與出拳速度，這也是蒼究世界普通人類的平均拳擊力量。此為蒼和元年前，自大分裂時代流傳至今的物理衝擊衡量單位。

純能量體系破壞力單位：奎因
定義：排除能量層次、能量性質、元素屬性、純度與密度、能量應用模式等要素，純粹以能量進行衝擊所達到的破壞效果。

「1奎因」為擊殺藍色史萊姆所需最小能量。

此為拉普拉斯建立標準直擊魔法時，所創造出用於衡量魔法破壞力之能量單位，現已被廣泛運用至各大能量體系。

★相較於「龐頓」，做為能量單位的「奎因」更適用於衡量破壞力，但兩者無法換算。

最高破壞力單位：奎頓（奎因+龐頓）
定義：考核者釋放出當下所能施展之最強攻擊，擊打「改造黃金史萊姆之王」，並由其評斷傷害程度。若可瞬間擊殺「改造黃金史萊姆之王」，則至少為巔峰級以上的攻擊水準。

技術綜合值
將武之心得、能量運用技術、動作精密度、技巧純熟度之要素綜合評估。

速度
將爆發速度、反應速度、感知速度、最高持續速度之要素綜合評估。

智慧
將洞察力、決策力、戰鬥經驗、策略、心理強健之要素綜合評估。

[智慧強度](%E6%99%BA%E6%85%A7%E5%BC%B7%E5%BA%A6%2017bf38cb84a18006b38efd80d4518f4c.md)

生存
將基礎防禦力、最高防禦力、回復力、生命力、保命手段之要素綜合評估。

特殊專長
