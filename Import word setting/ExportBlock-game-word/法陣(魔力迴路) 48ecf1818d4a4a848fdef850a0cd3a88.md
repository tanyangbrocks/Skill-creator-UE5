# 法陣(魔力迴路)

搭配煉金術，於軀體或物品上事先刻下魔力迴路。

優點：能夠在緊要關頭瞬發魔法。

缺點：需事先刻印，越高級的魔法刻印難度越高，也要價不斐。

法陣應用

生體法陣

在軀體刻下法陣，通常用於瞬發複雜魔法、調和體內能量。

然而，當魔力流淌在軀體法陣上，使用者會感受到灼燒般的劇烈疼痛。

區域法陣

事先在某處地面或空間佈下法陣，以便特定時刻啟動。通常用於暗算對手、發動複雜魔法、召喚、佈置結界、傳送等，是法陣最普遍之應用。

然而，區域法陣通常需要較長時間準備，也相對非常容易被阻礙、破壞甚至反支配。

魔導法陣（魔紋）

在器物刻下法陣，主要有「魔器具」、「魔導器」兩種應用。

魔器具

在器物刻下法陣，通常用於製作武器、鎧甲等，或是其它隨身小型魔器具。

魔導器

通常代表大型魔器具，如魔導車、魔導測試儀、魔導防盜系統、魔導堡壘等，結合源能與西方泰克拿勒的科技結晶。

魔紋機械

利用魔紋編程技術製作的高科技魔導機械，如魔力電腦、魔力手機等。
