# 遺跡

由無主的價值力量所產生的特殊空間。

其中多蘊含無法用普通能量破解的強大法則之力，欲破解者只能在遺跡中尋得線索，按其規則破解。

由於遺跡本身蘊含強大法則力量，有機率在其中孕育出至寶。

常見遺跡種類

歷史型遺跡

部份重大歷史事件會產生大量無人可回收的價值，這些停留在世間的價值將隨時間自然產生意識，誕生遺跡。

揭示性遺跡

或是某單一或大量生靈在臨死前渴望告知世人真相，也可能在死前催發出強烈的「願」，進而創造出揭示性遺跡。

自然性遺跡（魔境/洞天福地）

世間能量和空間波動，自然而然從某空間節點誕生出的遺跡，又被稱作「魔境」、「洞天福地」。通常擁有大量珍稀資源，但也具有未知風險，是個災禍與機緣並存的神秘之地。
