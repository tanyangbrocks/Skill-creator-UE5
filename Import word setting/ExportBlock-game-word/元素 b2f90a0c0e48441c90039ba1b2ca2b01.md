# 元素

由能量體系引發之「特定性質」，經常為自然現象，元素又細分為：

基礎元素與其特性

金：鋒銳、堅硬

木：生機、蔓生

水：變幻、綿延

火：毀滅、灼熱

土：厚重、龐大

風：遍佈、無形

雷：剋邪、極速

進階元素

毒、光、暗、幻、冰、

極元素（以元素型態涉及法則）

超、聖、沌、空、言、時、慟...…
