# 裝扮

人族裝扮

修練體系服飾（此處以六龍聖天為主）

休閒：長袍、皇袍、官袍、蓑衣、制式儒袍、暗色斗篷等。

戰鬥：制式道袍、制式武道服、重鎧甲、輕鎧甲等。

配飾：髮髻、胸針、玉佩、香囊、耳墜、手鐲、冠冕、鈴鐺、烏紗帽、斗笠、緞帶、隨身法寶等。

衣物材料：棉、麻、竹、蠶絲、獸毛、妖獸&靈獸素材、絲綢、皮革、金屬礦。

主要風格：多以素色+刺繡為主，許多人會在衣物刺下自身座右銘。人們更傾向透過各種配飾彰顯身份地位和品味。

由於武學、書院盛行，多數民眾穿著自身所屬勢力的制式儒袍、制式武道服，即植物纖維素色布料，繡上勢力代表徽章。

★成衣技術較差，衣物普遍寬大、粗糙，且修煉體系切磋破壞衣物，故民眾對衣物的需求量也較多。

★相比美觀，修煉體系的衣物普遍更注重輕便、保暖與象徵意涵，民眾普遍無穿搭概念。但士紳階級重視禮節，會刻意以鮮豔色調妝點衣物來展現地位。

支配體系服飾（此處以戴洛艾色為主）

休閒：衣褲、螺旋蜘蛛絲毛衣、貴族禮袍、洋裝、半固化史萊姆等。

戰鬥：中世紀魔導鎧甲、輕便軟甲、法師袍、自適應緊身衣、聖袍、元素晶裝等。

配飾：花冠、金冠、戒指、胸針、項鍊、耳環、護身符、紋章、手鍊、吊墜等。

衣物材料：棉、麻、竹、蜘蛛絲、獸毛、妖獸&魔獸素材、皮革、金屬礦、史萊姆、元素晶體等。

主要風格：由於能從史萊姆身上大量獲取染料，衣物色彩較為豐富鮮豔。貴族、魔法使、皇室、教會等權貴透過大量魔道具配飾展現財力及地位。

★衣物相對重視美觀，成衣技術較佳，衣物原料較為豐富，其中魔法的協助功不可沒。

★許多魔道具、天物會製作成配飾，如自動防禦、增強魔力的戒指、項鍊、耳環，或發動神聖魔法的吊墜等。

科技體系服飾（此處以泰克諾爾為主）

休閒：客製化高科技衣物、精選潮流模板衣、萬能變形衣、動態效果衣等。

戰鬥：機戰神鎧、生體特戰裝甲、彈性束裝超能衣（代表自身警戒顏色）、防彈軍服、隱身衣、流體電漿衣等。

★受管制，普通民眾無法透過常規渠道取得戰鬥衣物。

配飾：穿戴式模擬實體特效、各部位森羅萬象的配飾等。

衣物材料：源導體、超合金、原子纖維、輕粒子凝聚物、能量聚合體、幽光塗層。

主要風格：有多種穿搭流派，具複雜且快速迭代的時尚潮流。民眾重視透過穿搭展現個人特質，幾乎所有休閒衣物都是客製化或模組化，除非刻意照抄他人設計，否則不太可能出現兩件相同款式的衣物。

★有極為先進的成衣技術，能透過高科技進行低成本大量客製化。

★衣物不僅提供保暖、美觀功能，還幾乎具備「五自一不」（自適應版型、自動穿脫、自動調溫、自動清潔、自動避險、絕不破損），以及肌膚保養、即時變型、極致舒適、等基本要求。

其它科技體系裝扮

髮型：千奇百怪，甚至能讓髮型有動態效果，或是變成其它事物，如火焰、光團、鍬形蟲等。

妝造：由於整容及醫美技術達巔峰，泰克諾爾人也不會有皮膚問題，每人都能擁有理想外貌。民眾僅有節慶、拍戲、化妝舞會等特殊狀況會化妝。

體味：民眾能透過醫美手術及生化改造來永久散發想要的體味，還能隨時調整，故不需香水類產品。
