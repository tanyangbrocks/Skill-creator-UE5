# 天球族

萬法根源，元素創造者，擁有最強魔力天賦。外型是漂浮在空中的球體，認為自己是蒼究世界最高貴的種族，自詡為神明。

天球族的魔法雖極度強大，但其肉身十分脆弱，也幾乎無法行使除魔力以外的能量體系。

天球族有三種性別

布基拉：赤瞳黑球

梅特：金瞳白球

薛千派歐：瞳色與球體紋路隨心情變化

天球族必須同時與另外兩種性別交配才能產生後代，但天球族做為高智慧種族，此舉易產生感情糾紛。此外，亦有諸多天球族為同性戀。
