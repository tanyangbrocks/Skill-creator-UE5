> **核心原則**：
> - 蒼究世界觀（`game-word`）為**主軸**；`game-setting` 文件在描述相同概念時以蒼究版本為準

> - 蒼究勢力/歷史/文化/地圖屬長期劇情素材

## 蒼究世界觀架構速覽

### 能量體系（= MP 來源）
```
18 基礎體系（三大根源）：
  修煉六道：武道 / 仙道 / 法道 / 意道 / 魂道 / 詭道（圖騰）
  支配六法：魔法 / 妖力 / 奧術 / 神聖力 / 源能 / 星能
  世界六意：技能 / 職業 / 超能 / 神力 / 概念 / 尋能

複合體系（53 種）：
   9 種修煉同源複合（`九種修煉同源複合體系.md`）
  15 種支配同源複合（`十五種支配同源複合體系.md`）
  29 種異源複合  （`二十九種異源複合體系.md`）

合計 71 種潛在 MP → 需資料驅動設計（ManaTypeRegistry）
同時承受能量的限制採**相容性判斷**（見下方 W-6 說明），非固定數字上限
```

### 境界等級（採用星盟通用戰力等級）
```
正式徽章（低→高）：
  學徒（灰）→ 超凡（黑）→ 征戰（綠）→ 主將（藍）
  → 群星（黃）→ 耀日（紅）→ 巔峰（紫）

客製化（巔峰以上）：
  特異 → 超特 → 封王 → 超王 → 封帝 → 超帝
  → 封聖 → 超聖 → 滅世

遊戲等級（LV 1–100+）與境界的對應數值，留到 W-1 實作時確認
```

### 種族體系（採用蒼究完整種族設定）
- 四階血脈（黃/玄/地/天），決定成神難度
- 見 `種族.md` 及各種族子文件；`game-setting/選擇.md` 的種族清單可供快速查閱
- 選擇階級：1 階免費、2 階付費解鎖、3 階需先解鎖 2 階（神族限神界轉世）

### 元素與法則
- 元素：7 基礎（金木水火土風雷）+ 進階（毒光暗幻冰）+ 極元素（見 `元素.md`）
- 碰撞系統：15+ 種雙元素組合效果（見 `元素.md`）
- 法則：12 大法則（基軸/天道/根源各四，見 `法則.md`）

### 長期劇情素材（近期不實作）
勢力、歷史、文化、地理、NPC 身分立場 → 見 `現今勢力.md`、`歷史(人族視角).md`、各大陸文件

### 第二批：核心機制擴充（需設計決策）

**W-6｜MP 多類型框架**
- 71 種潛在 MP → 資料驅動（`ManaTypeRegistry`，以 string/int id 取代 enum）
- 玩家持有「已解鎖 MP 槽」（上限 3 種，特殊 5 種）
- 儲能瓶（`功能.md`）：道具，可儲存暫時不用的能量種類，切換時使用
- **能量相容性判斷**（取代 game-setting 的「3 種硬上限 + 自爆」規則）：
  - 同一根源內的能量天然相容；跨根源存在衝突機率，機率由種族/體質/能量距離決定
  - 成功完成複合體系融合的能量視為同一種，不產生衝突
  - 衝突後果：走火入魔（氣脈逆亂、體質崩潰等），嚴重程度分級，非必然自爆
  - 特異體質（特異體）降低衝突機率或提高可承受的衝突上限
  - 具體數值與衝突公式留到實作時設計
- 實作分三階段：框架 → 18 基礎 → 53 種複合（W-12）
- 實作時讀取：`能量複合理論.md`、`戰鬥&三大通用要素&十八體系三大根源.md`
- 改動：新增 `ManaType.cs`、`ManaTypeRegistry.cs`，`PlayerController.cs`、`SpellArray.cs`
- 依賴：W-1、W-CC3（種族影響 MP 親和）

## 設計原則備忘

1. **蒼究 > game-setting**：兩者描述同一概念時以蒼究版本為準
2. **`game-setting` 補機制**：蒼究無定義的機制設計（角色創建流程/數值公式/功能清單）採 game-setting
3. **細節留到實作時讀**：每項 W-N 均標注「實作時讀取哪份文件」，計畫只定義架構
4. **每項 W-N 可獨立交付**：不強制連帶，除非標注依賴
5. **第三批純內容型**：不阻擋前兩批的系統開發
